hint: |
  クラスにはフィールド（データ）とメソッド（処理）を定義する。フィールドには型と変数名を、メソッドには戻り値の型とメソッド名を指定する。
explanation: |
  **解答例**：
  ```java
  class BankAccount {
      String owner;
      int balance;

      void deposit(int amount) {
          balance += amount;
          System.out.println(amount + "円を入金しました");
      }

      void withdraw(int amount) {
          if (amount <= balance) {
              balance -= amount;
              System.out.println(amount + "円を出金しました");
          } else {
              System.out.println("残高不足です");
          }
      }

      void showBalance() {
          System.out.println(owner + "の残高: " + balance + "円");
      }
  }
  ```

  クラスはフィールドでデータを保持し、メソッドでそのデータを操作する。`withdraw` メソッドでは残高チェックを行い、不正な出金を防いでいる。
