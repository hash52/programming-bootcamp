  クラスはオブジェクトの **設計図（テンプレート）** であり、フィールド（データ）やメソッド（処理）を定義する。オブジェクト（インスタンス）は、クラスという設計図から `new` キーワードで作られた **実体** である。1つのクラスから複数のオブジェクトを作ることができる。
