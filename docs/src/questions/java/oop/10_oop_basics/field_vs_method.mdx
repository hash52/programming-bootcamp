  フィールド（メンバ変数）はオブジェクトが持つ **データ** を保持する変数である。メソッドはオブジェクトが行う **処理** を定義する。例えば「車」クラスでは、`color`（色）や `speed`（速度）がフィールド、`accelerate()`（加速する）や `brake()`（ブレーキをかける）がメソッドである。
