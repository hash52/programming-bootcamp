  **解答例**：
  ```java
  Student s1 = new Student();
  s1.name = "田中";
  s1.score = 85;

  Student s2 = new Student();
  s2.name = "佐藤";
  s2.score = 92;

  System.out.println(s1.name + ": " + s1.score);
  System.out.println(s2.name + ": " + s2.score);
  ```

  1つのクラスから `new` で複数のオブジェクトを作成でき、各オブジェクトは **独立したフィールド値** を持つ。s1のnameを変更してもs2のnameには影響しない。
