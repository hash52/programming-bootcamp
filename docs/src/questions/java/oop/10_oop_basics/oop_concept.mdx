  オブジェクト指向プログラミング（OOP）とは、関連するデータ（フィールド）と処理（メソッド）を **オブジェクトとしてひとまとめにして** プログラムを構築する考え方である。現実世界の「もの」をプログラム内で表現しやすくなる。
