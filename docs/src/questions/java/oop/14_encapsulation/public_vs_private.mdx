explanation: |
  - **public**: **どこからでも** アクセスできる（他のクラス、他のパッケージからもOK）
  - **private**: **同じクラス内からのみ** アクセスできる（外部のクラスからはアクセス不可）

  フィールドは `private`、メソッド（getter/setter等）は `public` にするのがカプセル化の基本パターンである。
