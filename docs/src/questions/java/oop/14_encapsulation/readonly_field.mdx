explanation: |
  読み取り専用（外部から変更不可）のフィールドを作るには、フィールドを **private** にして **getter のみ** を提供する。setterを作らないことで、外部からの値の変更を防げる。コンストラクタでのみ値を設定する設計パターンがよく使われる。
