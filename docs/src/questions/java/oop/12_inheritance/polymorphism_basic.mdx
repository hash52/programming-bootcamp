hint: |
  親クラス型の変数に子クラスのオブジェクトを代入できる。メソッドを呼び出すと、実際のオブジェクトの型のメソッドが実行される（ポリモーフィズム）。
explanation: |
  ポリモーフィズムにより、親クラス型の変数で子クラスのオブジェクトを扱える。`animals[i].sound()` で、実際のオブジェクトの型に応じたメソッドが実行される。

  **解答例**：
  ```java
  class Animal {
      String sound() {
          return "...";
      }
  }

  class Dog extends Animal {
      @Override
      String sound() {
          return "ワン！";
      }
  }

  class Cat extends Animal {
      @Override
      String sound() {
          return "ニャー！";
      }
  }

  // mainメソッド内
  Animal[] animals = { new Dog(), new Cat() };
  for (int i = 0; i < animals.length; i++) {
      System.out.println(animals[i].sound());
  }
  ```
