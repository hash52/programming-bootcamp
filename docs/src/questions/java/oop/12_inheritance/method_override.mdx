hint: |
  オーバーライドとは、親クラスのメソッドと同じ名前・引数・戻り値のメソッドを子クラスで再定義すること。`@Override` アノテーションを付けるのが推奨。
explanation: |
  オーバーライドにより、同じメソッド名でもクラスごとに異なる動作を定義できる。

  **解答例**：
  ```java
  class Animal {
      String speed() {
          return "普通の速さ";
      }
  }

  class Dog extends Animal {
      @Override
      String speed() {
          return "時速60km";
      }
  }

  class Cat extends Animal {
      @Override
      String speed() {
          return "時速48km";
      }
  }
  ```

AnimalクラスのspeedメソッドをDogクラスとCatクラスでそれぞれオーバーライドし、異なる速度を返すようにせよ。

**要件**：
- Animalクラスにはspeedメソッド（String型を返す）を定義する
- DogクラスとCatクラスでspeedメソッドをオーバーライドし、それぞれ異なる文字列を返す
- `@Override` アノテーションを付ける
