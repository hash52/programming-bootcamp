  メソッド内で宣言した変数（ローカル変数）は、そのメソッドの **スコープ（有効範囲）** 内でのみ使用できる。メソッドの実行が終了すると、ローカル変数はメモリから解放される。他のメソッドからはアクセスできない。引数もそのメソッド内でのみ有効である。
