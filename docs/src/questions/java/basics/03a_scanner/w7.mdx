explanation: |
  **nextInt()メソッド**を使うと、標準入力から整数を読み取れる。

  ```java
  Scanner scanner = new Scanner(System.in);
  int number = scanner.nextInt();  // 整数の入力を待つ
  ```

  **nextInt()の動作**：
  1. ユーザーが整数を入力してEnterキーを押すまで待機
  2. 入力された数値を`int`型として取得
  3. 数値以外が入力された場合はエラー（InputMismatchException）

  **使い所**：
  - 年齢や数量など、整数のみを読み取る
  - 計算に使う数値
  - ユーザーの選択肢（1, 2, 3など）

  **注意点**：
  - `nextInt()`の後に`nextLine()`を使う場合は、間に`scanner.nextLine()`を挟む必要がある

  **解答例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.print("1つ目の数値を入力してください: ");
          int num1 = scanner.nextInt();

          System.out.print("2つ目の数値を入力してください: ");
          int num2 = scanner.nextInt();

          int sum = num1 + num2;
          System.out.println("合計: " + sum);
      }
  }
  ```

  **実行例**：
  ```
  1つ目の数値を入力してください: 10
  2つ目の数値を入力してください: 25
  合計: 35
  ```

  **応用例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.println("=== 簡易計算機 ===");

          System.out.print("1つ目の数値: ");
          int a = scanner.nextInt();

          System.out.print("2つ目の数値: ");
          int b = scanner.nextInt();

          System.out.println("\n=== 計算結果 ===");
          System.out.println(a + " + " + b + " = " + (a + b));
          System.out.println(a + " - " + b + " = " + (a - b));
          System.out.println(a + " * " + b + " = " + (a * b));
          System.out.println(a + " / " + b + " = " + (a / b));
          System.out.println(a + " % " + b + " = " + (a % b));
      }
  }
  ```
