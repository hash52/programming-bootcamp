## 解説

標準入力を使うには、**Scannerクラスをインポートして、Scannerオブジェクトを作成する**必要がある。

**手順の詳細**：

1. **import文を記述**
   ```java
   import java.util.Scanner;  // ファイルの先頭
   ```
   - `java.util`パッケージの`Scanner`クラスを使うことを宣言

2. **Scannerオブジェクトを作成**
   ```java
   Scanner scanner = new Scanner(System.in);
   ```
   - `Scanner`：クラス名
   - `scanner`：変数名（任意だが、小文字で始めるのが慣習）
   - `new Scanner(System.in)`：標準入力を読み取るScannerオブジェクトを作成
