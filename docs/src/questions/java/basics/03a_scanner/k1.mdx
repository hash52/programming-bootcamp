explanation: |
  **標準入力**（Standard Input）は、**ユーザーがキーボードから文字や数値を入力する仕組み**である。

  プログラムは、標準入力を使って以下のようなことができる：
  - ユーザーの名前を入力してもらう
  - 年齢や数値を入力してもらう
  - 選択肢から選んでもらう

  **標準入力の流れ**：
  1. プログラムが入力を待つ
  2. ユーザーがキーボードから入力
  3. Enterキーを押す
  4. プログラムが入力されたデータを受け取る

## 問題

標準入力の説明として最も適切なものを選べ。

## 解説

標準入力は、**ユーザーとプログラムが対話するための基本的な仕組み**である。

例えば、以下のようなプログラムで標準入力を使う：
- 計算機アプリ：ユーザーが数値を入力して計算結果を表示
- クイズアプリ：ユーザーが答えを入力して正誤判定
- ログイン画面：ユーザー名とパスワードを入力

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("あなたの名前を入力してください: ");

**標準入力の仕組み**：
- **System.in**：標準入力を表すオブジェクト
- **Scanner**：標準入力を読み取るためのクラス
- ユーザーがEnterキーを押すまで、プログラムは待機する

**他の入力方法との違い**：
- **標準入力**：ユーザーがキーボードから入力
- **ファイル入力**：ファイルからデータを読み込む
- **ネットワーク入力**：インターネット経由でデータを受け取る
