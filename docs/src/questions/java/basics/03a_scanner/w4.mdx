explanation: |
  **標準入力**を使うと、プログラム実行中にユーザーがキーボードから文字を入力できる。

  **基本的な流れ**：
  1. `Scanner`オブジェクトを作成
  2. `nextLine()`メソッドで1行を読み取る
  3. 読み取った文字列を変数に格納
  4. 変数を使って処理や表示を行う

  **ポイント**：
  - `import java.util.Scanner;`でScannerクラスをインポート
  - `Scanner scanner = new Scanner(System.in);`でScannerオブジェクトを作成
  - `scanner.nextLine()`で1行を読み取る
  - ユーザーがEnterキーを押すまで入力待ちになる

  **解答例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.print("好きな食べ物を入力してください: ");
          String food = scanner.nextLine();

          System.out.println("あなたの好きな食べ物は" + food + "ですね！");
      }
  }
  ```

  **実行例**：
  ```
  好きな食べ物を入力してください: カレー
  あなたの好きな食べ物はカレーですね！
  ```

  **応用例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.print("あなたの名前を入力してください: ");
          String name = scanner.nextLine();

          System.out.print("あなたの出身地を入力してください: ");
          String hometown = scanner.nextLine();

          System.out.println("\n=== 自己紹介 ===");
          System.out.println("名前: " + name);
          System.out.println("出身地: " + hometown);
      }
  }
  ```
