explanation: |
  **nextDouble()メソッド**を使うと、標準入力から小数を読み取れる。

  ```java
  Scanner scanner = new Scanner(System.in);
  double number = scanner.nextDouble();  // 小数の入力を待つ
  ```

  **nextDouble()の動作**：
  1. ユーザーが数値を入力してEnterキーを押すまで待機
  2. 入力された数値を`double`型として取得
  3. 整数を入力しても`double`型として扱われる（例：`5` → `5.0`）
  4. 数値以外が入力された場合はエラー（InputMismatchException）

  **使い所**：
  - 身長や体重など、小数を含む数値
  - 価格や金額（小数点以下を含む）
  - 科学計算や統計計算

  **整数との違い**：
  - `nextInt()`：整数のみ（小数点以下は不可）
  - `nextDouble()`：整数も小数も受け付ける

  **解答例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.print("身長(m)を入力してください: ");
          double height = scanner.nextDouble();

          System.out.print("体重(kg)を入力してください: ");
          double weight = scanner.nextDouble();

          double bmi = weight / (height * height);
          System.out.println("BMI: " + bmi);
      }
  }
  ```

  **実行例**：
  ```
  身長(m)を入力してください: 1.70
  体重(kg)を入力してください: 65.0
  BMI: 22.49134948096886
  ```

  **応用例**：
  ```java
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);

          System.out.println("=== 円の面積計算 ===");

          System.out.print("半径を入力してください: ");
          double radius = scanner.nextDouble();

          double area = Math.PI * radius * radius;
          System.out.println("\n半径 " + radius + " の円の面積: " + area);

          // 小数点以下2桁で表示
          System.out.printf("面積（小数点以下2桁）: %.2f\n", area);
      }
  }
  ```

  **補足**：
  - `Math.PI`：円周率（3.141592...）
  - `System.out.printf()`：書式指定で小数点以下の桁数を制御できる
