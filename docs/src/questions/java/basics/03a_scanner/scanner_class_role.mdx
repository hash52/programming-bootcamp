explanation: |
  **Scannerクラス**は、**標準入力からデータを読み取り、文字列や数値に変換するクラス**である。

  **Scannerクラスの主なメソッド**：
  - `nextLine()`：1行を文字列として読み取る
  - `nextInt()`：整数を読み取る
  - `nextDouble()`：小数を読み取る
  - `next()`：次の単語を読み取る

  **Scannerクラスの使い方**：
  ```java
  import java.util.Scanner;

  Scanner scanner = new Scanner(System.in);
  String input = scanner.nextLine();
  ```

## 問題

Scannerクラスの役割として最も適切なものを選べ。

## 解説

Scannerクラスは、**標準入力からデータを読み取るための便利なツール**である。

Scannerクラスがない場合、標準入力からデータを読み取るのは非常に複雑になる。Scannerクラスを使うことで、簡単にユーザーの入力を受け取れる。

import { OneCompilerCodeBlock } from '@site/src/components/OneCompilerCodeBlock';

<OneCompilerCodeBlock language="java" code={`import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("あなたの年齢を入力してください: ");
        int age = scanner.nextInt();

        System.out.println("あなたは" + age + "歳ですね！");

        // 20年後の年齢を計算
        int futureAge = age + 20;
        System.out.println("20年後は" + futureAge + "歳です。");
    }
}`} />

**Scannerクラスの便利な点**：
1. **型変換が自動**：`nextInt()`で整数、`nextDouble()`で小数を取得
2. **簡潔な記述**：複雑な処理を1行で書ける
3. **エラー処理**：数値以外が入力された場合に例外を投げる

**Scannerクラスの使用には`import`が必要**：
```java
import java.util.Scanner;  // ファイルの先頭に記述
```
