  複数クラスに分けるメリットは、**コードの見通しが良くなる**（A）、**チーム開発がしやすくなる**（C）、**再利用性が高まる**（D）である。実行速度の向上（B）はクラス分割のメリットではない。クラス分割は主に **コードの管理しやすさ** を目的としている。
