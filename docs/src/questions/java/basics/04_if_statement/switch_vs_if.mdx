  switch文は、1つの変数の値が特定の値に一致するかを判定するのに適している。`case 1:`, `case 2:` のように固定値との一致を判定する場面で使う。一方、大小比較（`age >= 18`）や複雑な条件（`&&`, `||`）を使う場合はif文の方が適切である。
