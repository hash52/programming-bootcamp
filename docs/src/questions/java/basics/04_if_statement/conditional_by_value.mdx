explanation: |
  **解答例**：
  ```java
  if (score >= 90) {
      System.out.println("優");
  } else if (score >= 70) {
      System.out.println("良");
  } else if (score >= 50) {
      System.out.println("可");
  } else {
      System.out.println("不可");
  }
  ```

  if-else if文では条件を上から順に評価するため、`score >= 70` の時点ですでに `score < 90` が保証されている。このように条件を絞り込むことで、シンプルに書ける。
