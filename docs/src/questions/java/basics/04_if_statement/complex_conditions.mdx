  **ポイント**: `&&` は「かつ（AND）」を意味する論理演算子である。2つの条件を両方満たす場合に `true` になる。
explanation: |
  **解答例**：
  ```java
  if (age >= 18 && hasTicket) {
      System.out.println("入場できます");
  } else {
      System.out.println("入場できません");
  }
  ```

  `&&`（AND演算子）は左辺と右辺の両方が `true` の場合のみ全体が `true` になる。`boolean` 型の変数は `== true` を省略して直接条件に使える。
