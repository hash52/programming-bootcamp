hint: |
  合計を求めるには、変数 `sum` を0で初期化し、ループで各要素を加算していく。平均は `合計 / 要素数` で求められるが、**int同士の割り算は小数点以下が切り捨て** られる点に注意しよう。
explanation: |
  **解答例**：
  ```java
  int sum = 0;
  for (int score : scores) {
      sum += score;
  }
  System.out.println("合計: " + sum);
  System.out.println("平均: " + (double) sum / scores.length);
  ```

  合計は拡張for文で各要素を `sum` に加算して求める。平均を求めるときは `(double)` でキャストするか、`1.0 *` を掛けることでdouble型の除算にする。int同士の除算では小数点以下が切り捨てられてしまう。
