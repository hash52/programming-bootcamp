hint: |
  最大値を求めるには、最初の要素を仮の最大値として設定し、残りの要素と比較して大きい方に更新していく。最小値も同様の考え方で求められる。
explanation: |
  **解答例**：
  ```java
  int max = numbers[0];
  int min = numbers[0];
  for (int i = 1; i < numbers.length; i++) {
      if (numbers[i] > max) {
          max = numbers[i];
      }
      if (numbers[i] < min) {
          min = numbers[i];
      }
  }
  System.out.println("最大値: " + max);
  System.out.println("最小値: " + min);
  ```

  最大値・最小値のアルゴリズムでは、最初の要素を初期値として設定するのがポイントである。0を初期値にすると、全要素が負の場合に正しい結果が得られない。
