  correct: ["C"]
explanation: |
  Javaで配列を宣言と同時に初期化するには、**波括弧 `{}`** を使う。`int[] numbers = {1, 2, 3};` のように書く。丸括弧 `()` や角括弧 `[]` で初期値を囲むのは構文エラーである。`new` キーワードを使った `int[] numbers = new int[]{1, 2, 3};` という書き方も正しい。
