explanation: |
  Javaの配列のインデックス（添え字）は **0から始まる**。要素数が5の配列であれば、インデックスは0〜4となる。`array[0]` が最初の要素、`array[4]` が最後の要素である。存在しないインデックスにアクセスすると `ArrayIndexOutOfBoundsException` が発生する。
