explanation: |
  配列の要素数を取得するには **`配列名.length`** を使う。`length` はメソッドではなく **フィールド** であるため、丸括弧 `()` を付けない。`length()` はString（文字列）のメソッド、`size()` はArrayListなどのコレクションのメソッドである。
