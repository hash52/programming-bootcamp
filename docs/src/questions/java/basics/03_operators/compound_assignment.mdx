explanation: |
  **複合代入演算子**は、**演算と代入を同時に行う演算子**である。

  主な複合代入演算子：
  - `+=`：加算して代入（`a += 5`は`a = a + 5`と同じ）
  - `-=`：減算して代入（`a -= 3`は`a = a - 3`と同じ）
  - `*=`：乗算して代入（`a *= 2`は`a = a * 2`と同じ）
  - `/=`：除算して代入（`a /= 4`は`a = a / 4`と同じ）
  - `%=`：剰余演算して代入（`a %= 3`は`a = a % 3`と同じ）

  **メリット**：
  - コードが短く、読みやすくなる
  - 変数名を2回書かなくて済む

  例：
  ```java
  int score = 100;
  score += 50;  // score = score + 50; と同じ
  System.out.println(score);  // 150
  ```

## 解説

複合代入演算子を使うと、**演算と代入を1行で簡潔に書ける**。

**通常の書き方と複合代入演算子の比較**：

| 通常の書き方 | 複合代入演算子 |
|-------------|---------------|
| `count = count + 5` | `count += 5` |
| `count = count - 3` | `count -= 3` |
| `count = count * 2` | `count *= 2` |
| `count = count / 4` | `count /= 4` |

- 商品の在庫減少：`stock -= soldCount;`
