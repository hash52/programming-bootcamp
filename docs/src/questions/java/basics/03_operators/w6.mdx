explanation: |
  **剰余演算**（`%`）は、**割り算の余り**を求める演算である。

  例：
  ```java
  int a = 10 % 3;  // 10を3で割った余りは1
  int b = 15 % 4;  // 15を4で割った余りは3
  int c = 20 % 5;  // 20を5で割った余りは0（割り切れる）
  ```

  **剰余演算の使い所**：
  - 偶数・奇数の判定（`n % 2 == 0`なら偶数）
  - 特定の倍数の判定（`n % 3 == 0`なら3の倍数）
  - 時計の計算（24時を超えたら0時に戻る）
  - 配列のインデックスの循環

  **ポイント**：
  - 割り切れる場合、余りは`0`
  - 余りは常に割る数より小さい（`10 % 3`の余りは0〜2の範囲）

## 問題

剰余演算を使って、数値が偶数か奇数かを判定するプログラムを書いて実行してください。

**要件**：
- 変数`number`に任意の整数を代入する
- `number`を2で割った余りを求める
- 余りが0なら「偶数」、余りが1なら「奇数」と表示する（まだif文を学んでいないので、両方表示して確認）

## 解答例

<OneCompilerCodeBlock
  language="java"
  code={`public class Main {
    public static void main(String[] args) {
        int number = 7;

        int remainder = number % 2;
        System.out.println(number + "を2で割った余り: " + remainder);

        // 別の例
        int number2 = 10;
        int remainder2 = number2 % 2;
        System.out.println(number2 + "を2で割った余り: " + remainder2);
    }
}`}
/>

**実行結果**：
```
7を2で割った余り: 1
10を2で割った余り: 0
```
