  **インクリメント演算子**（`++`）と**デクリメント演算子**（`--`）は、値を1増やしたり減らしたりする演算子である。

  - `count++`：countを1増やす（`count = count + 1`と同じ）
  - `count--`：countを1減らす（`count = count - 1`と同じ）

  例：
  ```java
  int count = 0;
  count++;  // countが1になる
  count++;  // countが2になる
  count--;  // countが1になる
  ```

  **インクリメント・デクリメント演算子の使い所**：
  - カウンターを1ずつ増やす
  - ループで回数を数える
  - 在庫数を1減らす

## 解説

インクリメント・デクリメント演算子を使うと、**値を1増減させる処理を簡潔に書ける**。

**通常の書き方との比較**：

| 通常の書き方 | インクリメント・デクリメント |
|-------------|---------------------------|
| `count = count + 1` | `count++` |
| `count = count - 1` | `count--` |

**実用例**：
- ゲームのスコア加算：`score++;`
- 商品の在庫減少：`stock--;`
- ループカウンター：`i++`（for文で頻繁に使う）

**補足**：
- 前置（`++count`）と後置（`count++`）の違いがあるが、単独で使う場合は同じ
- 式の中で使う場合のみ、前置と後置で動作が異なる（詳細は応用編で学習）
