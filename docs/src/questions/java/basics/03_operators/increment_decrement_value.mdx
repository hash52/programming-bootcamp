explanation: |
  **インクリメント演算子**（`++`）と **デクリメント演算子**（`--`）は、値を1増やしたり減らしたりする演算子である。

  - `count++`：countを1増やす（`count = count + 1` と同じ）
  - `count--`：countを1減らす（`count = count - 1` と同じ）

  ```java
  int count = 0;
  count++;  // countが1になる
  count++;  // countが2になる
  count--;  // countが1になる
  ```

  **補足**：
  - 前置（`++count`）と後置（`count++`）の違いがあるが、単独で使う場合は同じ
  - 式の中で使う場合のみ、前置と後置で動作が異なる（詳細は応用編で学習）
