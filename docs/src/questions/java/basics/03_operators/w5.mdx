explanation: |
  **四則演算**とは、足し算（`+`）、引き算（`-`）、掛け算（`*`）、割り算（`/`）の4つの計算である。

  Javaでは以下の演算子を使う：
  - `+`：足し算
  - `-`：引き算
  - `*`：掛け算
  - `/`：割り算

  **ポイント**：
  - 掛け算は`*`（アスタリスク）を使う（`×`ではない）
  - 割り算は`/`（スラッシュ）を使う（`÷`ではない）
  - 計算の優先順位は数学と同じ（掛け算・割り算が先、足し算・引き算が後）
  - カッコ`()`を使って優先順位を変えられる

  **注意点**：
  - 整数同士の割り算は小数点以下が切り捨てられる（例：`5 / 2`は`2`）
  - 小数を含む計算にする場合は、`double`型を使う

## 問題

2つの商品の価格と消費税を計算するプログラムを書いて実行してください。

**要件**：
- 商品Aの価格を`100`、商品Bの価格を`200`とする
- 合計金額を計算する
- 消費税（10%）を計算する（合計金額の10%）
- 税込み金額を計算する（合計金額 + 消費税）
- 各金額を表示する

**実行環境**：
- 以下のOneCompiler（ブラウザ上で実行可能）
- または、Eclipse等のIDE

<OneCompilerCodeBlock
  language="java"
  code={`public class Main {
    public static void main(String[] args) {
        // ここにコードを書いてください

    }
}`}
/>

**ヒント**：
- 合計金額 = 商品A + 商品B
- 消費税 = 合計金額 * 10 / 100
- 税込み金額 = 合計金額 + 消費税
- `System.out.println()` で結果を表示

## 解答例

<OneCompilerCodeBlock
  language="java"
  code={`public class Main {
    public static void main(String[] args) {
        int priceA = 100;
        int priceB = 200;

        int total = priceA + priceB;
        System.out.println("合計金額: " + total + "円");

        int tax = total * 10 / 100;
        System.out.println("消費税: " + tax + "円");

        int totalWithTax = total + tax;
        System.out.println("税込み金額: " + totalWithTax + "円");
    }
}`}
/>

**実行結果**：
```
合計金額: 300円
消費税: 30円
税込み金額: 330円
```
