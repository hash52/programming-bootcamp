explanation: |
  **比較演算子**は、**値の大小や等しいかどうかを比較し、結果を真偽値（true/false）で返す演算子**である。

  主な比較演算子：
  - `==`（等しい）
  - `!=`（等しくない）
  - `<`（より小さい）
  - `>`（より大きい）
  - `<=`（以下）
  - `>=`（以上）

  例：
  ```java
  int age = 20;
  boolean isAdult = age >= 18;  // true
  boolean isTeenager = age < 20;  // false
  ```

## 問題

比較演算子の説明として最も適切なものを選べ。

## 解説

比較演算子は、**値を比較して、その結果を真偽値（true/false）で返す演算子**である。

プログラムでは、以下のような判定を行うことが多い：
- ユーザーの年齢が18歳以上か
- 入力された値が正の数か
- パスワードが一致するか

このような判定に比較演算子を使う。

import { OneCompilerCodeBlock } from '@site/src/components/OneCompilerCodeBlock';

<OneCompilerCodeBlock language="java" code={`public class Main {
    public static void main(String[] args) {
        int score = 85;

        boolean isPerfect = score == 100;
        System.out.println("満点か: " + isPerfect);  // false

        boolean isPassing = score >= 60;
        System.out.println("合格か: " + isPassing);  // true

        boolean isFailing = score < 60;
        System.out.println("不合格か: " + isFailing);  // false
    }
}`} />

**重要な注意点**：
- **等しいかどうかの比較は`==`（2つのイコール）**を使う
- `=`（1つのイコール）は代入演算子なので、比較には使えない
