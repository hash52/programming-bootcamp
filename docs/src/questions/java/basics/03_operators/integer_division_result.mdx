hint: |
  **同じ型同士の演算結果は同じ型になる** というJavaのルールを思い出そう。
  `int` 同士の計算結果は何型になるだろうか？
explanation: |
  Javaでは、 **整数同士の割り算の結果は整数になり、小数点以下は切り捨てられる** 。

  ```java
  int a = 5 / 2;  // 2（2.5ではない）
  int b = 7 / 3;  // 2（2.333...ではない）
  int c = 10 / 4;  // 2（2.5ではない）
  ```

  **小数を含む計算にする方法**：
  - `double` 型を使う
  - 少なくとも1つの数値を `double` 型にする

  ```java
  double d = 5.0 / 2;  // 2.5
  double e = 5 / 2.0;  // 2.5
  double f = (double) 5 / 2;  // 2.5（キャスト）
  ```

  **よくある間違い**：
  ```java
  int a = 5;
  int b = 2;
  double result = a / b;  // これは2.0になる（2.5ではない）
  ```
  `a / b` の時点で整数の割り算（結果2）になり、その後 `double` 型に変換されるため `2.0` になる。正しくは `(double) a / b` とする。
