hint: |
  1行ずつ変数の値を追っていこう。 `+=` は「加算して代入」、 `-=` は「減算して代入」である。

  | ステップ | コード | a | b |
  |---------|--------|---|---|
  | 1 | `int a = 5;` | 5 | - |
  | 2 | `int b = 10;` | 5 | 10 |
  | 3 | `a += 5;` | ? | 10 |
  | ... | ... | ... | ... |
explanation: |
  このコードを1行ずつ追っていくと：

  1. `int a = 5;` → aは5
  2. `int b = 10;` → bは10
  3. `a += 5;` → aは `5 + 5` で10になる
  4. `b -= 3;` → bは `10 - 3` で7になる
  5. `int result = a + b;` → resultは `10 + 7` で17になる
  6. `result += 3;` → resultは `17 + 3` で **20** になる

  **変数の値の変化を表にまとめる**：

  | ステップ | コード | a | b | result |
  |---------|--------|---|---|--------|
  | 1 | `int a = 5;` | 5 | - | - |
  | 2 | `int b = 10;` | 5 | 10 | - |
  | 3 | `a += 5;` | 10 | 10 | - |
  | 4 | `b -= 3;` | 10 | 7 | - |
  | 5 | `int result = a + b;` | 10 | 7 | 17 |
  | 6 | `result += 3;` | 10 | 7 | **20** |

  **コードを読む際のコツ**：
  - 変数の初期値を確認する
  - 1行ずつ実行して、変数がどう変化するかを追う
  - 複雑な式は小さな部分に分けて考える
  - 表やメモを使って変数の値を記録する

以下のJavaコードを実行した場合、最後に変数 `result` に格納される値はいくつか。

```java
int a = 5;
int b = 10;
a += 5;
b -= 3;
int result = a + b;
result += 3;
System.out.println(result);
```
