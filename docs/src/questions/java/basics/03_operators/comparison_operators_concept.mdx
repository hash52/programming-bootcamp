explanation: |
  **比較演算子** は、 **値の大小や等しいかどうかを比較し、結果を真偽値（true/false）で返す演算子** である。

  主な比較演算子：
  - `==`（等しい）
  - `!=`（等しくない）
  - `<`（より小さい）
  - `>`（より大きい）
  - `<=`（以下）
  - `>=`（以上）

  ```java
  int age = 20;
  boolean isAdult = age >= 18;  // true
  boolean isTeenager = age < 20;  // false
  ```

  **重要な注意点**：
  - 等しいかどうかの比較は `==`（2つのイコール）を使う
  - `=`（1つのイコール）は代入演算子なので、比較には使えない
---

比較演算子の説明として最も適切なものを選べ。
