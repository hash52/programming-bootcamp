## 解説

否定演算子（`!`）を使うと、**真偽値を反転できる**。

**否定演算子の真理値表**：

| A | !A |
|---|----|
| true | false |
| false | true |

    public static void main(String[] args) {

        int age = 15;

**論理演算子との組み合わせ**：

```java
boolean hasLicense = false;
int age = 20;

// 年齢が18歳以上 かつ 免許を持っていない
boolean needsLicense = age >= 18 && !hasLicense;  // true
```

**実用例**：
- エラーチェック：「入力が空でない」 → `!isEmpty`
- 権限チェック：「管理者でない」 → `!isAdmin`
- 状態確認：「完了していない」 → `!isCompleted`
