  主な比較演算子：
  - `==`：等しい
  - `!=`：等しくない
  - `<`：より小さい
  - `>`：より大きい
  - `<=`：以下
  - `>=`：以上

  ```java
  int age = 20;
  boolean isAdult = age >= 18;  // true
  boolean isChild = age < 13;   // false
  boolean isTwenty = age == 20; // true
  ```

## 解説

比較演算子を使うと、**条件を判定して真偽値を得ることができる**。

**比較演算子の一覧**：

| 演算子 | 意味 | 例 | 結果 |
|--------|------|-----|------|
| `==` | 等しい | `5 == 5` | true |
| `!=` | 等しくない | `5 != 3` | true |
| `<` | より小さい | `3 < 5` | true |
| `>` | より大きい | `5 > 3` | true |
| `<=` | 以下 | `5 <= 5` | true |
| `>=` | 以上 | `5 >= 3` | true |
