explanation: |
  **比較演算子** を使うと、値を比較して真偽値（true/false）を得ることができる。

  主な比較演算子：
  - `==`：等しい
  - `!=`：等しくない
  - `<`：より小さい
  - `>`：より大きい
  - `<=`：以下
  - `>=`：以上

  ```java
  int age = 20;
  boolean isAdult = age >= 18;  // true
  boolean isChild = age < 13;   // false
  boolean isTwenty = age == 20; // true
  ```

  **重要な注意点**：
  - 等しいかどうかの比較は `==`（2つのイコール）
  - `=`（1つのイコール）は代入演算子なので、比較には使えない
