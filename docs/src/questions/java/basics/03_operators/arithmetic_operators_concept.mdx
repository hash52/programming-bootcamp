explanation: |
  **算術演算子**は、**四則演算や剰余演算を行う演算子**である。

  主な算術演算子：
  - `+`（足し算）
  - `-`（引き算）
  - `*`（掛け算）
  - `/`（割り算）
  - `%`（剰余：割り算の余り）

  例：
  ```java
  int a = 10 + 5;  // 15
  int b = 10 - 5;  // 5
  int c = 10 * 5;  // 50
  int d = 10 / 5;  // 2
  int e = 10 % 3;  // 1（10を3で割った余り）
  ```

## 問題

算術演算子の説明として最も適切なものを選べ。

## 解説

算術演算子は、**数値の計算を行うための演算子**である。

プログラムでは、以下のような計算を頻繁に行う：
- ユーザーの入力値を合計する
- 商品の価格を計算する
- ゲームのスコアを加算する

このような計算に算術演算子を使う。

**よくある間違い**：
- `*`（掛け算）を`×`と書いてしまう
- `/`（割り算）を`÷`と書いてしまう
- `%`（剰余）の意味を理解していない
