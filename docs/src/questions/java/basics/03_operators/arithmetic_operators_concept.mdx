explanation: |
  **算術演算子** は、 **四則演算や剰余演算を行う演算子** である。

  主な算術演算子：
  - `+`（足し算）
  - `-`（引き算）
  - `*`（掛け算）
  - `/`（割り算）
  - `%`（剰余：割り算の余り）

  ```java
  int a = 10 + 5;  // 15
  int b = 10 - 5;  // 5
  int c = 10 * 5;  // 50
  int d = 10 / 5;  // 2
  int e = 10 % 3;  // 1（10を3で割った余り）
  ```

  **よくある間違い**：
  - `*`（掛け算）を `×` と書いてしまう
  - `/`（割り算）を `÷` と書いてしまう
  - `%`（剰余）の意味を理解していない
---

算術演算子の説明として最も適切なものを選べ。
