explanation: |
  **否定演算子**（`!`）は、 **条件を反転させる演算子** である。

  - `true` を `false` に変換
  - `false` を `true` に変換

  ```java
  boolean isRaining = false;
  boolean isNotRaining = !isRaining;  // true

  boolean hasLicense = true;
  boolean hasNoLicense = !hasLicense;  // false
  ```

  **否定演算子の真理値表**：

  | A | !A |
  |---|----|
  | true | false |
  | false | true |

  **実用例**：
  - エラーチェック：「入力が空でない」 → `!isEmpty`
  - 権限チェック：「管理者でない」 → `!isAdmin`
  - 状態確認：「完了していない」 → `!isCompleted`
