explanation: |
  **インクリメント演算子**（`++`）は値を1増やし、 **デクリメント演算子**（`--`）は値を1減らす演算子である。

  ```java
  int count = 0;
  count++;  // countが1増える（count = count + 1と同じ）
  System.out.println(count);  // 1

  count--;  // countが1減る（count = count - 1と同じ）
  System.out.println(count);  // 0
  ```

  **インクリメント・デクリメント演算子の使い所**：
  - カウンターを1ずつ増やす
  - ループで回数を数える
  - 在庫数を1減らす
