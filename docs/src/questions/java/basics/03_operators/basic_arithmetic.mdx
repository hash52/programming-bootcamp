explanation: |
  **四則演算** とは、足し算（`+`）、引き算（`-`）、掛け算（`*`）、割り算（`/`）の4つの計算である。

  **ポイント**：
  - 掛け算は `*`（アスタリスク）を使う（`×` ではない）
  - 割り算は `/`（スラッシュ）を使う（`÷` ではない）
  - 計算の優先順位は数学と同じ（掛け算・割り算が先、足し算・引き算が後）
  - カッコ `()` を使って優先順位を変えられる

  **注意点**：
  - 整数同士の割り算は小数点以下が切り捨てられる（例：`5 / 2` は `2`）

  **解答例**：
  ```java
  public class Main {
      public static void main(String[] args) {
          int priceA = 100;
          int priceB = 200;

          int total = priceA + priceB;
          System.out.println("合計金額: " + total + "円");

          int tax = total * 10 / 100;
          System.out.println("消費税: " + tax + "円");

          int totalWithTax = total + tax;
          System.out.println("税込み金額: " + totalWithTax + "円");
      }
  }
  ```
