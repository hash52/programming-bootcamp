explanation: |
  ループとは、同じ処理や似た処理を **繰り返し実行** する仕組みである。Javaではfor文、while文、do-while文を使ってループを実現する。条件による切り替えは条件分岐（if文）の役割である。
---

ループ（繰り返し）の目的として最も適切なものを選べ。
