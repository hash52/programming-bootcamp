answers:
  correct: ["A"]
explanation: |
  `while (true)` は条件が常に `true` であるため、ループが永遠に終わらない **無限ループ** になる。無限ループは `break` 文と組み合わせて使うことがある。`while (false)` は条件が常にfalseなので一度も実行されない。
