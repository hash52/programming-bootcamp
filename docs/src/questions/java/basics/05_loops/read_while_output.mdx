explanation: |
  変数nの変化を追跡する。

  | ステップ | nの値（ループ開始時） | n > 1 | n / 2 | 出力 |
  |---------|---------------------|-------|-------|------|
  | 1 | 10 | true | 5 | 5 |
  | 2 | 5 | true | 2 | 2 |
  | 3 | 2 | true | 1 | 1 |
  | 4 | 1 | false | - | 終了 |

  int型の除算は **小数点以下が切り捨て** られる（5 / 2 = 2）。nが1になると条件 `n > 1` がfalseになりループが終了する。

以下のJavaコードを実行した場合、何が出力されるか。

```java
int n = 10;
while (n > 1) {
    n = n / 2;
    System.out.println(n);
}
```
