explanation: |
  for文の3つの部分はそれぞれ以下の役割を持つ。

  - **初期化** `int i = 0`: カウンタ変数iを0で初期化
  - **条件式** `i < 5`: iが5未満の間、繰り返す（0,1,2,3,4の5回）
  - **更新処理** `i++`: 1回の繰り返しごとにiを1増やす
