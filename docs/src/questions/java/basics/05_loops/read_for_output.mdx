  ループ変数iの変化と出力を追跡する。

  | 繰り返し | iの値 | i * 2 | 出力 |
  |---------|-------|-------|------|
  | 1回目 | 1 | 2 | 2 |
  | 2回目 | 2 | 4 | 4 |
  | 3回目 | 3 | 6 | 6 |
  | 4回目 | i=4, 4 <= 3 はfalse | - | 終了 |

  各繰り返しで `i * 2` の結果が出力されるため、**2, 4, 6** が順に表示される。

以下のJavaコードを実行した場合、何が出力されるか。

```java
for (int i = 1; i <= 3; i++) {
    System.out.println(i * 2);
}
```
