  **解答例**：
  ```java
  while (count <= 5) {
      System.out.println("count: " + count);
      count++;
  }
  ```

  while文では条件式 `count <= 5` がtrueの間、ループ内の処理が繰り返される。**カウンタ変数の更新（`count++`）を忘れると無限ループ** になるため注意が必要である。
