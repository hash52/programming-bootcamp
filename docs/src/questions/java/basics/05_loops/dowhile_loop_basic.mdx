hint: |
  do-while文の構文は `do { 処理; } while (条件式);` である。while文と違い、**条件の判定がループの最後** にある点に注目しよう。
explanation: |
  **解答例**：
  ```java
  do {
      System.out.println(num);
      num--;
  } while (num > 0);
  ```

  do-while文は `do { }` 内の処理を **最低1回実行した後** に条件を判定する。仮に `num` が0以下で始まっても、1回は実行される点がwhile文との違いである。末尾の `while (条件式);` のセミコロンを忘れないこと。
