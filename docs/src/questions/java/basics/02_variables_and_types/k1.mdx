explanation: |
  変数とは、**データを一時的に保存するための箱**のようなものです。

  プログラム実行中に値を保持し、後で取り出して使うことができます。

  例えば、以下のコードでは変数`age`に値`25`を保存しています。

  ```java
  int age = 25;
  System.out.println(age); // 25が出力される
  ```

  このように、変数に保存した値を後から取り出して使うことができます。
