explanation: |
  **double型**で小数を扱うには、変数宣言時に `double` を指定します。

  ```java
  double pi = 3.14;
  ```

  - `double`: 小数型の型名
  - `pi`: 変数名
  - `3.14`: 小数値

  **double型で扱える値の例**：
  - 小数：`3.14`, `-0.5`, `99.99`
  - 整数も代入可能：`100.0`, `0.0`
  - 非常に大きな数や小さな数も扱える

  **使い分け**：
  - 小数が必要な場合（価格、温度など）：`double`
  - 整数のみの場合（年齢、カウント数など）：`int`
