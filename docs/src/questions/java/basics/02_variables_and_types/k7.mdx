explanation: |
  **代入**とは、**変数に値を格納する**操作のことです。

  `=`（代入演算子）を使って、右側の値を左側の変数に入れます。

  ```java
  int age = 25;       // 変数ageに25を代入
  age = 30;           // 変数ageの値を30に更新
  String name = "田中"; // 変数nameに"田中"を代入
  ```

  **代入の仕組み**：
  - `変数名 = 値;` の形で書く
  - **右から左**に値が流れる（`=` は「等しい」ではなく「代入」）
  - 同じ変数に何度でも代入できる（値を更新できる）

  **注意**：
  - `25 = age;` はエラー（左側は必ず変数）
  - `=` は数学の等号ではなく、代入を意味する
  - 代入前に変数を宣言する必要がある
