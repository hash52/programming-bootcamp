  変数を宣言するには、まず型（この場合は`int`）を指定し、次に変数名、そして`=`で値を代入します。

  ```java
  int age = 25;
  ```

  **解答の説明：**
  - `int`：整数型を表す
  - `age`：変数名
  - `25`：代入する値

  変数宣言の基本構文は `型 変数名 = 値;` である。
