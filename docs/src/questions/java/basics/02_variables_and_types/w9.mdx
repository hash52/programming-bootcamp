explanation: |
  **int型**で整数を扱うには、変数宣言時に `int` を指定します。

  ```java
  int score = 42;
  ```

  - `int`: 整数型の型名
  - `score`: 変数名
  - `42`: 整数値

  **int型で扱える値の例**：
  - 正の整数：`1`, `100`, `2024`
  - 負の整数：`-1`, `-100`, `-2024`
  - ゼロ：`0`
  - 小数は扱えない（`int x = 3.14;` はエラー）
