  **自分の情報を表示するプログラム**を書いてみよう。

  **ポイント**：
  - `System.out.println()`で1行ずつ表示
  - 文字列はダブルクォート`"`で囲む
  - 各行の最後にセミコロン`;`を付ける
  - 日本語も使える

  **注意点**：
  - クラス名は`Main`のまま（OneCompilerでは固定）
  - `main`メソッドの中にコードを書く
  - エラーが出たら、セミコロンやダブルクォートが正しいか確認

  **解答例**：
  ```java
  public class Main {
      public static void main(String[] args) {
          System.out.println("名前: 田中太郎");
          System.out.println("年齢: 25歳");
      }
  }
  ```

  **実行結果**：
  ```
  名前: 田中太郎
  年齢: 25歳
  ```

  **応用例**：
  ```java
  public class Main {
      public static void main(String[] args) {
          System.out.println("=== 自己紹介 ===");
          System.out.println("名前: 田中太郎");
          System.out.println("年齢: 25歳");
          System.out.println("趣味: プログラミング");
      }
  }
  ```
