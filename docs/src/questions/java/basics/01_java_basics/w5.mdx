explanation: |
  **「Hello World」を画面に表示する**には、`System.out.println()`を使う。

  ```java
  System.out.println("Hello World");
  ```

  **コードの説明**：
  - `System.out.println()`: 画面に文字を表示し、改行するメソッド
  - `"Hello World"`: 表示したい文字列（ダブルクォートで囲む）
  - 最後に`;`（セミコロン）を付ける

  **実行結果**：
  ```
  Hello World
  ```

  **println と print の違い**：
  - `println`: 表示後に改行する
  - `print`: 改行しない

  ```java
  System.out.print("Hello ");
  System.out.print("World");
  // 出力: Hello World（改行なし）

  System.out.println("Hello ");
  System.out.println("World");
  // 出力:
  // Hello 
  // World
  ```

  **注意**：
  - 文字列は必ずダブルクォート `"` で囲む
  - 日本語も表示できる：`System.out.println("こんにちは");`
  - 最後の`;`（セミコロン）を忘れないこと
