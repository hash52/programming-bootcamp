explanation: |
  **EclipseでJavaプログラムを実行する**には、いくつかの方法がある。

  **方法1：実行ボタンを使う**
  1. 画面上部のツールバーにある**緑の三角形ボタン**（実行ボタン）をクリック
  2. プログラムが実行され、コンソールに結果が表示される

  **方法2：キーボードショートカットを使う**
  - **Ctrl + F11**（Windows/Linux）
  - **⌘ + F11**（Mac）

  **方法3：右クリックメニューを使う**
  1. エディタ内で右クリック
  2. 「実行」→「Javaアプリケーション」を選択

  **実行時の注意点**：
  - `main`メソッドがあるクラスでないと実行できない
  - エラーがある場合は実行できない（赤い波線がないか確認）
  - 初回実行時は実行構成の確認ダイアログが表示されることがある

  **コンソールで結果を確認**：
  - 画面下部の「コンソール」タブに実行結果が表示される
  - `System.out.println()`で出力した内容がここに表示される

  **エラーが出た場合**：
  - コンソールに赤字でエラーメッセージが表示される
  - エラーメッセージを読んで、原因を特定する
  - 行番号が表示されるので、該当行を確認

  **よくある問題**：
  - セミコロン`;`を忘れている
  - ダブルクォート`"`を閉じ忘れている
  - 波括弧`{}`が対応していない
  - クラス名とファイル名が一致していない
