explanation: |
  **読みやすいコード**を書くことは、プログラミングで非常に重要である。

  **良いコードの特徴**：

  1. **インデント（字下げ）を揃える**
     ```java
     // ✅ 良い例
     if (age >= 20) {
         System.out.println("成人です");
     }

     // ❌ 悪い例
     if (age >= 20) {
     System.out.println("成人です");
     }
     ```

  2. **適切に改行する**
     ```java
     // ✅ 良い例
     int age = 20;
     String name = "田中";
     System.out.println(name + "さんは" + age + "歳です");

     // ❌ 悪い例
     int age = 20; String name = "田中"; System.out.println(name + "さんは" + age + "歳です");
     ```

  3. **適度にスペースを入れる**
     ```java
     // ✅ 良い例
     int result = a + b;

     // ❌ 悪い例
     int result=a+b;
     ```

  **なぜ読みやすさが重要か**：
  - 他の人がコードを理解しやすくなる
  - 自分が後で見返したときに理解しやすい
  - バグを見つけやすくなる
  - チーム開発では必須のスキル
---

読みやすいコードを書くために重要なことを **すべて** 選べ。
