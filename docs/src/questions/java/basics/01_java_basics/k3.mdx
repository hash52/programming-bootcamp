explanation: |
  **波括弧（`{}`）**は、**コードのブロック（範囲）を示す**ために使われる。

  波括弧で囲まれた部分は、1つのまとまりとして扱われる。

  **使用例**：

  1. **クラスの範囲**
     ```java
     public class Main {
         // この中がMainクラスの範囲
     }
     ```

  2. **メソッドの範囲**
     ```java
     public static void main(String[] args) {
         // この中がmainメソッドの範囲
     }
     ```

  3. **if文の範囲**
     ```java
     if (age >= 20) {
         // この中が条件が真の時に実行される範囲
         System.out.println("成人です");
     }
     ```

  4. **ループの範囲**
     ```java
     for (int i = 0; i < 3; i++) {
         // この中が繰り返される範囲
         System.out.println(i);
     }
     ```

  **重要なポイント**：
  - 開き波括弧`{`と閉じ波括弧`}`は必ずペアになる
  - 波括弧の中にさらに波括弧を入れることができる（ネスト）
  - 波括弧で囲まれた範囲を「ブロック」と呼ぶ
