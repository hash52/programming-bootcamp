explanation: |
  **Randomクラス**は、**乱数を生成するためのクラス**である。

  **Randomクラスの主なメソッド**：
  - `nextInt(n)`：0〜n-1のランダムな整数を返す
  - `nextDouble()`：0.0〜1.0のランダムな小数を返す
  - `nextBoolean()`：true or false をランダムに返す

  **Randomクラスの使い方**：
  ```java
  import java.util.Random;

  Random random = new Random();
  int number = random.nextInt(10);  // 0〜9のランダムな整数
  ```

  **Randomクラスの利点**：
  - 簡単に乱数を生成できる
  - さまざまな型の乱数を生成できる（整数、小数、真偽値）
  - シード値を指定して再現性を持たせることもできる

## 問題

Randomクラスの役割として最も適切なものを選べ。

## 解説

Randomクラスは、**プログラムで乱数を生成するための便利なツール**である。

Randomクラスを使うことで、以下のような処理が簡単に実装できる：
- サイコロやガチャのランダムな結果
- ランダムなパスワードやトークンの生成
- テストデータのランダム生成
- ゲームの敵の行動パターン

public class Main {
    public static void main(String[] args) {
        Random random = new Random();

        // ランダムな整数（0〜99）

    }
}`} />

**Randomクラスの使用には`import`が必要**：
```java
import java.util.Random;  // ファイルの先頭に記述
```

**補足**：
- Javaには`Math.random()`という別の乱数生成方法もあるが、Randomクラスの方が機能が豊富で便利
