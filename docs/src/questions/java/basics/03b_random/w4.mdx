  **Randomクラス**を使うと、ランダムな数を簡単に生成できる。

  **基本的な流れ**：
  1. `Random`オブジェクトを作成
  2. `nextInt(n)`メソッドで0〜n-1のランダムな整数を取得

  **ポイント**：
  - `import java.util.Random;`でRandomクラスをインポート
  - `Random random = new Random();`でRandomオブジェクトを作成
  - `random.nextInt(n)`で乱数を取得

## 問題

サイコロを振るプログラム（1〜6のランダムな整数を表示）を書いて実行してください。

**要件**：
- 「サイコロを振ります...」と表示
- 1〜6のランダムな整数を生成
- 「出目: 〇」と表示

**実行環境**：
- 以下のOneCompiler（ブラウザ上で実行可能）
- または、Eclipse等のIDE

<OneCompilerCodeBlock
  language="java"
  code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        // ここにコードを書いてください

    }
}`}
/>

**ヒント**：
- `Random random = new Random();`でRandomオブジェクトを作成
- `1 + random.nextInt(6)`で1〜6のランダムな整数

---

## 解答例

<OneCompilerCodeBlock
  language="java"
  code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random random = new Random();

        System.out.println("サイコロを振ります...");
        int dice = 1 + random.nextInt(6);
        System.out.println("出目: " + dice);
    }
}`}
/>

**実行例**：
```
サイコロを振ります...
出目: 4
```
（毎回異なる値が表示される）

**応用例**：

<OneCompilerCodeBlock
