## 解説

0〜9のランダムな整数を生成するには、**nextInt(10)**を使う。

**範囲と引数の関係**：

| 目的の範囲 | nextIntの引数 | 理由 |
|-----------|--------------|------|
| 0〜9 | `nextInt(10)` | 10個の値（0, 1, ..., 9） |
| 0〜4 | `nextInt(5)` | 5個の値（0, 1, 2, 3, 4） |
| 0〜99 | `nextInt(100)` | 100個の値（0, 1, ..., 99） |
