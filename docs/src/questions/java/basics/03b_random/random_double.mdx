  ```java
  Random random = new Random();
  double number = random.nextDouble();  // 0.0〜1.0未満
  ```

  **nextDouble()の特徴**：
  - 引数は不要
  - 0.0以上1.0未満の範囲（1.0は含まれない）
  - 例：0.0, 0.5, 0.9999...など

  **範囲を変える方法**：
  - **0.0〜10.0**：`random.nextDouble() * 10`
  - **0.0〜100.0**：`random.nextDouble() * 100`
  - **5.0〜10.0**：`5.0 + random.nextDouble() * 5`

## 解説

0.0〜1.0のランダムな小数を生成するには、**nextDouble()**を使う。

**nextInt()とnextDouble()の違い**：

| メソッド | 戻り値の型 | 範囲 | 引数 |
|----------|-----------|------|------|
| `nextInt(n)` | int | 0〜n-1 | 必要 |
| `nextDouble()` | double | 0.0〜1.0未満 | 不要 |

<OneCompilerCodeBlock language="java" code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random random = new Random();

        System.out.println("=== 0.0〜1.0のランダムな小数を5個生成 ===");
        for (int i = 0; i < 5; i++) {
            double number = random.nextDouble();
            System.out.println(number);
        }

        System.out.println("\\n=== 0.0〜100.0のランダムな小数を3個生成 ===");
        for (int i = 0; i < 3; i++) {
            double number = random.nextDouble() * 100;
            System.out.printf("%.2f\\n", number);
        }
    }
}`} />
