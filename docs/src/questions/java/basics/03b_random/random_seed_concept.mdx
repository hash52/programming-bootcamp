explanation: |
  **シード値**（Seed）を指定すると、**同じ乱数列を再現できる**。

  **シード値とは**：
  - 乱数生成のスタート地点を決める値
  - 同じシード値を使うと、毎回同じ順序で乱数が生成される

  **シード値の指定方法**：
  ```java
  Random random = new Random(42);  // シード値42
  ```

  **シード値を使わない場合**：
  ```java
  Random random = new Random();  // 現在時刻をシード値として使う
  ```

  **シード値の使い所**：
  - **テスト**：同じ結果を再現したい場合
  - **デバッグ**：問題を再現しやすくする
  - **ゲーム**：ステージ生成を再現可能にする

## 問題

乱数の再現性を制御する方法として最も適切なものを選べ。

## 解説

シード値を指定することで、**同じ乱数列を何度でも再現できる**。

import { OneCompilerCodeBlock } from '@site/src/components/OneCompilerCodeBlock';

**シード値を使わない例（毎回異なる）**：

<OneCompilerCodeBlock language="java" code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random random = new Random();  // シード値なし

        System.out.println("=== 実行1回目 ===");
        for (int i = 0; i < 3; i++) {
            System.out.println(random.nextInt(10));
        }
        // 実行するたびに異なる値が表示される
    }
}`} />

**シード値を使う例（毎回同じ）**：

<OneCompilerCodeBlock language="java" code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random random = new Random(42);  // シード値42

        System.out.println("=== シード値42で生成 ===");
        for (int i = 0; i < 5; i++) {
            System.out.println(random.nextInt(10));
        }
        // 何度実行しても同じ順序で値が表示される
    }
}`} />

**比較**：

<OneCompilerCodeBlock language="java" code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        System.out.println("=== シード値100 ===");
        Random random1 = new Random(100);
        for (int i = 0; i < 3; i++) {
            System.out.println(random1.nextInt(10));
        }

        System.out.println("\\n=== シード値100（再現） ===");
        Random random2 = new Random(100);  // 同じシード値
        for (int i = 0; i < 3; i++) {
            System.out.println(random2.nextInt(10));  // 同じ値が出る
        }

        System.out.println("\\n=== シード値200（異なる） ===");
        Random random3 = new Random(200);  // 異なるシード値
        for (int i = 0; i < 3; i++) {
            System.out.println(random3.nextInt(10));  // 異なる値が出る
        }
    }
}`} />

**シード値の用途**：
- **テスト**：同じ条件で何度もテストしたい
- **デバッグ**：バグを再現しやすくする
- **ゲーム**：マップ生成を再現可能にする（マインクラフトのシード値など）
