explanation: |
  **1〜100のランダムな整数**を生成するには、`1 + random.nextInt(100)`を使う。

  ```java
  Random random = new Random();
  int number = 1 + random.nextInt(100);  // 1, 2, 3, ..., 100
  ```

  **考え方**：
  1. `random.nextInt(100)`は0〜99の範囲
  2. それに1を足すと1〜100になる

  **一般的なパターン**：
  - **a〜bの範囲**：`a + random.nextInt(b - a + 1)`
  - 例：1〜10 → `1 + random.nextInt(10)`
  - 例：5〜15 → `5 + random.nextInt(11)`
  - 例：1〜100 → `1 + random.nextInt(100)`

## 解答例

<OneCompilerCodeBlock
  language="java"
  code={`import java.util.Random;

public class Main {
    public static void main(String[] args) {
        Random random = new Random();

        int number = 1 + random.nextInt(100);
        System.out.println("ランダムな数: " + number);
    }
}`}
/>

**実行例**：
```
ランダムな数: 42
```
（毎回異なる値が表示される）
