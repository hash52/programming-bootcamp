## 解説

Randomクラスを使うには、**Randomクラスをインポートして、Randomオブジェクトを作成する**必要がある。

**手順の詳細**：

1. **import文を記述**
   ```java
   import java.util.Random;  // ファイルの先頭
   ```
   - `java.util`パッケージの`Random`クラスを使うことを宣言

2. **Randomオブジェクトを作成**
   ```java
   Random random = new Random();
   ```
   - `Random`：クラス名
   - `random`：変数名（任意だが、小文字で始めるのが慣習）
   - `new Random()`：Randomオブジェクトを作成

**よくある間違い**：
- ❌ `import Random;`（パッケージ名が必要）
- ✅ `import java.util.Random;`
- ❌ `Random random = Random();`（`new`が必要）
- ✅ `Random random = new Random();`
