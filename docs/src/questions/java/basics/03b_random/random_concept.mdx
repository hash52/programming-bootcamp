explanation: |
  **乱数**（Random Number）は、**予測できないランダムな数値**である。

  **乱数の使い所**：
  - ゲームのサイコロやカードのシャッフル
  - パスワードやトークンの生成
  - テストデータの作成
  - シミュレーションや統計処理

  **乱数の種類**：
  - **整数の乱数**：0, 1, 2, 3, ...
  - **小数の乱数**：0.0〜1.0の範囲

  例：
  ```java
  // 1〜6のサイコロ
  int dice = 1 + random.nextInt(6);  // 1, 2, 3, 4, 5, 6のいずれか
  ```

## 問題

乱数の説明として最も適切なものを選べ。

## 解説

乱数は、**プログラムに予測不可能性を持たせるための仕組み**である。

ゲームやセキュリティなど、さまざまな場面で乱数が使われる：
- **ゲーム**：サイコロ、ガチャ、敵の行動パターン
- **セキュリティ**：パスワード、認証トークン、暗号鍵
- **テスト**：ランダムなテストデータの生成
- **シミュレーション**：天気予報、株価予測

public class Main {
    public static void main(String[] args) {
        Random random = new Random();

        System.out.println("=== サイコロを振る ===");
        for (int i = 1; i <= 3; i++) {
            int dice = 1 + random.nextInt(6);
            System.out.println(i + "回目: " + dice);
        }

        // コイントス（表か裏か）
        System.out.println("\\n=== コイントス ===");
        int coin = random.nextInt(2);  // 0 or 1
        if (coin == 0) {
            System.out.println("表");
        } else {
            System.out.println("裏");
        }
    }
}`} />

**乱数の特徴**：
- 毎回異なる値が生成される（予測不可能）
- 同じプログラムを実行しても、結果は毎回変わる
- 公平性が保証される（特定の値に偏らない）
