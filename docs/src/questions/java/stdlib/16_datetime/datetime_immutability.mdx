explanation: |
  Java 8の日時クラス（LocalDate, LocalTime, LocalDateTime等）はすべて **イミュータブル（不変）** である。`plusDays()` や `minusHours()` などのメソッドは **元のオブジェクトを変更せず、新しいオブジェクトを返す**。

  ```java
  LocalDate today = LocalDate.now();
  LocalDate tomorrow = today.plusDays(1);
  // today は変更されない（todayは今日のまま）
  ```

  この性質により、意図しない変更を防ぎ、スレッドセーフに使用できる。
