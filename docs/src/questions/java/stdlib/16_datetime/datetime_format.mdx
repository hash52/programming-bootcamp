  `DateTimeFormatter.ofPattern()` で日時の表示形式を指定する。主なパターン文字：

  | パターン | 意味 | 例 |
  |---------|------|-----|
  | yyyy | 年（4桁） | 2024 |
  | MM | 月（2桁） | 04 |
  | dd | 日（2桁） | 01 |
  | HH | 時（24時間） | 14 |
  | mm | 分 | 30 |

  `today.format(formatter)` で指定した形式の文字列に変換される。
