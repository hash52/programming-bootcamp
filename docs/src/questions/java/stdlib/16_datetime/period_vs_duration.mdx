explanation: |
  - **Period**: **年・月・日** 単位の差を表す（`LocalDate` と一緒に使う）
  - **Duration**: **時・分・秒** 単位の差を表す（`LocalTime`, `LocalDateTime` と一緒に使う）

  例：`Period.between(date1, date2)` で日付の差、`Duration.between(time1, time2)` で時刻の差を取得する。
