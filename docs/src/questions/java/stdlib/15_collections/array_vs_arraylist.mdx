explanation: |
  - **配列**: 作成時にサイズが **固定** される。後から要素数を変更できない
  - **ArrayList**: 要素の追加・削除に応じてサイズが **動的に変わる**

  ArrayListは便利だが、基本型（int, double等）は直接格納できず、ラッパークラス（Integer, Double等）を使う必要がある。
