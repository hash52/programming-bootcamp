explanation: |
  PRの説明文に `Closes #番号` と書くと、そのPRが `main` にマージされた瞬間に、指定したIssueが自動的にクローズされる。

  ```markdown
  ## 変更内容
  ログイン機能を実装しました。

  Closes #5
  ```

  **使えるキーワード**（大文字・小文字不問）：
  - `Closes #番号`
  - `Fixes #番号`
  - `Resolves #番号`

  **紐づけの効果**：
  1. PRページにIssueへのリンクが表示される
  2. PRがマージされるとIssueが自動クローズされる
  3. 「この変更はIssue #5に対応した作業だ」という履歴が残る

  **複数のIssueを紐づける場合**：
  ```markdown
  Closes #5, Closes #8
  ```
