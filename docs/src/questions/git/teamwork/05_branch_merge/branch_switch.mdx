explanation: |
  `git switch -c ブランチ名` は、新しいブランチを作成して同時に切り替えるコマンドである。

  - `-c` オプション: create（新規作成）の意味

  ```bash
  git switch -c feature/login
  # Switched to a new branch 'feature/login'
  ```

  **ブランチ操作のコマンド一覧**：
  ```bash
  git switch -c feature/login  # 作成 + 切り替え
  git switch main              # 既存ブランチへの切り替え
  git branch                   # ブランチ一覧表示（現在のブランチに*）
  git branch -d feature/login  # ブランチを削除（マージ済みのもの）
  ```

  **命名規則の例**：
  - `feature/ログイン機能` ではなく英語で `feature/login` とする
  - `fix/null-pointer-error` のように修正内容を表す名前にする
  - チームで命名規則を統一するとよい
