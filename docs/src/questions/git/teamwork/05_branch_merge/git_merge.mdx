explanation: |
  ブランチをマージするには、取り込み先のブランチ（`main`）に移動してから `git merge` を実行する。

  ```bash
  git switch main              # mainブランチに切り替える
  git merge feature/login      # feature/loginをmainに取り込む
  # Fast-forward または Merge commit が作成される
  ```

  **マージの種類**：
  - **Fast-forward**: `main` が分岐以降変更されていない場合。ポインタが移動するだけ（マージコミットなし）
  - **Merge commit**: 両ブランチで変更がある場合。マージコミットが作成される

  **マージ後の後処理**：
  ```bash
  git branch -d feature/login   # マージ済みのブランチを削除
  git push origin main          # リモートに反映
  ```

  **注意点**：
  - マージ先（`main`）にいることを確認してから `git merge` を実行する
  - `git status` や `git branch` で現在のブランチを確認するとよい
