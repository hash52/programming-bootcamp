explanation: |
  ブランチ上でのコミットは、**カレントブランチにのみ記録**される。他のブランチには影響しない。

  **確認方法**：
  ```bash
  git switch -c feature/login   # feature/loginブランチを作成・切り替え
  # Login.java を作成・編集
  git add Login.java
  git commit -m "ログイン画面を追加"

  git switch main               # mainブランチに戻る
  ls                            # Login.javaが見えない（mainには影響していない）

  git switch feature/login      # feature/loginに戻る
  ls                            # Login.javaが見える
  ```

  **ブランチの独立性**：
  - ブランチを切り替えると、作業ディレクトリの内容もそのブランチの状態に切り替わる
  - `feature` ブランチでの変更は、`main` にマージするまで `main` には反映されない
  - これによってチームメンバーが互いの作業を妨げずに開発できる
---

`feature/login` ブランチでコミットした場合、そのコミットはどこに記録されるか。正しいものを選べ。
