explanation: |
  `.gitignore` は、Gitのバージョン管理から除外したいファイルやディレクトリを指定するファイルである。

  プロジェクトルートに `.gitignore` という名前で作成し、除外したいパターンを記述する。

  ```
  # コンパイル済みファイル
  *.class
  *.jar

  # IDEの設定ファイル
  .idea/
  *.iml

  # macOSのシステムファイル
  .DS_Store
  ```

  **除外すべき代表的なファイル**：
  - コンパイル済みファイル（`.class`, `.jar`）: 生成物であり、ソースコードから再生成できる
  - IDE設定ファイル（`.idea/`, `.vscode/`）: 個人の環境設定を共有する必要はない
  - パスワードや秘密鍵などの機密情報: 絶対にリポジトリに含めてはいけない

  **確認方法**：`git status` でグリーンに表示されていれば `.gitignore` が正しく機能している
