explanation: |
  `git init` は、カレントディレクトリをGitリポジトリとして初期化するコマンドである。

  実行すると `.git` ディレクトリが作成され、Gitがそのディレクトリの変更履歴を管理できるようになる。

  ```bash
  git init
  # → Initialized empty Git repository in /path/to/project/.git/
  ```

  **注意点**：
  - `git init` は一度だけ実行すればよい。プロジェクト作成時に1回実行する
  - すでに `.git` が存在するディレクトリで実行しても問題ないが、通常は不要
  - `.git` ディレクトリを削除するとGitの管理対象外になる（履歴も消える）
