explanation: |
  `git config` コマンドは、Gitの設定を変更するコマンドである。

  `--global` オプションを付けると、そのPCのすべてのリポジトリに共通の設定になる。

  ```bash
  git config --global user.name "名前"
  git config --global user.email "メールアドレス"
  ```

  **設定の確認方法**：
  ```bash
  git config --list           # 全設定を一覧表示
  git config user.name        # user.nameの値だけ確認
  ```

  **なぜ設定が必要か**：
  - コミットには「誰が変更したか」という情報が記録される
  - `user.name` と `user.email` を設定しないと、GitHubのコミット履歴に正しく紐づかない
