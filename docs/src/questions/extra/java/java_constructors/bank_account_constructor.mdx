hint: |
  `BankAccount(String owner, int initialBalance)` コンストラクタを定義する。
  初期残高が負の場合は0に設定するバリデーションをコンストラクタ内で行う。
  `deposit`, `withdraw`, `showBalance` メソッドも定義する。
sampleAnswer: |
  class BankAccount {
      String owner;
      int balance;
      int transactionCount;

      BankAccount(String owner, int initialBalance) {
          this.owner = owner;
          this.balance = initialBalance >= 0 ? initialBalance : 0;
          this.transactionCount = 0;
      }

      void deposit(int amount) {
          if (amount > 0) {
              balance += amount;
              transactionCount++;
          }
      }

      void withdraw(int amount) {
          if (amount > 0 && amount <= balance) {
              balance -= amount;
              transactionCount++;
          } else {
              System.out.println("出金失敗（残高不足または無効な金額）");
          }
      }

      void showBalance() {
          System.out.println(owner + ": 残高=" + balance + "円, 取引回数=" + transactionCount);
      }
  }

  public class Main {
      public static void main(String[] args) {
          BankAccount a1 = new BankAccount("田中", 10000);
          BankAccount a2 = new BankAccount("佐藤", 0);
          BankAccount a3 = new BankAccount("山田", -500); // 無効→0に

          a1.deposit(5000);
          a1.withdraw(3000);
          a2.deposit(20000);
          a3.withdraw(100); // 残高不足

          a1.showBalance();
          a2.showBalance();
          a3.showBalance();
      }
  }
explanation: |
  **コンストラクタでのバリデーション**：初期値を検証し、不正な値を補正することで、オブジェクトを常に有効な状態で生成できる。

  **三項演算子による簡潔な条件代入**：`initialBalance >= 0 ? initialBalance : 0` で正の値はそのまま、負の値は0にする。

  **取引カウントの追跡**：`transactionCount` フィールドで操作回数を追跡できる。コンストラクタで0に初期化することで管理する。

## 問題

コンストラクタで所有者と初期残高を設定する `BankAccount` クラスを実装せよ。

**出力**：
```
出金失敗（残高不足または無効な金額）
田中: 残高=12000円, 取引回数=2
佐藤: 残高=20000円, 取引回数=1
山田: 残高=0円, 取引回数=0
```

**要件**：
- `BankAccount(String owner, int initialBalance)` コンストラクタを定義すること
- 初期残高が負なら0に補正すること
- 取引回数を追跡する `transactionCount` フィールドを持つこと
