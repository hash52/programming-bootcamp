sampleAnswer: |
  class Person {
      String name;
      int age;

      Person(String name, int age) {
          this.name = name;
          this.age = age;
      }

      String introduce() {
          return "私は" + name + "です。" + age + "歳です。";
      }
  }

  public class Main {
      public static void main(String[] args) {
          Person[] people = {
              new Person("田中", 25),
              new Person("佐藤", 32),
              new Person("山田", 19)
          };

          for (Person p : people) {
              System.out.println(p.introduce());
          }
      }
  }
explanation: |
  **コンストラクタによる簡潔な生成**：`new Person("田中", 25)` と書くことで、名前と年齢を一度に設定できる。

  **配列での一括生成**：`new Person[]{ ... }` ではなく `{}` で初期化しつつコンストラクタ呼び出しを並べることができる。

  **`introduce()` メソッド**：自己紹介の文字列を返すメソッド。`void` で直接表示するのではなく `String` を返すことで、呼び出し元が柔軟に使える。
