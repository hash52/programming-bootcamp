hint: |
  `class Circle` を定義し、`double radius` フィールドを持たせる。
  コンストラクタで `radius` を受け取り初期化する。
  `area()` で面積（π × radius²）を返し、`display()` で半径と面積を表示する。
  π は `3.14159` を使ってよい。
sampleAnswer: |
  class Circle {
      double radius;

      Circle(double radius) {
          this.radius = radius;
      }

      double area() {
          return 3.14159 * radius * radius;
      }

      void display() {
          System.out.println("半径: " + radius + " 面積: " + area());
      }
  }

  public class Main {
      public static void main(String[] args) {
          Circle c1 = new Circle(5.0);
          Circle c2 = new Circle(3.0);
          c1.display();
          c2.display();
      }
  }
explanation: |
  **コンストラクタで値を強制**：コンストラクタがあると `new Circle()` とだけ書くとコンパイルエラーになる。
  必要な値を強制的に渡させられるのがコンストラクタの利点である。

  **計算メソッドとコンストラクタの組み合わせ**：コンストラクタで初期化したフィールドをメソッドで使う基本パターンである。

  **Math.PI との比較**：より正確な円周率は `Math.PI` で使えるが、ここでは `3.14159` で十分である。
