sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 5;
          for (int i = 1; i <= n; i++) {
              for (int j = 1; j <= i; j++) {
                  System.out.print("*");
              }
              System.out.println();
          }
      }
  }
explanation: |
  **二重ループの仕組み**：外側ループの `i` が段数（1〜5）、内側ループの `j` が `*` の個数（1〜i個）を表す。

  **`print` と `println` の使い分け**：`System.out.print` は改行せずに出力、`System.out.println()` は改行だけを出力する。内側ループで `*` を並べ、外側ループで改行する。

  **1段目から5段目**：1段目は `*` が1個、2段目は2個、...、5段目は5個。
