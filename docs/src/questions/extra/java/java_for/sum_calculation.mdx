sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int sum = 0;
          for (int i = 1; i <= 100; i++) {
              sum += i;
          }
          System.out.println("合計: " + sum);
      }
  }
explanation: |
  **累積加算のパターン**：`sum = 0` と初期化した変数にループで `sum += i` を繰り返す。
  この「変数を0で初期化 → ループで加算」というパターンは様々な集計処理の基本形である。

  **答えの確認**：1〜100の合計は等差数列の公式 `n(n+1)/2 = 100×101/2 = 5050` で求められる。
