  public class Main {
      public static void main(String[] args) {
          for (int i = 1; i <= 30; i++) {
              if (i % 2 == 0) {
                  System.out.println(i);
              }
          }
      }
  }
explanation: |
  **for + if の組み合わせ**：ループの中で `if` を使うことで、条件に合う値だけを処理できる。

  **2ずつ増やす別解**：`for (int i = 2; i <= 30; i += 2)` でも同じ結果が得られる。こちらの方がより効率的だが、`for + if` パターンはあらゆる条件に応用できるので覚えておくこと。
