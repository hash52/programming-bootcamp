  ループ内で `sum += i * i` を繰り返す。
  `i * i` は `i` の2乗（平方数）を表す。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 5;
          int sum = 0;
          for (int i = 1; i <= n; i++) {
              sum += i * i;
          }
          System.out.println("1²+2²+...+" + n + "² = " + sum);
      }
  }
explanation: |
  **平方和**：1² + 2² + 3² + 4² + 5² = 1 + 4 + 9 + 16 + 25 = 55。

  **累積加算の応用**：`sum += i` で合計を求めるパターンを、`sum += i * i` に変えるだけで平方和が求まる。

  **公式による検算**：n個の平方和は `n(n+1)(2n+1)/6` で求められる。`5 × 6 × 11 / 6 = 55` ✓
