sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 4;
          for (int i = 0; i < n; i++) {
              for (int j = 0; j < n; j++) {
                  System.out.print("*");
              }
              System.out.println();
          }
      }
  }
explanation: |
  **二重ループで2次元パターンを描く**：行を外側ループ、列を内側ループで制御する。n×n の四角形では両方のループが n 回繰り返される。

  **三角形との違い**：三角形は内側ループが `j <= i` まで（行によって個数が変わる）。四角形は内側ループが常に `j < n` まで（全行で同じ個数）。

  **応用**：外枠だけ表示する（`i == 0 || i == n-1 || j == 0 || j == n-1` のときだけ `*`）などにも挑戦できる。
