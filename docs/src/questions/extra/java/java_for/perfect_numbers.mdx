hint: |
  完全数とは「自分自身を除く約数の合計が自分と等しい数」。例：6 = 1+2+3。
  外側のループで候補の数 `n` を調べ、内側のループで `n` の約数の合計 `sum` を求め、`sum == n` なら完全数。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          for (int n = 1; n <= 1000; n++) {
              int sum = 0;
              for (int i = 1; i < n; i++) {
                  if (n % i == 0) {
                      sum += i;
                  }
              }
              if (sum == n) {
                  System.out.println(n + " は完全数");
              }
          }
      }
  }
explanation: |
  **完全数**：自分自身を除く約数の総和が自分に等しい数。1000以下では 6・28・496 の3つがある。

  **6の約数（自身を除く）**：1, 2, 3 → 合計 = 6 ✓  
  **28の約数（自身を除く）**：1, 2, 4, 7, 14 → 合計 = 28 ✓

  **二重ループの役割**：外側で候補 `n`（1〜1000）を順に確認し、内側で `n` の約数の合計 `sum` を計算する。`sum == n` なら出力する。

  **`i < n` のポイント**：自分自身（n）は含めないので、内側ループは `i < n` までとする。
