hint: |
  階乗は1からnまでの整数をすべて掛け合わせた値。
  `result = 1` で初期化し、ループ内で `result *= i` を繰り返す。
  `result = 0` で初期化すると何を掛けても0になってしまうので注意。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 5;
          int result = 1;
          for (int i = 1; i <= n; i++) {
              result *= i;
          }
          System.out.println(n + "! = " + result);
      }
  }
explanation: |
  **階乗（n!）**：1 × 2 × 3 × ... × n。5! = 1×2×3×4×5 = 120。

  **初期値は1**：掛け算の初期値は1にする（0にするとどんな数を掛けても0になる）。加算の初期値が0なのと同じ理由で、乗算では1が「単位元」となる。

  **`*=` 演算子**：`result *= i` は `result = result * i` の省略形である。
