sampleAnswer: |
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);
          System.out.print("整数を入力してください: ");
          int n = scanner.nextInt();
          int sum = 0;
          for (int i = 1; i <= n; i++) {
              sum += i;
          }
          System.out.println("1から" + n + "の合計: " + sum);
      }
  }
explanation: |
  **Scannerで入力を受け取る**：`scanner.nextInt()` でキーボードから整数を入力できる。ループに入力値を使うことで、汎用的なプログラムになる。

  **累積加算パターン**：`sum = 0` で初期化し、ループ内で `sum += i` を繰り返す。これは集計処理の基本パターンである。

  **等差数列の公式で確認**：`n = 10` のとき、1〜10の合計 = `10 × 11 / 2 = 55` で正しいかチェックできる。

```
整数を入力してください: 10
1から10の合計: 55
```

- `Scanner` を使って整数を入力する
- `for` ループで1から `n` まで繰り返す
- 累積変数 `sum` を使って合計を求める
