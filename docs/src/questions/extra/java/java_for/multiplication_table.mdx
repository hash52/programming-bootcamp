      public static void main(String[] args) {
          for (int i = 1; i <= 9; i++) {
              System.out.println("--- " + i + "の段 ---");
              for (int j = 1; j <= 9; j++) {
                  System.out.println(i + " × " + j + " = " + (i * j));

          }
      }

  **実行回数**：外側9回 × 内側9回 = 合計81回の掛け算が実行される。

  **応用**：ネストされたループはグリッド状の処理（表・行列・画像など）で非常によく使われるパターンである。

## 問題

九九の表全体（1の段〜9の段）を表示するプログラムを実装せよ。

**出力（一部）**：
```
--- 1の段 ---
1 × 1 = 1
1 × 2 = 2
...
--- 2の段 ---
2 × 1 = 2
...
--- 9の段 ---
...
9 × 9 = 81
```

**要件**：
- ネストされた `for` ループを使う（外側: 段、内側: かける数）
- 各段の前に `--- N の段 ---` を表示する

    public static void main(String[] args) {

            // 内側のループ: かける数（j = 1〜9）を書いてください
        }
    }
}`}
