sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int dan = 7;
          for (int i = 1; i <= 9; i++) {
              System.out.println(dan + " × " + i + " = " + (dan * i));
          }
      }
  }
explanation: |
  **ループで繰り返す計算**：`dan * i` をループの中で毎回計算することで、9行分の九九を自動で出力できる。

  **括弧の必要性**：`"= " + (dan * i)` の括弧がないと `"= " + dan + i` となり、文字列連結になって誤った出力になる。

  **応用**：`dan` の値を変えれば任意の段の九九を表示できる。
