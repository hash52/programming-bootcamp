  華氏→摂氏：`C = (F - 32) × 5.0 / 9.0`

sampleAnswer: |
  public class Main {
      static double celsiusToFahrenheit(double celsius) {
          return celsius * 9.0 / 5.0 + 32;
      }

      static double fahrenheitToCelsius(double fahrenheit) {
          return (fahrenheit - 32) * 5.0 / 9.0;
      }

      public static void main(String[] args) {
          double[] celsiusValues = {0, 20, 37, 100};
          System.out.println("=== 摂氏 → 華氏 ===");
          for (double c : celsiusValues) {
              System.out.printf("%.1f°C = %.1f°F%n", c, celsiusToFahrenheit(c));
          }

          System.out.println("=== 華氏 → 摂氏 ===");
          double[] fahrenheitValues = {32, 68, 98.6, 212};
          for (double f : fahrenheitValues) {
              System.out.printf("%.1f°F = %.1f°C%n", f, fahrenheitToCelsius(f));
          }
      }
  }
explanation: |
  **変換公式**：`F = C × 9/5 + 32`、`C = (F - 32) × 5/9` は覚えておくと便利。

  **整数除算の回避**：`9 / 5` は整数除算で `1` になってしまう。`9.0 / 5.0` または `(double) 9 / 5` と書くこと。

  **逆変換の検証**：`celsiusToFahrenheit(0)` が `32.0` になり、`fahrenheitToCelsius(32)` が `0.0` になることで逆変換が正しいことを確認できる。

## 問題

摂氏を華氏に変換する `celsiusToFahrenheit` と、華氏を摂氏に変換する `fahrenheitToCelsius` の2つのメソッドを実装せよ。

**出力**：
```
=== 摂氏 → 華氏 ===
0.0°C = 32.0°F
20.0°C = 68.0°F
37.0°C = 98.6°F
100.0°C = 212.0°F
=== 華氏 → 摂氏 ===
32.0°F = 0.0°C
68.0°F = 20.0°C
98.6°F = 37.0°C
212.0°F = 100.0°C
```

**要件**：
- 摂氏→華氏：`F = C × 9.0 / 5.0 + 32`
- 華氏→摂氏：`C = (F - 32) × 5.0 / 9.0`
- 整数除算に注意すること
