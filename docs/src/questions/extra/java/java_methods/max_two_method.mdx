hint: |
  `static int max(int a, int b)` のようにメソッドを定義する。
  if-else で比較して大きい方を `return` する。
  三項演算子を使うと `return a >= b ? a : b;` と1行で書ける。
sampleAnswer: |
  public class Main {
      static int max(int a, int b) {
          if (a >= b) {
              return a;
          } else {
              return b;
          }
      }

      public static void main(String[] args) {
          System.out.println("max(3, 7) = " + max(3, 7));
          System.out.println("max(10, 5) = " + max(10, 5));
          System.out.println("max(4, 4) = " + max(4, 4));
      }
  }
explanation: |
  **複数のreturn文**：メソッドの中に `return` が複数あっても問題ない。
  最初に実行された `return` でメソッドが終了する。

  **等しい場合の扱い**：`a >= b` とすることで等しい場合は `a` を返す（`a > b` では等しい場合に `b` が返る）。

  **三項演算子**：`a >= b ? a : b` は「a >= b なら a、そうでなければ b」という意味。
  シンプルな条件分岐に便利。
