hint: |
  `static boolean isPrime(int n)` のようにメソッドを定義する。
  1以下は素数でない。2以上で n の約数（2〜√n）が存在しないなら素数。
  ループで2から n-1 まで試す（または2から √n まで）。
  `main` で2から100をループし、`isPrime(i)` が `true` の数だけ表示する。
sampleAnswer: |
  public class Main {
      static boolean isPrime(int n) {
          if (n <= 1) return false;
          for (int i = 2; i * i <= n; i++) {
              if (n % i == 0) return false;
          }
          return true;
      }

      public static void main(String[] args) {
          System.out.print("100以下の素数: ");
          for (int i = 2; i <= 100; i++) {
              if (isPrime(i)) {
                  System.out.print(i + " ");
              }
          }
          System.out.println();
      }
  }
explanation: |
  **素数判定の効率化**：2から n-1 まで全て試すと遅い。
  n の約数は必ず √n 以下のものとペアを持つため、`i * i <= n` までチェックすれば十分。

  **早期return**：約数が見つかった時点で `return false` でメソッドを抜けると効率的。

  **メソッドの分離**：`isPrime` を別メソッドに分離することで、main の処理が「何をしているか」が明確になる。
