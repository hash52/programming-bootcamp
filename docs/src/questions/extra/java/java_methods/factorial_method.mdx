hint: |
  `static int factorial(int n)` のようにメソッドを定義する。
  `result = 1` で初期化し、1から `n` までの整数を順に掛ける。
  例：5! = 1 × 2 × 3 × 4 × 5 = 120
sampleAnswer: |
  public class Main {
      static int factorial(int n) {
          int result = 1;
          for (int i = 1; i <= n; i++) {
              result *= i;
          }
          return result;
      }

      public static void main(String[] args) {
          for (int i = 0; i <= 7; i++) {
              System.out.println(i + "! = " + factorial(i));
          }
      }
  }
explanation: |
  **階乗の定義**：n! = 1 × 2 × 3 × ... × n（0! = 1 と定義される）。

  **ループの範囲**：`for (int i = 1; i <= n; i++)` で1から n まで掛け合わせる。
  `n = 0` のとき、ループは一度も実行されないため `result = 1` が返る（0! = 1 と一致）。

  **メソッドの再利用**：ループの中で `factorial(i)` と呼び出すことで0〜7の階乗をまとめて表示できる。

## 問題

整数 `n` の階乗（n!）を返す `factorial` メソッドを実装し、0! から 7! を表示せよ。

**出力**：
```
0! = 1
1! = 1
2! = 2
3! = 6
4! = 24
5! = 120
6! = 720
7! = 5040
```

**要件**：
- `static int factorial(int n)` というシグネチャでメソッドを定義すること
- ループで1から n まで掛け合わせること
