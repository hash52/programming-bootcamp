sampleAnswer: |
  import java.util.ArrayList;

  public class Main {
      public static void main(String[] args) {
          ArrayList<Integer> numbers = new ArrayList<>();
          numbers.add(34);
          numbers.add(67);
          numbers.add(12);
          numbers.add(89);
          numbers.add(45);
          numbers.add(23);
          numbers.add(78);

          ArrayList<Integer> highScores = new ArrayList<>();
          ArrayList<Integer> evenNumbers = new ArrayList<>();

          for (int n : numbers) {
              if (n >= 50) highScores.add(n);
              if (n % 2 == 0) evenNumbers.add(n);
          }

          System.out.println("50以上: " + highScores);
          System.out.println("偶数: " + evenNumbers);
      }
  }
explanation: |
  **フィルタリングパターン**：元のコレクションはそのままに、新しいリストに条件を満たす要素だけを追加する方法。
  Java 8以降は `stream().filter()` でより簡潔に書けるが、基本は for-each + if のパターンを理解しておくとよい。

  **複数条件の並行フィルタ**：1つのループ内で複数の条件を独立してチェックし、別々のリストに振り分けることができる。
