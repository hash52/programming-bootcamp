sampleAnswer: |
  import java.util.ArrayList;
  import java.util.HashMap;

  public class Main {
      public static void main(String[] args) {
          HashMap<String, ArrayList<String>> groups = new HashMap<>();

          addMember(groups, "Aチーム", "田中");
          addMember(groups, "Bチーム", "鈴木");
          addMember(groups, "Aチーム", "佐藤");
          addMember(groups, "Bチーム", "山田");
          addMember(groups, "Aチーム", "高橋");

          for (String team : groups.keySet()) {
              System.out.println(team + ": " + groups.get(team));
          }
      }

      static void addMember(HashMap<String, ArrayList<String>> groups, String team, String name) {
          if (!groups.containsKey(team)) {
              groups.put(team, new ArrayList<>());
          }
          groups.get(team).add(name);
      }
  }

  **ネストされたコレクション**：`HashMap<String, ArrayList<String>>` で「グループ名」→「メンバーリスト」の関係を表現する。

  **初期化チェック**：新しいキーに対して `ArrayList` がまだ存在しないため、`containsKey()` で確認してから `put()` する必要がある。
  `computeIfAbsent()` を使えば `groups.computeIfAbsent(team, k -> new ArrayList<>()).add(name);` と1行で書ける。
