sampleAnswer: |
  import java.util.ArrayList;

  public class Main {
      public static void main(String[] args) {
          ArrayList<String> fruits = new ArrayList<>();
          fruits.add("りんご");
          fruits.add("バナナ");
          fruits.add("みかん");

          System.out.println("要素数: " + fruits.size());
          System.out.println("0番目: " + fruits.get(0));

          for (String fruit : fruits) {
              System.out.println(fruit);
          }
      }
  }
explanation: |
  **`ArrayList` の基本操作**：`add()` で末尾に追加、`get(i)` でインデックスアクセス、`size()` で要素数を取得。

  **ジェネリクス**：`ArrayList<String>` の `<String>` で要素の型を指定する。型安全なコレクション操作が可能になる。

  **for-each 文**：`for (String fruit : fruits)` で全要素を順に処理できる。配列と同じ記法が使える。
