  `HashMap<String, Boolean>` でタスク名→完了フラグを管理する。

sampleAnswer: |
  import java.util.ArrayList;
  import java.util.HashMap;

  public class Main {
      static ArrayList<String> taskOrder = new ArrayList<>();
      static HashMap<String, Boolean> tasks = new HashMap<>();

      public static void main(String[] args) {
          addTask("資料作成");
          addTask("メール送信");
          addTask("会議準備");
          addTask("コードレビュー");

          complete("メール送信");
          complete("会議準備");

          printAll();
      }

      static void addTask(String name) {
          taskOrder.add(name);
          tasks.put(name, false);
      }

      static void complete(String name) {
          if (tasks.containsKey(name)) {
              tasks.put(name, true);
              System.out.println("完了: " + name);
          }
      }

      static void printAll() {
          System.out.println("\n=== ToDo一覧 ===");
          for (String name : taskOrder) {
              String status = tasks.get(name) ? "[完了]" : "[未完]";
              System.out.println(status + " " + name);
          }
      }
  }
explanation: |
  **ArrayList と HashMap の組み合わせ**：`ArrayList` で挿入順を保持し、`HashMap` でタスクの状態（完了/未完）を管理する。
  2つのコレクションを組み合わせて、それぞれの長所を活かす設計。

  **staticフィールドとstaticメソッド**：`main` と補助メソッドが同じクラス内でデータを共有するため、staticフィールドに格納する。

  **`Boolean` → 三項演算子**：`tasks.get(name) ? "[完了]" : "[未完]"` で完了状態を文字列に変換する。

```

完了: 会議準備

=== ToDo一覧 ===
[未完] 資料作成
[完了] メール送信
[完了] 会議準備
[未完] コードレビュー
```

- `ArrayList<String>` で追加順序を保持すること
