sampleAnswer: |
  class Vehicle {
      String brand;
      int speed;

      Vehicle(String brand, int speed) {
          this.brand = brand;
          this.speed = speed;
      }

      String describe() {
          return brand + "（最高速度: " + speed + "km/h）";
      }
  }

  class Car extends Vehicle {
      int seats;

      Car(String brand, int speed, int seats) {
          super(brand, speed);
          this.seats = seats;
      }

      @Override
      String describe() {
          return "車: " + super.describe() + " 座席数: " + seats;
      }
  }

  class Bike extends Vehicle {
      boolean hasSidecar;

      Bike(String brand, int speed, boolean hasSidecar) {
          super(brand, speed);
          this.hasSidecar = hasSidecar;
      }

      @Override
      String describe() {
          return "バイク: " + super.describe() + (hasSidecar ? " サイドカーあり" : "");
      }
  }

  class Truck extends Vehicle {
      int loadCapacity;

      Truck(String brand, int speed, int loadCapacity) {
          super(brand, speed);
          this.loadCapacity = loadCapacity;
      }

      @Override
      String describe() {
          return "トラック: " + super.describe() + " 積載量: " + loadCapacity + "t";
      }
  }

  public class Main {
      public static void main(String[] args) {
          Vehicle[] vehicles = {
              new Car("トヨタ", 180, 5),
              new Bike("ホンダ", 200, false),
              new Truck("いすゞ", 120, 10)
          };

          for (Vehicle v : vehicles) {
              System.out.println(v.describe());
          }
      }
  }
explanation: |
  **継承ツリー**：`Vehicle` → `Car`, `Bike`, `Truck` という1段階の継承ツリー。共通の属性は親に、固有の属性は子に置く。

  **`super.describe()` の活用**：サブクラスの `describe()` で `super.describe()` を呼ぶことで、親クラスの表示を再利用できる。

  **ポリモーフィズムの活用**：`Vehicle[]` に異なる種類の乗り物を格納して、同一のループで処理できる。

## 問題

`Vehicle` → `Car`, `Bike`, `Truck` の継承ツリーを実装し、ポリモーフィズムで一覧表示するプログラムを作れ。

**出力**：
```
車: トヨタ（最高速度: 180km/h） 座席数: 5
バイク: ホンダ（最高速度: 200km/h）
トラック: いすゞ（最高速度: 120km/h） 積載量: 10t
```

**要件**：
- 各サブクラスで `@Override String describe()` を実装すること
- `Vehicle[]` 型の配列で管理すること
