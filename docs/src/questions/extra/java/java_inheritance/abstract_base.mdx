hint: |
  基底クラスに共通処理を置き、サブクラスで異なる部分（面積計算など）を実装する。
  Javaの `abstract` キーワードも使えるが、この演習では通常のクラスで「実装を強制」する設計パターンを学ぶ。
  基底クラスの `area()` はデフォルト値を返し、サブクラスでオーバーライドする。
sampleAnswer: |
  class Shape {
      String name;
      String color;

      Shape(String name, String color) {
          this.name = name;
          this.color = color;
      }

      double area() {
          return 0.0; // サブクラスでオーバーライドする
      }

      double perimeter() {
          return 0.0; // サブクラスでオーバーライドする
      }

      void printInfo() {
          System.out.printf("[%s] %s: 面積=%.2f, 周囲長=%.2f%n",
              color, name, area(), perimeter());
      }
  }

  class Circle extends Shape {
      double radius;

      Circle(String color, double radius) {
          super("円", color);
          this.radius = radius;
      }

      @Override
      double area() {
          return Math.PI * radius * radius;
      }

      @Override
      double perimeter() {
          return 2 * Math.PI * radius;
      }
  }

  class Rectangle extends Shape {
      double width;
      double height;

      Rectangle(String color, double width, double height) {
          super("長方形", color);
          this.width = width;
          this.height = height;
      }

      @Override
      double area() {
          return width * height;
      }

      @Override
      double perimeter() {
          return 2 * (width + height);
      }
  }

  class EquilateralTriangle extends Shape {
      double side;

      EquilateralTriangle(String color, double side) {
          super("正三角形", color);
          this.side = side;
      }

      @Override
      double area() {
          return Math.sqrt(3) / 4 * side * side;
      }

      @Override
      double perimeter() {
          return 3 * side;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Shape[] shapes = {
              new Circle("赤", 5.0),
              new Rectangle("青", 4.0, 6.0),
              new EquilateralTriangle("緑", 6.0)
          };

          for (Shape s : shapes) {
              s.printInfo();
          }
      }
  }
explanation: |
  **基底クラスの設計**：`area()` と `perimeter()` は `Shape` に定義するが、0を返すだけ。サブクラスでオーバーライドすることで具体的な計算を行う。

  **`printInfo()` の再利用**：フォーマット出力の処理を基底クラスにまとめることで、全てのサブクラスで自動的に利用できる。

  **正三角形の面積公式**：`√3/4 × 辺²`。`Math.sqrt(3)` で √3 を計算できる。

## 問題

`Shape` 基底クラスを継承して `Circle`, `Rectangle`, `EquilateralTriangle` を実装し、面積と周囲長を表示するプログラムを作れ。

**出力**：
```
[赤] 円: 面積=78.54, 周囲長=31.42
[青] 長方形: 面積=24.00, 周囲長=20.00
[緑] 正三角形: 面積=15.59, 周囲長=18.00
```

**要件**：
- 各サブクラスで `area()` と `perimeter()` をオーバーライドすること
- `printInfo()` は基底クラスに1つだけ定義すること
