hint: |
  `class Dog extends Animal` で `Animal` を継承する。
  `Dog` クラスは `Animal` のフィールドとメソッドを引き継ぐ。
  `Dog` 固有のフィールド（`breed` など）やメソッドを追加できる。
sampleAnswer: |
  class Animal {
      String name;
      int age;

      void eat() {
          System.out.println(name + "が食事をしている");
      }

      void sleep() {
          System.out.println(name + "が眠っている");
      }
  }

  class Dog extends Animal {
      String breed;

      void bark() {
          System.out.println(name + "がワンワンと吠えた！");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Dog dog = new Dog();
          dog.name = "ポチ";
          dog.age = 3;
          dog.breed = "柴犬";

          System.out.println(dog.name + "（" + dog.breed + ", " + dog.age + "歳）");
          dog.eat();
          dog.sleep();
          dog.bark();
      }
  }
explanation: |
  **継承（`extends`）**：`Dog extends Animal` と書くことで、`Dog` は `Animal` の全フィールドとメソッドを引き継ぐ。

  **is-a 関係**：継承は「Dog is-a Animal（犬は動物である）」という関係を表す。

  **フィールドとメソッドの引き継ぎ**：`Dog` のインスタンスで `dog.eat()` や `dog.name` が使えるのは、`Animal` から引き継いでいるため。

  **追加のフィールド・メソッド**：`Dog` 固有の `breed` フィールドや `bark()` メソッドを追加できる。

## 問題

`Animal` クラスを継承して `Dog` クラスを作り、`Animal` のメソッドと `Dog` 固有のメソッドを使うプログラムを実装せよ。

**出力**：
```
ポチ（柴犬, 3歳）
ポチが食事をしている
ポチが眠っている
ポチがワンワンと吠えた！
```

**要件**：
- `class Dog extends Animal` で継承すること
- `Dog` クラスに `breed` フィールドと `bark()` メソッドを追加すること
