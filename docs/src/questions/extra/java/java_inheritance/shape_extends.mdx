hint: |
  `class Shape` に `color` フィールドと `describe()` メソッドを定義する。
  `class Circle extends Shape` と `class Rectangle extends Shape` で継承する。
  各サブクラスで `area()` メソッドを定義する（この時点では @Override なし）。
sampleAnswer: |
  class Shape {
      String color;

      void describe() {
          System.out.println("色: " + color);
      }
  }

  class Circle extends Shape {
      double radius;

      double area() {
          return Math.PI * radius * radius;
      }
  }

  class Rectangle extends Shape {
      double width;
      double height;

      double area() {
          return width * height;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Circle c = new Circle();
          c.color = "赤";
          c.radius = 5.0;

          Rectangle r = new Rectangle();
          r.color = "青";
          r.width = 4.0;
          r.height = 6.0;

          c.describe();
          System.out.printf("円の面積: %.2f%n", c.area());

          r.describe();
          System.out.printf("長方形の面積: %.2f%n", r.area());
      }
  }
explanation: |
  **共通属性の抽象化**：`color` は全図形に共通するため `Shape` に定義し、サブクラスで引き継ぐ。

  **型ごとに異なる実装**：`area()` は図形によって計算式が異なるため、サブクラスそれぞれで定義する。

  **継承の利点**：`describe()` のような共通メソッドを `Shape` にまとめることで、`Circle` と `Rectangle` 両方で使える。
---


## 問題

`Shape` クラスを継承して `Circle` と `Rectangle` を作り、各図形の面積を計算するプログラムを実装せよ。

**出力**：
```
色: 赤
円の面積: 78.54
色: 青
長方形の面積: 24.00
```

**要件**：
- `Shape` クラスに `color` フィールドと `describe()` を定義すること
- `Circle` と `Rectangle` で `area()` を定義すること
