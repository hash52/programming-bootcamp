hint: |
  サブクラスのコンストラクタで `super(引数)` を呼ぶとスーパークラスのコンストラクタが実行される。
  `super()` はコンストラクタの先頭行に書く。
  `Person(String name, int age)` があるなら `Employee` のコンストラクタで `super(name, age)` と書く。
sampleAnswer: |
  class Person {
      String name;
      int age;

      Person(String name, int age) {
          this.name = name;
          this.age = age;
      }

      void introduce() {
          System.out.println("私は" + name + "（" + age + "歳）です");
      }
  }

  class Employee extends Person {
      String department;

      Employee(String name, int age, String department) {
          super(name, age); // Personのコンストラクタを呼ぶ
          this.department = department;
      }

      void introduce() {
          super.introduce();
          System.out.println("  → " + department + "部門に所属しています");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Person p = new Person("田中", 30);
          Employee e = new Employee("佐藤", 25, "開発");

          p.introduce();
          System.out.println("---");
          e.introduce();
      }
  }
explanation: |
  **`super()`**：スーパークラスのコンストラクタを呼ぶ。サブクラスのコンストラクタ先頭に書く必要がある。
  書かない場合、Javaは自動的に引数なしの `super()` を挿入するため、スーパークラスに引数なしコンストラクタがないとコンパイルエラーになる。

  **`super.メソッド()`**：スーパークラスのメソッドを呼ぶ。`Employee.introduce()` の中で `super.introduce()` を呼ぶと `Person` の `introduce()` が実行される。

  **コンストラクタの連鎖**：`new Employee(...)` → `Employee(...)` コンストラクタ → `super(name, age)` → `Person(...)` コンストラクタ の順に実行される。

## 問題

`Employee` が `Person` を継承し、`super()` でスーパークラスのコンストラクタを呼ぶプログラムを実装せよ。

**出力**：
```
私は田中（30歳）です
---
私は佐藤（25歳）です
  → 開発部門に所属しています
```

**要件**：
- `Employee(String name, int age, String department)` コンストラクタで `super(name, age)` を呼ぶこと
- `Employee.introduce()` で `super.introduce()` を呼ぶこと
