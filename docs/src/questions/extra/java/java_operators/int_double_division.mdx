sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int a = 10, b = 3;
          int intResult = a / b;
          double dblResult = (double) a / b;
          System.out.println("int / int: " + intResult);
          System.out.println("double / int: " + dblResult);
      }
  }
explanation: |
  **整数除算**：`int` 同士の `/` は小数点以下を切り捨てた整数になる。`10 / 3` は `3.333...` ではなく `3`。

  **double除算**：`(double) a / b` のように一方を `double` にキャストすると、小数の結果が得られる。

  **キャストの位置**：`(double)(a / b)` と書くと整数除算の後でキャストするため `3.0` になってしまう。`(double) a / b` のように除算の前にキャストすることが重要である。

## 問題

```

double / int: 3.3333333333333335
```
