  面積は `width * height`、周囲長は `(width + height) * 2` で求められる。

sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int width = 8;
          int height = 5;
          int area = width * height;
          int perimeter = (width + height) * 2;
          System.out.println("面積: " + area);
          System.out.println("周囲長: " + perimeter);
      }
  }
explanation: |
  **面積**：縦 × 横 = `width * height`。`8 * 5 = 40`。

  **周囲長**：`(縦 + 横) * 2 = (width + height) * 2`。`(8 + 5) * 2 = 26`。

  **括弧の重要性**：`width + height * 2` と書くと `*` が先に計算されて `8 + 10 = 18` になってしまう。正しい計算順序のために括弧が必要である。
