  換算は `円 × レート` で求められる。
  `int` 型の `yen` に `double` 型のレートを掛けると自動的に `double` になる。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int yen = 10000;
          double usdRate = 0.0067;
          double eurRate = 0.0062;
          double usd = yen * usdRate;
          double eur = yen * eurRate;
          System.out.println(yen + "円 = " + usd + " USD");
          System.out.println(yen + "円 = " + eur + " EUR");
      }
  }
explanation: |
  **換算式**：`金額 × 為替レート` で換算後の金額が求まる。

  **int × double の型変換**：`yen`（int）に `usdRate`（double）を掛けると、Javaは自動的に `int` を `double` へ widening 変換してから計算する。結果は `double` になる。

  **実用上の注意**：為替レートは常に変動するため、実際のアプリではAPIから最新レートを取得する。この問題では学習用に固定値を使っている。
