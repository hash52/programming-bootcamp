sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int speed = 60;
          double time = 2.5;
          double distance = speed * time;
          int minutes = (int)(time * 60);
          System.out.println("速さ: " + speed + " km/h");
          System.out.println("走行時間: " + time + " 時間（" + minutes + "分）");
          System.out.println("走行距離: " + distance + " km");
      }
  }
explanation: |
  **速さ・時間・距離**：「距離 = 速さ × 時間」は理科でも学ぶ基本的な公式である。単位に注意して計算する。

  **時間を分に変換**：`time * 60` で時間を分に変換できる。`2.5 時間 = 150 分`。`(int)` でキャストして整数化する。

  **int × double の自動変換**：`speed`（int 60）× `time`（double 2.5）の結果は自動的に `double` の `150.0` になる。
