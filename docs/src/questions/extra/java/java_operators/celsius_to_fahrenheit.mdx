sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          double celsius = 100.0;
          double fahrenheit = celsius * 9.0 / 5 + 32;
          System.out.println(celsius + "℃ = " + fahrenheit + "°F");
      }
  }
explanation: |
  **変換式**：`F = C × 9 / 5 + 32`。水の沸点（100℃）は 212°F、氷点（0℃）は 32°F になる。

  **演算子の優先順位**：`*` と `/` は `+` より先に計算される。式は左から `celsius * 9.0 → / 5 → + 32` の順に処理される。

  **整数除算に注意**：`9 / 5` と書くと整数除算で `1` になる。`9.0 / 5` と書くことで `1.8` になり正しい結果が得られる。
