sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int principal = 100000;
          double rate = 0.03;
          double year1 = principal * (1 + rate);
          double year2 = year1 * (1 + rate);
          double year3 = year2 * (1 + rate);
          int interest = (int)year3 - principal;
          System.out.println("元金: " + principal + "円");
          System.out.println("年利: " + (int)(rate * 100) + "%");
          System.out.println("1年後: " + (int)year1 + "円");
          System.out.println("2年後: " + (int)year2 + "円");
          System.out.println("3年後: " + (int)year3 + "円");
          System.out.println("3年間の利息: " + interest + "円");
      }
  }
explanation: |
  **複利計算の仕組み**：「元金に利息がつき、その利息にさらに利息がつく」のが複利である。毎年「前年の残高 × (1 + 利率)」を繰り返す。

  **単利との違い**：単利なら毎年 `principal * rate = 3000円` の利息。複利では利息が雪だるま式に増えていく。

  **各年を変数に格納**：`year1`→`year2`→`year3` と連鎖させる。`year2 = year1 * (1 + rate)` のように前の年の結果を使う。

  **計算の確認**：100000 × 1.03 = 103000、103000 × 1.03 = 106090、106090 × 1.03 ≈ 109272。利息 = 9272円。
