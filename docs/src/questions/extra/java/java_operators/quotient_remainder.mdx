hint: |
  商（何箱できるか）= 個数 / 1箱の容量
  余り（余る個数）= 個数 % 1箱の容量
  `%` は余りを求める演算子である。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int apples = 23;
          int perBox = 6;
          int boxes = apples / perBox;
          int remaining = apples % perBox;
          System.out.println(apples + "個のりんごを" + perBox + "個入りの箱に詰める");
          System.out.println(boxes + "箱できて、" + remaining + "個余る");
      }
  }
explanation: |
  **`/` は商、`%` は余り**：`23 / 6 = 3`（3箱分）、`23 % 6 = 5`（5個余り）となる。

  **日常での応用**：「○○個を△個ずつ分けると何グループで何人余るか」という問題は、すべてこのパターンで解ける。

  **整数除算の特性**：Java の `/` はint同士では小数を切り捨てた整数を返す。`23 / 6 = 3`（3.833...の小数部分は無視）。
