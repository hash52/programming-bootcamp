hint: |
  `a++` はインクリメント（1を加算）、`a--` はデクリメント（1を減算）する演算子である。
  `++` は `+= 1` と同じ意味だが、1ずつ変化させる場合により短く書ける。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int a = 5;
          System.out.println("初期値: " + a);
          a++;
          System.out.println("a++ 後: " + a);
          a++;
          System.out.println("a++ 後: " + a);
          a--;
          System.out.println("a-- 後: " + a);
      }
  }
explanation: |
  **`++`（インクリメント）**：変数の値を1増やす演算子。`a++` は `a = a + 1` と同じ意味である。

  **`--`（デクリメント）**：変数の値を1減らす演算子。`a--` は `a = a - 1` と同じ意味である。

  **使い道**：ループのカウンタ変数（`i++`）や、個数を1ずつ増やす処理でよく使われる。
