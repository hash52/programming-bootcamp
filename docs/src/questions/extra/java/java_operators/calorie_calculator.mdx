## 問題

炭水化物 50g、タンパク質 30g、脂質 15g の食事の総カロリーを計算して表示するプログラムを実装せよ。

**出力**：
```
炭水化物: 50.0g → 200.0kcal
タンパク質: 30.0g → 120.0kcal
脂質: 15.0g → 135.0kcal
合計: 455.0kcal
```

**カロリー換算**：炭水化物・タンパク質 = 4 kcal/g、脂質 = 9 kcal/g

**要件**：
- 各栄養素のグラム数を `double` 型で宣言する
- 各栄養素のカロリーをそれぞれ変数に格納してから合計を求める
