hint: |
  面積 = π × r²（r は半径）
  円周 = 2 × π × r
  π（pi）を `double pi = 3.14;` として宣言し、`radius * radius` で r² を求める。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int radius = 5;
          double pi = 3.14;
          double area = pi * radius * radius;
          double circumference = 2 * pi * radius;
          System.out.println("半径: " + radius);
          System.out.println("面積: " + area);
          System.out.println("円周: " + circumference);
      }
  }
explanation: |
  **定数として変数を使う**：`double pi = 3.14;` のように、変わらない値も変数に入れておくと式が読みやすくなる。

  **int × double = double**：`radius` が `int` でも、`pi`（double）との演算結果は自動的に `double` になる。

  **浮動小数点の誤差**：`2 * pi * radius` の計算結果が `31.400000000000002` のように表示される場合がある。これはコンピュータが小数を2進数で表現する際に生じる微小な誤差で、仕様の範囲内である。
