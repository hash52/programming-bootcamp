hint: |
  合計は `math + english + science` で求められる。
  平均は `合計 ÷ 3` だが、`int` 同士の割り算は小数が切り捨てられる。
  `(double)(合計) / 3` のようにキャストして小数の平均を求めよう。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int math = 85, english = 72, science = 90;
          int total = math + english + science;
          double average = (double) total / 3;
          System.out.println("数学: " + math);
          System.out.println("英語: " + english);
          System.out.println("理科: " + science);
          System.out.println("合計: " + total);
          System.out.println("平均: " + average);
      }
  }
explanation: |
  **合計**：3つの変数を `+` で加算する。`85 + 72 + 90 = 247`。

  **平均（double）**：`(double) total / 3` で整数を double に変換してから割ることで `82.33...` が得られる。`total / 3` だけだと整数除算で `82` になってしまう。

  **なぜ total を先にキャストするか**：`(double)(total / 3)` と書くと先に整数除算してしまうため正しい結果にならない。キャストは除算より前に行う必要がある。
