  税込み価格 = 税抜き価格 × (1 + 税率)
  `double` の計算結果を `int` に変換するには `(int)` でキャストする（小数点以下切り捨て）。
  例: `int taxIncluded = (int)(price * (1 + taxRate));`
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int price = 1500;
          double taxRate = 0.10;
          int taxIncluded = (int)(price * (1 + taxRate));
          System.out.println("税抜き価格: " + price + "円");
          System.out.println("消費税率: " + (int)(taxRate * 100) + "%");
          System.out.println("税込み価格: " + taxIncluded + "円");
      }
  }
explanation: |
  **計算式**：`price * (1 + taxRate)` で税込み価格を求める。`1500 * 1.10 = 1650.0`（double）となる。

  **型キャスト**：`double` の計算結果を `(int)` でキャストすると小数点以下が切り捨てられる。`(int) 1650.0` → `1650`。

  **演算子の優先順位**：`price * (1 + taxRate)` の括弧は必須。括弧がないと `price * 1 + taxRate` と解釈され、正しい結果にならない。
