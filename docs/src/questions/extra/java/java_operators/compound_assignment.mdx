sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int score = 100;
          System.out.println("初期値: " + score);
          score += 20;
          System.out.println("+=20 後: " + score);
          score -= 15;
          System.out.println("-=15 後: " + score);
          score *= 2;
          System.out.println("*=2 後: " + score);
          score /= 3;
          System.out.println("/=3 後: " + score);
          score %= 9;
          System.out.println("%=9 後: " + score);
      }
  }
explanation: |
  **複合代入演算子**：右辺の演算を変数に適用して結果を再代入する演算子である。

  | 演算子 | 意味 |
  |--------|------|
  | `+=` | 加算して代入 |
  | `-=` | 減算して代入 |
  | `*=` | 乗算して代入 |
  | `/=` | 除算して代入 |
  | `%=` | 余りを代入 |

  **順に追うと**：100 → 120（+20）→ 105（-15）→ 210（*2）→ 70（/3）→ 7（%9）

+=20 後: 120
-=15 後: 105
*=2 後: 210
/=3 後: 70
