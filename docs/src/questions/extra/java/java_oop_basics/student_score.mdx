  `class Student` を定義し、`name`（String）と `scores`（int[]）フィールドを持たせる。
  `average()` メソッドで全科目の平均点を計算する。
  `showReport()` メソッドで名前・各科目の点数・平均を表示する。
sampleAnswer: |
  class Student {
      String name;
      int[] scores;

      double average() {
          int sum = 0;
          for (int s : scores) {
              sum += s;
          }
          return (double) sum / scores.length;
      }

      void showReport() {
          System.out.println("=== " + name + " ===");
          String[] subjects = {"国語", "数学", "英語"};
          for (int i = 0; i < scores.length; i++) {
              System.out.println(subjects[i] + ": " + scores[i]);
          }
          System.out.printf("平均: %.1f%n", average());
      }
  }

  public class Main {
      public static void main(String[] args) {
          Student s1 = new Student();
          s1.name = "鈴木";
          s1.scores = new int[]{80, 75, 90};

          Student s2 = new Student();
          s2.name = "田中";
          s2.scores = new int[]{65, 95, 70};

          s1.showReport();
          s2.showReport();
      }
  }
explanation: |
  **フィールドに配列を持つ**：`int[] scores` のように、フィールドとして配列を持つことができる。
  `new int[]{80, 75, 90}` で配列を作って代入する。

  **メソッド内での配列処理**：`average()` メソッドの中でフィールドの `scores` 配列を処理できる。

  **`subjects` 配列との対応**：インデックスを合わせることで科目名と点数を対応させる。

## 問題

`Student` クラスを定義し、3科目の点数管理と平均計算を実装せよ。

**出力**：
```
=== 鈴木 ===
国語: 80
数学: 75
英語: 90
平均: 81.7
=== 田中 ===
国語: 65
数学: 95
英語: 70
平均: 76.7
```

**要件**：
- `double average()` と `void showReport()` を定義すること
- `scores` フィールドは `int[]` 型にすること
