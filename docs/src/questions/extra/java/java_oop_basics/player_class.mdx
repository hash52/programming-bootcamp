  `class Player` を定義し、`name`（String）、`hp`（int）、`attackPower`（int）フィールドを持たせる。
  `attack(Player target)` で `target.hp -= this.attackPower` する。

sampleAnswer: |
  class Player {
      String name;
      int hp;
      int attackPower;

      void attack(Player target) {
          System.out.println(name + "が" + target.name + "に" + attackPower + "ダメージ！");
          target.hp -= attackPower;
      }

      boolean isAlive() {
          return hp > 0;
      }

      void showStatus() {
          System.out.println(name + " HP: " + hp + (isAlive() ? "" : "（死亡）"));
      }
  }

  public class Main {
      public static void main(String[] args) {
          Player hero = new Player();
          hero.name = "勇者";
          hero.hp = 100;
          hero.attackPower = 30;

          Player monster = new Player();
          monster.name = "スライム";
          monster.hp = 80;
          monster.attackPower = 15;

          hero.attack(monster);
          monster.attack(hero);
          hero.attack(monster);
          monster.attack(hero);
          hero.attack(monster);

          hero.showStatus();
          monster.showStatus();
      }
  }
explanation: |
  **オブジェクト同士の相互作用**：`attack(Player target)` で、`this`（攻撃する側）と `target`（攻撃される側）の2オブジェクトが関わる。

  **状態の変化の追跡**：`hp` がメソッドを通じて変化することで、バトルの経過を表現できる。

  **`isAlive()` の活用**：`showStatus()` の中で `isAlive()` を呼ぶことで、生死を判定する処理を再利用できる。
