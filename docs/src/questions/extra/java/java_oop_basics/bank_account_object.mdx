sampleAnswer: |
  class BankAccount {
      String owner;
      int balance;

      void deposit(int amount) {
          if (amount <= 0) {
              System.out.println("入金額は1円以上でなければなりません");
              return;
          }
          balance += amount;
          System.out.println("[入金] " + amount + "円 → 残高: " + balance + "円");
      }

      void withdraw(int amount) {
          if (amount <= 0) {
              System.out.println("出金額は1円以上でなければなりません");
              return;
          }
          if (amount > balance) {
              System.out.println("[出金失敗] 残高不足（残高: " + balance + "円）");
              return;
          }
          balance -= amount;
          System.out.println("[出金] " + amount + "円 → 残高: " + balance + "円");
      }

      void showBalance() {
          System.out.println(owner + "の残高: " + balance + "円");
      }
  }

  public class Main {
      public static void main(String[] args) {
          BankAccount account = new BankAccount();
          account.owner = "山田太郎";
          account.balance = 0;

          account.deposit(50000);
          account.withdraw(20000);
          account.withdraw(40000);
          account.deposit(-100);
          account.showBalance();
      }
  }
explanation: |
  **入力値の検証**：入金・出金額が負や0の場合を事前にチェックして、不正な操作を防ぐ。

  **早期returnパターン**：条件を満たさない場合に `return;` で処理を打ち切る。これにより `else` ブロックが不要になり、コードが読みやすくなる。

  **残高管理の責務**：お金の出し入れ処理を `BankAccount` オブジェクトに集中させることで、一貫した残高管理が実現できる。

## 問題

`BankAccount` クラスを定義し、入金・出金・残高確認を実装せよ。

**出力**：
```
[入金] 50000円 → 残高: 50000円
[出金] 20000円 → 残高: 30000円
[出金失敗] 残高不足（残高: 30000円）
入金額は1円以上でなければなりません
山田太郎の残高: 30000円
```

**要件**：
- `void deposit(int amount)`, `void withdraw(int amount)`, `void showBalance()` を定義すること
- 不正な操作（負の金額・残高不足）を適切に処理すること
