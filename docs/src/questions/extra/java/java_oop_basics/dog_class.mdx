hint: |
  `class Dog` を定義し、`name`（String）と `breed`（String：犬種）フィールドを持たせる。
  `bark()` メソッドで「〇〇がワンワンと吠えた！」と表示する。
  `introduce()` メソッドで名前と犬種を表示する。
sampleAnswer: |
  class Dog {
      String name;
      String breed;

      void bark() {
          System.out.println(name + "がワンワンと吠えた！");
      }

      void introduce() {
          System.out.println("名前: " + name + ", 犬種: " + breed);
      }
  }

  public class Main {
      public static void main(String[] args) {
          Dog d1 = new Dog();
          d1.name = "ポチ";
          d1.breed = "柴犬";

          Dog d2 = new Dog();
          d2.name = "マックス";
          d2.breed = "ゴールデンレトリバー";

          d1.introduce();
          d1.bark();
          d2.introduce();
          d2.bark();
      }
  }
explanation: |
  **インスタンスメソッド**：クラスに定義したメソッドはインスタンスを通じて呼ぶ（`d1.bark()`）。
  メソッド内では `name` のようにフィールドを直接参照できる（`this.name` の省略形）。

  **クラスの責務**：`Dog` クラスは「犬」に関連するデータ（名前・犬種）と振る舞い（吠える・自己紹介）をまとめている。

  **複数インスタンス**：同じクラスから複数のオブジェクトを作ると、それぞれ独立したフィールドを持つ。

## 問題

名前と犬種を持つ `Dog` クラスを定義し、吠えるメソッドと自己紹介メソッドを実装せよ。

**出力**：
```
名前: ポチ, 犬種: 柴犬
ポチがワンワンと吠えた！
名前: マックス, 犬種: ゴールデンレトリバー
マックスがワンワンと吠えた！
```

**要件**：
- `void bark()` と `void introduce()` を定義すること
- 2つのインスタンスを作ること
