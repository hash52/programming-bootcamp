  `class Circle` を定義し、`radius`（double）フィールドを持たせる。
  `area()` → `Math.PI * radius * radius`
  `circumference()` → `2 * Math.PI * radius`
  `describe()` で半径・面積・周囲長をまとめて表示する。
sampleAnswer: |
  class Circle {
      double radius;

      double area() {
          return Math.PI * radius * radius;
      }

      double circumference() {
          return 2 * Math.PI * radius;
      }

      void describe() {
          System.out.printf("半径: %.1f, 面積: %.2f, 周囲長: %.2f%n",
              radius, area(), circumference());
      }
  }

  public class Main {
      public static void main(String[] args) {
          double[] radii = {1.0, 3.0, 5.0, 10.0};

          for (double r : radii) {
              Circle c = new Circle();
              c.radius = r;
              c.describe();
          }
      }
  }
explanation: |
  **メソッドの相互呼び出し**：`describe()` の中で `area()` と `circumference()` を呼ぶことで、計算ロジックを再利用できる。

  **ループでの生成**：`for` ループでさまざまな半径の円を順に生成して処理するパターン。

  **`printf` のフォーマット**：`%.1f` で小数点以下1桁、`%.2f` で2桁に揃えられる。
