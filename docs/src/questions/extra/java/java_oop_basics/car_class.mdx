  `class Car` を定義し、`brand`（String）、`speed`（int）、`totalDistance`（int）フィールドを持たせる。
  `accelerate(int amount)` で `speed += amount`、`drive(int km)` で `totalDistance += km`。

sampleAnswer: |
  class Car {
      String brand;
      int speed;
      int totalDistance;

      void accelerate(int amount) {
          speed += amount;
          System.out.println(brand + "が加速: " + speed + "km/h");
      }

      void drive(int km) {
          totalDistance += km;
          System.out.println(km + "km走行しました（累計: " + totalDistance + "km）");
      }

      void showStatus() {
          System.out.println("--- " + brand + " の状態 ---");
          System.out.println("速度: " + speed + "km/h");
          System.out.println("累計走行距離: " + totalDistance + "km");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Car car = new Car();
          car.brand = "トヨタ";

          car.accelerate(30);
          car.accelerate(20);
          car.drive(50);
          car.drive(30);
          car.showStatus();
      }
  }
explanation: |
  **状態を変化させるメソッド**：`accelerate` と `drive` はオブジェクトの状態（`speed`, `totalDistance`）を変化させる。
  メソッドを通して状態を変えることで、変化の過程をメッセージとして出力できる。

  **累積値の管理**：`totalDistance += km` のようにフィールドに累積させることで、合計を追跡できる。

  **クラスによるシミュレーション**：車という現実の概念をクラスで表現することで、現実に近い形でコードを書ける。
