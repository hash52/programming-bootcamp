hint: |
  `class Library` を定義し、`String[] books`・`boolean[] available`・`int count` フィールドを持たせる。
  `addBook(String title)` で蔵書を追加する。
  `borrow(String title)` で対象の本が利用可能なら `available[i] = false` にする。
  `returnBook(String title)` で `available[i] = true` に戻す。
  `listBooks()` で全蔵書と「貸出中/在架」状態を表示する。
sampleAnswer: |
  class Library {
      String[] books = new String[20];
      boolean[] available = new boolean[20];
      int count = 0;

      void addBook(String title) {
          books[count] = title;
          available[count] = true;
          count++;
      }

      void borrow(String title) {
          for (int i = 0; i < count; i++) {
              if (books[i].equals(title)) {
                  if (available[i]) {
                      available[i] = false;
                      System.out.println(title + " を貸し出しました");
                  } else {
                      System.out.println(title + " は貸出中です");
                  }
                  return;
              }
          }
          System.out.println(title + " は見つかりません");
      }

      void returnBook(String title) {
          for (int i = 0; i < count; i++) {
              if (books[i].equals(title)) {
                  available[i] = true;
                  System.out.println(title + " が返却されました");
                  return;
              }
          }
          System.out.println(title + " は見つかりません");
      }

      void listBooks() {
          for (int i = 0; i < count; i++) {
              String status = available[i] ? "在架" : "貸出中";
              System.out.println(books[i] + " [" + status + "]");
          }
      }
  }

  public class Main {
      public static void main(String[] args) {
          Library lib = new Library();
          lib.addBook("Java入門");
          lib.addBook("アルゴリズム図鑑");
          lib.addBook("デザインパターン");
          lib.borrow("Java入門");
          lib.borrow("Java入門");
          lib.listBooks();
          lib.returnBook("Java入門");
          lib.listBooks();
      }
  }
explanation: |
  **並行配列**：`books[i]` と `available[i]` は同じインデックスで対応している。
  これを「並行配列（parallel arrays）」と呼ぶ。後のコレクション章ではオブジェクトで管理するより良い方法を学ぶ。

  **線形探索**：タイトルで本を検索する際は、配列を先頭から順に検索する線形探索を使っている。

  **三項演算子**：`available[i] ? "在架" : "貸出中"` のように if-else を1行で書ける三項演算子も有用である。

## 問題

図書館を表す `Library` クラスを定義し、蔵書の追加・貸出・返却管理を実装せよ。

**出力**：
```
Java入門 を貸し出しました
Java入門 は貸出中です
Java入門 [貸出中]
アルゴリズム図鑑 [在架]
デザインパターン [在架]
Java入門 が返却されました
Java入門 [在架]
アルゴリズム図鑑 [在架]
デザインパターン [在架]
```

**要件**：
- `String[] books`・`boolean[] available`・`int count` フィールドを持つこと
- `void addBook(String title)`、`void borrow(String title)`、`void returnBook(String title)`、`void listBooks()` を定義すること
