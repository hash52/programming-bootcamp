  `add(int n)` で枚数を増やし、`total()` で合計金額を返す。

sampleAnswer: |
  class Coin {
      int value;
      int count;

      void add(int n) {
          count += n;
      }

      int total() {
          return value * count;
      }

      void display() {
          System.out.println(value + "円硬貨が" + count + "枚（合計" + total() + "円）");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Coin c = new Coin();
          c.value = 100;
          c.add(3);
          c.display();
          c.add(2);
          c.display();
      }
  }
explanation: |
  **フィールドの組み合わせ**：`value` と `count` の2つのフィールドで、硬貨の状態を表現している。
  複数のフィールドを組み合わせることで、現実の概念をモデル化できる。

  **メソッドからメソッドを呼ぶ**：`display()` の中から `total()` を呼んでいる。
  メソッド内では他のメソッドを直接呼び出せる（`this.total()` の省略形）。

  **戻り値のあるメソッド**：`total()` は `int` を返す。戻り値を使って計算結果を再利用できる。
