  `class Pet` を定義し、`String name`・`String type`・`int hunger`（空腹度0〜10）フィールドを持たせる。

sampleAnswer: |
  class Pet {
      String name;
      String type;
      int hunger;

      void feed() {
          if (hunger > 0) {
              hunger--;
          }
      }

      void status() {
          System.out.println(name + "（" + type + "）空腹度: " + hunger);
      }
  }

  public class Main {
      public static void main(String[] args) {
          Pet p = new Pet();
          p.name = "ミケ";
          p.type = "猫";
          p.hunger = 5;
          p.status();
          p.feed();
          p.feed();
          p.status();
      }
  }
explanation: |
  **状態の制約**：`hunger` を0未満にしないようにif文でガードしている。
  現実の制約をコードで表現することがオブジェクト指向の重要な役割である。

  **複数フィールドの管理**：`name`・`type`・`hunger` の3つのフィールドでペットの状態を表現している。

  **インクリメント/デクリメント**：`hunger--` は `hunger = hunger - 1` と同じである。

## 問題

ペットを表す `Pet` クラスを定義し、名前・種類・空腹度の管理を実装せよ。

**出力**：
```
ミケ（猫）空腹度: 5
ミケ（猫）空腹度: 3
```

**要件**：
- `String name`・`String type`・`int hunger` フィールドを持つこと
- `void feed()`（空腹度を1減らす、0未満にはしない）と `void status()` を定義すること
