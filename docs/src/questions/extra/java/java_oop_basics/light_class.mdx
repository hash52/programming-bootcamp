hint: |
  `class Light` を定義し、`boolean isOn` フィールドを持たせる。
  `turnOn()` で `isOn = true`、`turnOff()` で `isOn = false`、
  `getStatus()` で「電気はON（またはOFF）です」と表示する。
sampleAnswer: |
  class Light {
      boolean isOn;

      void turnOn() {
          isOn = true;
      }

      void turnOff() {
          isOn = false;
      }

      void getStatus() {
          if (isOn) {
              System.out.println("電気はONです");
          } else {
              System.out.println("電気はOFFです");
          }
      }
  }

  public class Main {
      public static void main(String[] args) {
          Light light = new Light();
          light.getStatus();
          light.turnOn();
          light.getStatus();
          light.turnOff();
          light.getStatus();
      }
  }
explanation: |
  **boolean フィールド**：`isOn` のような真偽値フィールドは、状態を表すのに適している。
  初期値は `false`（デフォルト値）なので、最初は「OFF」になる。

  **状態変更メソッド**：`turnOn()` と `turnOff()` はフィールドを書き換えるだけのシンプルなメソッドである。
  このようにオブジェクトの状態を変えるメソッドを「ミューテーター」と呼ぶ。

  **if文でフィールドを参照**：メソッド内から `isOn` に直接アクセスできる点がOOPの基本である。

## 問題

電灯を表す `Light` クラスを定義し、点灯・消灯の状態管理を実装せよ。

**出力**：
```
電気はOFFです
電気はONです
電気はOFFです
```

**要件**：
- `boolean isOn` フィールドを持つこと
- `void turnOn()`、`void turnOff()`、`void getStatus()` を定義すること
