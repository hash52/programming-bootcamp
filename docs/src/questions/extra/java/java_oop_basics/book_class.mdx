  `class Book` を定義し、`title`（String）、`author`（String）、`pages`（int）フィールドを持たせる。
  `describe()` メソッドで情報を表示する。
  `isThickBook()` メソッドで300ページ以上なら `true` を返す（厚い本の判定）。
sampleAnswer: |
  class Book {
      String title;
      String author;
      int pages;

      void describe() {
          System.out.println("「" + title + "」著者: " + author + ", " + pages + "ページ");
      }

      boolean isThickBook() {
          return pages >= 300;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Book b1 = new Book();
          b1.title = "Java入門";
          b1.author = "山田太郎";
          b1.pages = 350;

          Book b2 = new Book();
          b2.title = "アルゴリズム基礎";
          b2.author = "佐藤花子";
          b2.pages = 250;

          Book[] books = {b1, b2};
          for (Book book : books) {
              book.describe();
              System.out.println("厚い本: " + book.isThickBook());
          }
      }
  }
explanation: |
  **boolean を返すメソッド**：`isThickBook()` のように状態を問い合わせるメソッドを定義すると、呼び出し元がロジックを持たなくて済む。

  **オブジェクトの配列**：`Book[]` 型の配列にオブジェクトを入れて for-each で処理できる。

  **クラスによるデータのモデリング**：現実の「本」という概念をクラスで表現する。

## 問題

`Book` クラスを定義し、本の情報を管理するプログラムを実装せよ。

**出力**：
```
「Java入門」著者: 山田太郎, 350ページ
厚い本: true
「アルゴリズム基礎」著者: 佐藤花子, 250ページ
厚い本: false
```

**要件**：
- `void describe()` と `boolean isThickBook()` を定義すること（300ページ以上が厚い本）
- 2冊の本を配列で管理してfor-eachで処理すること
