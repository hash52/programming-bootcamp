  `addScore(int s)` でスコアを追加し、`average()` で平均を返す。

sampleAnswer: |
  class ScoreBoard {
      int[] scores = new int[100];
      int count = 0;

      void addScore(int s) {
          scores[count] = s;
          count++;
      }

      double average() {
          int sum = 0;
          for (int i = 0; i < count; i++) {
              sum += scores[i];
          }
          return (double) sum / count;
      }

      int highest() {
          int max = scores[0];
          for (int i = 1; i < count; i++) {
              if (scores[i] > max) max = scores[i];
          }
          return max;
      }

      void display() {
          System.out.print("スコア: ");
          for (int i = 0; i < count; i++) {
              System.out.print(scores[i] + " ");
          }
          System.out.println();
          System.out.println("平均: " + average() + " 最高: " + highest());
      }
  }

  public class Main {
      public static void main(String[] args) {
          ScoreBoard board = new ScoreBoard();
          board.addScore(85);
          board.addScore(92);
          board.addScore(78);
          board.addScore(95);
          board.display();
      }
  }
explanation: |
  **配列フィールド**：`scores = new int[100]` のようにフィールド宣言と同時に初期化できる。
  `count` で実際に追加された要素数を管理するのが典型的なパターンである。

  **メソッドでのループ処理**：クラスのメソッド内でもfor文を使ってフィールドの配列を処理できる。

  **複数のメソッドが連携**：`display()` から `average()` と `highest()` を呼び出して結果を再利用している。
