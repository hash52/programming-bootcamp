hint: |
  `class Stopwatch` を定義し、`int seconds`（経過秒数）と `boolean running` フィールドを持たせる。
  `start()` で `running = true`、`stop()` で `running = false`、
  `tick()` で `running` が true のとき `seconds` を1増やす。
  `display()` で経過時間を「〇分〇秒」形式で表示する。
sampleAnswer: |
  class Stopwatch {
      int seconds;
      boolean running;

      void start() {
          running = true;
      }

      void stop() {
          running = false;
      }

      void tick() {
          if (running) {
              seconds++;
          }
      }

      void display() {
          int min = seconds / 60;
          int sec = seconds % 60;
          System.out.println(min + "分" + sec + "秒");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Stopwatch sw = new Stopwatch();
          sw.start();
          for (int i = 0; i < 75; i++) {
              sw.tick();
          }
          sw.display();
          sw.stop();
          sw.tick();
          sw.tick();
          sw.display();
      }
  }
explanation: |
  **実行状態の管理**：`running` フラグで計測中かどうかを管理し、`tick()` がそれを確認する。
  このように状態フラグを持つパターンはよく使われる。

  **時間の変換**：秒数を「分と秒」に変換するには `/`（商）と `%`（余り）を使う。
  `75 / 60 = 1分`、`75 % 60 = 15秒` となる。

  **tick方式**：実際のストップウォッチは時間経過を直接知るが、この例では `tick()` を明示的に呼ぶことで時間経過をシミュレートしている。

## 問題

ストップウォッチを表す `Stopwatch` クラスを定義し、計測の開始・停止・経過時間管理を実装せよ。

**出力**：
```
1分15秒
1分15秒
```

**要件**：
- `int seconds` と `boolean running` フィールドを持つこと
- `void start()`、`void stop()`、`void tick()`（計測中のみ秒を増やす）、`void display()` を定義すること
