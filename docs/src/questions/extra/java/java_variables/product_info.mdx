sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          String productName = "ノートPC";
          int price = 89800;
          double weight = 1.35;
          boolean inStock = true;
          System.out.println("商品名: " + productName);
          System.out.println("価格: " + price + "円");
          System.out.println("重量: " + weight + "kg");
          System.out.println("在庫: " + inStock);
      }
  }
explanation: |
  **型の選び方**：文字列は `String`、円などの整数は `int`、kg など小数が必要なら `double`、あり/なしのフラグは `boolean` を使う。

  **計算なしの変数利用**：変数はデータを「名前付きで保持する入れ物」であるため、計算しなくても宣言して出力するだけで使い方の練習になる。

  **文字列連結**：`"価格: " + price + "円"` のように `int` を `+` で囲むと、数値が文字列として連結される。

## 問題

商品情報を4種類の型の変数に格納し、以下の形式で出力するプログラムを実装せよ。

**出力**：
```
商品名: ノートPC
価格: 89800円
重量: 1.35kg
在庫: true
```

**要件**：
- `String` 型で商品名を宣言する
- `int` 型で価格を宣言する
- `double` 型で重量を宣言する
- `boolean` 型で在庫の有無を宣言する
