sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          char grade = 'A';
          System.out.println("成績: " + grade);
          System.out.println("文字コード: " + (int) grade);
      }
  }
explanation: |
  **char型**：1文字を格納する型。`'A'`（シングルクォート）で文字リテラルを表す。

  **文字コードへの変換**：`(int) grade` と型キャストすると、その文字のASCII（文字コード）が得られる。`'A'` は 65、`'B'` は 66、`'a'` は 97 である。

  **Stringとの違い**：`String` は文字「列」（複数文字可）で `"A"` と書く。`char` は文字「1つ」で `'A'` と書く。
