hint: |
  `char` 型の変数を `(int)` キャストするとその文字のASCIIコードが得られる。
  `'A'` は 65、`'a'` は 97、`'0'` は 48 である。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          char upper = 'A';
          char lower = 'a';
          char digit = '0';
          System.out.println(upper + " → " + (int) upper);
          System.out.println(lower + " → " + (int) lower);
          System.out.println(digit + " → " + (int) digit);
      }
  }
explanation: |
  **charとASCIIコード**：`char` 型は内部的に整数（文字コード）として格納されている。`(int)` キャストでその番号を取り出せる。

  **代表的な文字コード**：
  - `'A'`〜`'Z'` は 65〜90
  - `'a'`〜`'z'` は 97〜122
  - `'0'`〜`'9'` は 48〜57

  **連結時の注意**：`upper + " → "` と書くと `char` 型の `upper` は文字列として連結される。しかし `upper + (int) upper` のように `char` と `int` を `+` でつなぐと数値同士の加算になるため注意が必要である。

## 問題

大文字・小文字・数字の `char` 変数を宣言し、それぞれの文字とASCIIコードを一覧で出力するプログラムを実装せよ。

**出力**：
```
A → 65
a → 97
0 → 48
```

**要件**：
- `char upper = 'A'`、`char lower = 'a'`、`char digit = '0'` を宣言する
- `(int)` キャストで各文字のASCIIコードを取得する
- `文字 → コード` の形式で出力する
