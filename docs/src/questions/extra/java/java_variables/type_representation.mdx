  - `double` への代入：`double d = n` → widening 変換で `65.0`

sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 65;
          char asChar = (char) n;
          double asDouble = n;
          String asString = "" + n;
          System.out.println("int    : " + n);
          System.out.println("char   : " + asChar);
          System.out.println("double : " + asDouble);
          System.out.println("String : " + asString);
      }
  }
explanation: |
  **同じ値でも型で見え方が変わる**：`65` という整数は、型によってまったく異なる意味や表示になる。

  **int → char**：`(char) 65` はASCIIコード 65 の文字 `'A'` として解釈される。

  **int → double**：`double asDouble = n` は widening 変換（自動）で `65.0` になる。小数部分 `.0` が付く。

  **int → String**：`"" + n` は空文字列と整数を連結することで `"65"` という文字列になる。数値ではなく文字列として扱われるようになる。

  **型は「データをどう解釈するか」を決める**：コンピュータの中身はただのビット列であり、型がそのビットの意味を決定する。

## 問題

`int n = 65` を `char`・`double`・`String` の3つの型に変換し、型によって表示が変わることを確認するプログラムを実装せよ。

**出力**：
```
int    : 65
char   : A
double : 65.0
String : 65
```

**要件**：
- `(char) n` で `char asChar` に変換する（ASCIIコード 65 = `'A'`）
- `double asDouble = n` で widening 変換する（`65.0` になる）
- `"" + n` で `String asString` に変換する（`"65"` になる）
