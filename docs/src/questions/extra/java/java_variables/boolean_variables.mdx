  `boolean` 型には `true` か `false` のどちらかだけ代入できる。

sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          boolean isStudent = true;
          boolean hasLicense = false;
          boolean canVote = true;
          System.out.println("学生: " + isStudent);
          System.out.println("免許: " + hasLicense);
          System.out.println("投票権: " + canVote);
      }
  }
explanation: |
  **boolean型**：`true`（真）か `false`（偽）の2値のみを格納できる型である。条件を変数として保持したいときに使う。

  **命名慣習**：`isStudent`（学生かどうか）・`hasLicense`（免許を持っているか）のように `is〜` や `has〜` で始めると、読んだだけで「はい/いいえ」を答える変数だと分かりやすい。

  **出力**：`println` で `boolean` 型を出力すると `true` または `false` という文字列として表示される。

## 問題

3つの `boolean` 型変数を宣言し、以下の形式で出力するプログラムを実装せよ。

**出力**：
```
学生: true
免許: false
投票権: true
```

**要件**：
- `isStudent`（学生である）を `true` で宣言する
- `hasLicense`（免許を持っている）を `false` で宣言する
- `canVote`（投票権がある）を `true` で宣言する
