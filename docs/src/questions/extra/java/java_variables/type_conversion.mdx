hint: |
  `int` 同士の割り算（`a / b`）は結果も整数になり、小数点以下が切り捨てられる。
  `(double) a / b` のように片方の変数の前に `(double)` をつけると、小数で計算できる。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int a = 7;
          int b = 2;
          double result = (double) a / b;
          System.out.println(a + " / " + b + " = " + result);
      }
  }
explanation: |
  **整数除算の落とし穴**：`int` 同士の演算結果は `int` になるため、`7 / 2` は `3.5` ではなく `3` になる（小数点以下切り捨て）。

  **型キャスト（キャスト演算子）**：`(double) a` と書くと `a` を `double` として扱えるようになる。`double` と `int` の演算は結果が `double` になるため、`(double) a / b` は正しく `3.5` を返す。

  **キャストの位置に注意**：`(double)(a / b)` と書くと先に整数除算（`7 / 2 = 3`）が行われてから double に変換されるため `3.0` になってしまう。除算の前にキャストすることが重要である。

## 問題

変数 `a = 7`、`b = 2` が宣言されている。`a ÷ b` の結果を小数点以下まで正確に計算して出力するプログラムを実装せよ。

**出力**：
```
7 / 2 = 3.5
```

**要件**：
- 変数 `a`・`b` は変更しない
- 型キャストを使って `double` として計算する
- 出力形式は `{a} / {b} = {result}` とする
