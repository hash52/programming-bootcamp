  `a >= b` が `true` なら `a` の方が大きい（または等しい）。
  `else` の場合は `b` の方が大きい。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int a = 15;
          int b = 23;
          if (a >= b) {
              System.out.println("大きい方: " + a);
          } else {
              System.out.println("大きい方: " + b);
          }
      }
  }
explanation: |
  **`>=` で等値を考慮**：`a > b` ではなく `a >= b` にすることで、`a == b` のとき（両方同じ値）も `a` を出力できる。どちらを出力しても正しいが、`>=` を使う方が一般的である。

  **if-else の基本**：2択の判定は `if`-`else` の基本パターン。条件が `true` なら `if` ブロック、`false` なら `else` ブロックが実行される。
