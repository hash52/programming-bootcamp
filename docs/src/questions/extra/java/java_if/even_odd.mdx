sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 14;
          if (n % 2 == 0) {
              System.out.println(n + " は偶数です");
          } else {
              System.out.println(n + " は奇数です");
          }
      }
  }
explanation: |
  **偶数の定義**：2で割り切れる数（余りが0）が偶数。`n % 2 == 0` で確認できる。

  **`%` と if の組み合わせ**：余り演算子 `%` は if文の条件式の中で非常によく使われるパターンである。「〇の倍数か」の判定に応用できる。

  **3の倍数の場合**：`n % 3 == 0` で判定できる。FizzBuzz問題でも同じパターンを使う。
