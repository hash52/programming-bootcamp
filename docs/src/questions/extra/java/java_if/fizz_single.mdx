sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 15;
          if (n % 3 == 0 && n % 5 == 0) {
              System.out.println("FizzBuzz");
          } else if (n % 3 == 0) {
              System.out.println("Fizz");
          } else if (n % 5 == 0) {
              System.out.println("Buzz");
          } else {
              System.out.println(n);
          }
      }
  }
explanation: |
  **FizzBuzz の判定順序**：「3と5の両方の倍数」（15, 30, 45...）を最初に判定する。これを後回しにすると、15が先に `Fizz` として処理されてしまう。

  **`&&` 演算子**：`n % 3 == 0 && n % 5 == 0` は「3の倍数 かつ 5の倍数」。`n % 15 == 0` でも同じ意味になる。

  **動作確認**：`n` の値を 3, 5, 15, 7 に変えて出力を確認してみよう。

## 問題

1つの整数 `n` が Fizz・Buzz・FizzBuzz・それ以外のどれかを判定して出力するプログラムを実装せよ。

**ルール**：
- 3と5の両方の倍数 → `FizzBuzz`
- 3の倍数 → `Fizz`
- 5の倍数 → `Buzz`
- それ以外 → 数字をそのまま表示

**実行例**（`n = 15` のとき）：
```
FizzBuzz
```

**実行例**（`n = 9` のとき）：
```
Fizz
```
