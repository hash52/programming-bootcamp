## 問題

変数 `year` に設定された年がうるう年かどうかを判定して出力するプログラムを実装せよ。

**うるう年の条件**：
1. 400で割り切れる年 → うるう年
2. 100で割り切れる年（①を除く）→ 平年
3. 4で割り切れる年（②を除く）→ うるう年
4. それ以外 → 平年

**出力**（`year = 2024` のとき）：
```
2024年はうるう年です
```

**確認用の値**：
- 2024 → うるう年（4の倍数）
- 1900 → 平年（100の倍数だが400でない）
- 2000 → うるう年（400の倍数）
