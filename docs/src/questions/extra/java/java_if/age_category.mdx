sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int age = 25;
          if (age < 13) {
              System.out.println(age + "歳: 子ども");
          } else if (age < 18) {
              System.out.println(age + "歳: 中高生");
          } else if (age < 65) {
              System.out.println(age + "歳: 大人");
          } else {
              System.out.println(age + "歳: 高齢者");
          }
      }
  }
explanation: |
  **条件を小さい値から判定**：`age < 13` を先に判定することで、次の `age < 18` は「13以上18未満」を意味する。わざわざ `age >= 13 && age < 18` と書かなくてよい。

  **4分岐の構造**：`if` → `else if` → `else if` → `else` と連ねることで4つの場合分けができる。

  **境界値のテスト**：`age` に 0・12・13・17・18・64・65 を代入して動作を確認しよう。

## 問題

変数 `age = 25` の年齢区分を以下の基準で判定して出力するプログラムを実装せよ。

**判定基準**：

| 年齢 | 区分 |
|------|------|
| 13歳未満 | 子ども |
| 13歳以上18歳未満 | 中高生 |
| 18歳以上65歳未満 | 大人 |
| 65歳以上 | 高齢者 |

**出力**（`age = 25` のとき）：
```
25歳: 大人
```
