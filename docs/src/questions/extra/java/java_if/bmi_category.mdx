sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          double bmi = 22.5;
          if (bmi < 18.5) {
              System.out.println("BMI " + bmi + ": 低体重");
          } else if (bmi < 25.0) {
              System.out.println("BMI " + bmi + ": 普通体重");
          } else if (bmi < 30.0) {
              System.out.println("BMI " + bmi + ": 過体重");
          } else {
              System.out.println("BMI " + bmi + ": 肥満");
          }
      }
  }
explanation: |
  **double型の比較**：`double` 型の変数も `<`・`>=` で比較できる。`==` は浮動小数点の誤差で意図しない動作になるため、範囲の比較には `<` や `>=` を使う。

  **下から判定する利点**：`bmi < 18.5` を最初に判定することで、2番目の `bmi < 25.0` は「18.5以上25.0未満」を意味する。範囲を重ねて書く必要がない。

  **境界値の確認**：BMI = 18.5・25.0・30.0 ちょうどの場合が正しく判定されるかテストしよう。

## 問題

変数 `bmi = 22.5` の体重区分を以下の基準（WHO基準）で判定して出力するプログラムを実装せよ。

**判定基準**：

| BMI | 区分 |
|-----|------|
| 18.5未満 | 低体重 |
| 18.5以上25.0未満 | 普通体重 |
| 25.0以上30.0未満 | 過体重 |
| 30.0以上 | 肥満 |

**出力**（`bmi = 22.5` のとき）：
```
BMI 22.5: 普通体重
```
