sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 12;
          if (n % 3 == 0) {
              System.out.println(n + "は3の倍数です");
          } else {
              System.out.println(n + "は3の倍数ではありません");
          }
      }
  }
explanation: |
  **余りで倍数を判定**：`n % 3` が 0 のとき、n は3で割り切れる（=3の倍数）。

  **`%` 演算子の活用**：FizzBuzz や偶奇判定など、倍数の判定には `%` がよく使われる。偶数なら `n % 2 == 0`、5の倍数なら `n % 5 == 0` と同じパターンが使える。
