  `if` / `else if` / `else` の3分岐で判定できる。

sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int n = 7;
          if (n > 0) {
              System.out.println(n + " は正の数です");
          } else if (n < 0) {
              System.out.println(n + " は負の数です");
          } else {
              System.out.println(n + " はゼロです");
          }
      }
  }
explanation: |
  **3分岐の構造**：`if` → `else if` → `else` の順に条件を評価する。最初に一致した分岐だけが実行される。

  **ゼロを `else` で処理**：`n > 0` でも `n < 0` でもない場合は必ずゼロなので、`else` で受け取れる。

  **値を変えて確認**：`n` に `7`・`-3`・`0` を代入して実行し、それぞれ正しく出力されることを確認しよう。

## 問題

変数 `n = 7` に対して、正の数・負の数・ゼロのいずれかを判定して出力するプログラムを実装せよ。

**出力**（`n = 7` のとき）：
```
7 は正の数です
```

**要件**：
- `if` / `else if` / `else` の3分岐を使う
- `n` の値を `-3` や `0` に変えても正しく動作すること
