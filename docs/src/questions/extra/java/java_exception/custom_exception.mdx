hint: |
  `class 例外クラス名 extends Exception` で独自例外を定義する。
  コンストラクタで `super(メッセージ)` を呼ぶことでメッセージを設定できる。
  `throw new 例外クラス名("メッセージ")` で例外をスローする。
sampleAnswer: |
  class NegativeAgeException extends Exception {
      NegativeAgeException(String message) {
          super(message);
      }
  }

  public class Main {
      static void setAge(int age) throws NegativeAgeException {
          if (age < 0) {
              throw new NegativeAgeException("年齢は0以上でなければなりません: " + age);
          }
          System.out.println("年齢を設定しました: " + age);
      }

      public static void main(String[] args) {
          try {
              setAge(25);
              setAge(-5);
              setAge(30);
          } catch (NegativeAgeException e) {
              System.out.println("エラー: " + e.getMessage());
          }
      }
  }
explanation: |
  **独自例外の作り方**：`Exception`（または `RuntimeException`）を継承したクラスを作る。
  `super(message)` で `Exception` の親コンストラクタを呼び、メッセージを設定する。

  **`throw`**：例外を発生させるキーワード。`new` でインスタンスを作ってから `throw` する。

  **`throws`**：メソッドのシグネチャに書く。そのメソッドが特定の検査例外を投げる可能性があることを示す。

  **検査例外と非検査例外**：`Exception` を継承した独自例外は検査例外（呼び出し側で `try-catch` か `throws` が必要）。`RuntimeException` を継承すると非検査例外になる。

## 問題

負の年齢が渡された場合にスローする `NegativeAgeException` を定義し、メソッドで年齢の検証を行うプログラムを実装せよ。

**出力**：
```
年齢を設定しました: 25
エラー: 年齢は0以上でなければなりません: -5
```

**要件**：
- `NegativeAgeException extends Exception` を定義すること
- `setAge` メソッドが `throws NegativeAgeException` で宣言されていること
- `main` で `try-catch` を使うこと
