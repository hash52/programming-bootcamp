sampleAnswer: |
  import java.time.LocalDate;
  import java.time.format.DateTimeFormatter;
  import java.time.temporal.ChronoUnit;
  import java.util.ArrayList;

  public class Main {
      public static void main(String[] args) {
          LocalDate today = LocalDate.of(2026, 3, 15);
          DateTimeFormatter fmt = DateTimeFormatter.ofPattern("M月d日(E)");

          ArrayList<LocalDate> schedule = new ArrayList<>();
          schedule.add(LocalDate.of(2026, 3, 20));
          schedule.add(LocalDate.of(2026, 4, 1));
          schedule.add(LocalDate.of(2026, 4, 15));
          schedule.add(LocalDate.of(2026, 5, 3));

          System.out.println("=== スケジュール一覧 ===");
          System.out.println("（基準日: " + today.format(fmt) + "）\n");

          for (LocalDate date : schedule) {
              long days = ChronoUnit.DAYS.between(today, date);
              System.out.println(date.format(fmt) + " → " + days + "日後");
          }
      }
  }
explanation: |
  **`ArrayList<LocalDate>`**：日付オブジェクトをコレクションで管理する。ソートや検索も容易にできる。

  **`DateTimeFormatter` の再利用**：同じフォーマッターを複数の日付に使い回す。`DateTimeFormatter` は不変オブジェクトなので安全に共有できる。

  **日付とコレクションの組み合わせ**：実務でよく使うパターン。スケジュール管理・期限管理・締め切り計算など。
