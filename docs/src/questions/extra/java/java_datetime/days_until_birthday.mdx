hint: |
  `ChronoUnit.DAYS.between(start, end)` で2つの日付の間の日数を求める。
  今年の誕生日が過ぎている場合は、来年の誕生日を対象にする。
  `LocalDate.of(today.getYear(), birthMonth, birthDay)` で今年の誕生日を作れる。
sampleAnswer: |
  import java.time.LocalDate;
  import java.time.temporal.ChronoUnit;

  public class Main {
      public static void main(String[] args) {
          LocalDate today = LocalDate.of(2026, 3, 15);
          int birthMonth = 7;
          int birthDay = 20;

          LocalDate birthday = LocalDate.of(today.getYear(), birthMonth, birthDay);

          if (birthday.isBefore(today) || birthday.isEqual(today)) {
              birthday = birthday.plusYears(1);
          }

          long daysUntil = ChronoUnit.DAYS.between(today, birthday);
          System.out.println("今日: " + today);
          System.out.println("次の誕生日: " + birthday);
          System.out.println("誕生日まで: " + daysUntil + "日");
      }
  }
explanation: |
  **`ChronoUnit.DAYS.between()`**：2つの `Temporal`（日付や時刻）間の差を指定した単位で計算する。
  `WEEKS`, `MONTHS`, `YEARS` なども使える。戻り値は `long`（開始が終了より後なら負の値）。

  **今年の誕生日の扱い**：今年の誕生日がすでに過ぎていたら、来年に設定し直す。`isBefore()` と `plusYears(1)` を組み合わせる。

  **`between(start, end)`**：「start から end まで何日」を返す。`between(today, birthday)` は今日から誕生日までの日数。

## 問題

今日（`LocalDate.of(2026, 3, 15)` と仮定）から次の誕生日（7月20日）までの日数を計算するプログラムを作れ。

**出力**：
```
今日: 2026-03-15
次の誕生日: 2026-07-20
誕生日まで: 127日
```

**要件**：
- `ChronoUnit.DAYS.between()` で日数を計算すること
- 今年の誕生日が過ぎていたら来年の誕生日を使うこと
