hint: |
  通常のforループでインデックスを末尾から先頭に向けて動かす。
  開始: `i = 配列名.length - 1`（最後の要素）
  条件: `i >= 0`（先頭まで続ける）
  更新: `i--`（1ずつ減らす）
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[] data = {10, 20, 30, 40, 50};

          System.out.println("逆順:");
          for (int i = data.length - 1; i >= 0; i--) {
              System.out.println(data[i]);
          }
      }
  }
explanation: |
  **逆順ループ**：`for (int i = data.length - 1; i >= 0; i--)` が基本パターン。
  `data.length - 1` が最後の要素のインデックス（5要素なら4）。

  **for-eachでは逆順にできない**：for-eachは常に先頭から末尾の順で処理する。
  逆順・飛び飛び・インデックスが必要な処理は通常の for ループを使う。
