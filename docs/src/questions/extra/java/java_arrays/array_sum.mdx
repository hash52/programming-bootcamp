hint: |
  合計を格納する変数 `sum` を0で初期化し、ループで各要素を足し続ける。
  `sum += n;` と書くと `sum = sum + n;` と同じ意味になる。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[] numbers = {3, 7, 12, 5, 8};
          int sum = 0;

          for (int n : numbers) {
              sum += n;
          }

          System.out.println("合計: " + sum);
      }
  }
explanation: |
  **累積和パターン**：`sum = 0` で初期化し、ループで `sum += 要素` を繰り返すのが基本パターン。
  このパターンは配列の合計・最大値・カウントなど多くの場面で使う。

  **初期値を忘れずに**：`int sum;` だけで初期化しないとコンパイルエラーになる。
  合計なら0、最大値なら `Integer.MIN_VALUE`（または配列の先頭要素）が一般的な初期値。
