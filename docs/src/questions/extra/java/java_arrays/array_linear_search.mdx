hint: |
  インデックス付き for ループで全要素を順に調べる。
  `data[i] == target` のとき「インデックスX で見つかった」と表示する。
  ループが終わっても見つからなかった場合は「見つからなかった」と表示する。
  見つかったかどうかを `boolean found` で管理すると便利。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[] data = {15, 32, 8, 47, 23, 56};
          int target = 47;
          boolean found = false;

          for (int i = 0; i < data.length; i++) {
              if (data[i] == target) {
                  System.out.println(target + " はインデックス " + i + " で見つかった");
                  found = true;
                  break;
              }
          }

          if (!found) {
              System.out.println(target + " は見つからなかった");
          }
      }
  }
explanation: |
  **線形探索（リニアサーチ）**：先頭から順に1つずつ調べる最もシンプルな探索アルゴリズム。
  計算量は O(n)（要素数に比例）。

  **`break`**：見つかった時点でループを抜けると無駄な処理を省ける。

  **`boolean found`**：見つかったかどうかを追跡するフラグ変数。ループ後に「見つからなかった」を判定するために使う。
