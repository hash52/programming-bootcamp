  2次元配列は「配列の配列」。外側のループで行（row）を、内側のループで列（column）を処理する。
  `matrix.length` が行数、`matrix[i].length` が各行の列数。
  for-eachを2重にする方法：`for (int[] row : matrix) { for (int n : row) { ... } }`
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[][] matrix = {
              {1, 2, 3},
              {4, 5, 6},
              {7, 8, 9}
          };
          int sum = 0;

          for (int[] row : matrix) {
              for (int n : row) {
                  sum += n;
              }
          }

          System.out.println("合計: " + sum);
      }
  }
explanation: |
  **2次元配列のループ**：外側のfor-eachで各行（`int[]`型）を取り出し、内側のfor-eachで各要素を処理する。

  **`int[][] matrix`**：`int[][]` は「int配列の配列」を意味する。`matrix[0]` は `{1, 2, 3}` という1次元配列。

  **合計の確認**：1+2+3+4+5+6+7+8+9 = 45
