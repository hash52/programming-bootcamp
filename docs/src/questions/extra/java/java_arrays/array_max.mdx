  最大値を格納する変数 `max` を配列の先頭要素で初期化する（`max = values[0]`）。
  ループで各要素と比較し、`max` より大きければ `max` を更新する。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[] values = {23, 67, 12, 89, 45};
          int max = values[0];

          for (int v : values) {
              if (v > max) {
                  max = v;
              }
          }

          System.out.println("最大値: " + max);
      }
  }
explanation: |
  **最大値探索パターン**：`max = values[0]`（先頭要素）で初期化し、残りの要素と順に比較して更新する。
  `Integer.MIN_VALUE`（最小整数値）で初期化する方法もあるが、先頭要素での初期化が自然でわかりやすい。

  **なぜ先頭要素で初期化？**：`max = 0` で初期化すると、配列が全て負の値の場合に誤った結果になる。
  先頭要素なら必ず配列内の値なので安全。
