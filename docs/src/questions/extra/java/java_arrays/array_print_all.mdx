hint: |
  `for-each`（拡張for文）を使うと配列の全要素を順番に取り出せる。
  書き方：`for (int n : scores) { ... }`
  各要素を `System.out.println()` で出力する。
sampleAnswer: |
  public class Main {
      public static void main(String[] args) {
          int[] scores = {85, 92, 78, 95, 60};

          for (int score : scores) {
              System.out.println(score);
          }
      }
  }
explanation: |
  **for-each文**：`for (型 変数名 : 配列名)` の形式で、配列の先頭から末尾まで順に要素を取り出せる。
  インデックスを意識せずに書けるため、全要素を順番に処理したいときに最適。

  **通常のforとの使い分け**：インデックスが不要なときは for-each、インデックスが必要なとき（逆順・部分参照など）は通常の for を使う。
