hint: |
  `class Robot implements Movable, Speakable` のようにカンマ区切りで複数のインターフェースを実装できる。
  Javaはクラスの継承は1つだけだが、インターフェースの実装は何個でも可能。
  実装したインターフェースの全メソッドを定義する必要がある。
sampleAnswer: |
  interface Movable {
      void move(String direction);
  }

  interface Speakable {
      void speak(String message);
  }

  class Robot implements Movable, Speakable {
      String name;

      Robot(String name) {
          this.name = name;
      }

      @Override
      public void move(String direction) {
          System.out.println(name + "が" + direction + "に移動");
      }

      @Override
      public void speak(String message) {
          System.out.println(name + "「" + message + "」");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Robot r = new Robot("R2D2");

          r.speak("こんにちは！");
          r.move("北");
          r.move("東");
          r.speak("目的地に到達しました");

          // Movable型としてもSpeakable型としても使える
          Movable m = r;
          m.move("南");

          Speakable s = r;
          s.speak("さようなら！");
      }
  }
explanation: |
  **多重実装**：`implements Movable, Speakable` のように、複数のインターフェースを実装できる。
  クラスが「移動できる」かつ「話せる」という複数の能力を表現できる。

  **インターフェース型での参照**：`Robot` インスタンスを `Movable` 型や `Speakable` 型の変数で参照できる。
  それぞれの型では、そのインターフェースで定義されたメソッドのみ呼べる。

  **継承との違い**：クラスの継承（`extends`）は1つだけだが、インターフェースの実装（`implements`）は無制限。これがインターフェースの大きな利点。

## 問題

`Movable` と `Speakable` の2つのインターフェースを定義し、`Robot` クラスで両方を実装するプログラムを作れ。

**出力**：
```
R2D2「こんにちは！」
R2D2が北に移動
R2D2が東に移動
R2D2「目的地に到達しました」
R2D2が南に移動
R2D2「さようなら！」
```

**要件**：
- `Movable` と `Speakable` の2つのインターフェースを定義すること
- `class Robot implements Movable, Speakable` で実装すること
