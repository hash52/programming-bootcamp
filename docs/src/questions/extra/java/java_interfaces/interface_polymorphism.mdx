hint: |
  `Drawable` インターフェースを定義し、`Circle`, `Square`, `Star` で実装する。
  `Drawable d = new Circle();` のようにインターフェース型で変数を宣言できる。
  `d.draw()` を呼ぶと実際の型のメソッドが実行される（ポリモーフィズム）。
sampleAnswer: |
  interface Drawable {
      void draw();
      String getShapeName();
  }

  class Circle implements Drawable {
      double radius;

      Circle(double radius) {
          this.radius = radius;
      }

      @Override
      public void draw() {
          System.out.println("○ 円（半径" + radius + "）を描画");
      }

      @Override
      public String getShapeName() {
          return "円";
      }
  }

  class Square implements Drawable {
      double side;

      Square(double side) {
          this.side = side;
      }

      @Override
      public void draw() {
          System.out.println("□ 正方形（辺" + side + "）を描画");
      }

      @Override
      public String getShapeName() {
          return "正方形";
      }
  }

  class Star implements Drawable {
      int points;

      Star(int points) {
          this.points = points;
      }

      @Override
      public void draw() {
          System.out.println("★ " + points + "角の星を描画");
      }

      @Override
      public String getShapeName() {
          return points + "角星";
      }
  }

  public class Main {
      public static void main(String[] args) {
          Drawable[] shapes = {
              new Circle(5.0),
              new Square(4.0),
              new Star(5)
          };

          System.out.println("描画する図形: " + shapes.length + "個");
          for (Drawable d : shapes) {
              System.out.print(d.getShapeName() + " → ");
              d.draw();
          }
      }
  }
explanation: |
  **インターフェース型でのポリモーフィズム**：`Drawable[]` 配列に異なる実装クラスを格納し、同じ `draw()` 呼び出しで各クラス固有の描画処理を実行できる。

  **インターフェースに複数メソッド**：`draw()` と `getShapeName()` のように、インターフェースに複数のメソッドを定義できる。

  **実装の分離**：インターフェースを使うことで、「何ができるか（インターフェース）」と「どうやってやるか（実装）」を分離できる。

## 問題

`Drawable` インターフェースを定義し、`Circle`, `Square`, `Star` でポリモーフィズムを実現するプログラムを作れ。

**出力**：
```
描画する図形: 3個
円 → ○ 円（半径5.0）を描画
正方形 → □ 正方形（辺4.0）を描画
5角星 → ★ 5角の星を描画
```

**要件**：
- `interface Drawable { void draw(); String getShapeName(); }` を定義すること
- `Drawable[]` 型の配列で3クラスを管理すること
