sampleAnswer: |
  interface Printable {
      void print();
  }

  class Document implements Printable {
      String title;
      int pages;

      Document(String title, int pages) {
          this.title = title;
          this.pages = pages;
      }

      @Override
      public void print() {
          System.out.println("[文書] " + title + " (" + pages + "ページ) を印刷中...");
      }
  }

  class Photo implements Printable {
      String filename;
      String resolution;

      Photo(String filename, String resolution) {
          this.filename = filename;
          this.resolution = resolution;
      }

      @Override
      public void print() {
          System.out.println("[写真] " + filename + " (" + resolution + ") を印刷中...");
      }
  }

  class Invoice implements Printable {
      String recipient;
      int amount;

      Invoice(String recipient, int amount) {
          this.recipient = recipient;
          this.amount = amount;
      }

      @Override
      public void print() {
          System.out.println("[請求書] " + recipient + "宛 " + amount + "円 を印刷中...");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Printable[] items = {
              new Document("Java入門", 200),
              new Photo("photo01.jpg", "4K"),
              new Invoice("田中商事", 50000)
          };

          for (Printable p : items) {
              p.print();
          }
      }
  }
explanation: |
  **インターフェースによる統一操作**：`Document`, `Photo`, `Invoice` は継承関係がないが、全て `Printable` を実装することで、同一の `print()` メソッドで処理できる。

  **疎結合の実現**：インターフェースを使うことで、実際の型（`Document`/`Photo`/`Invoice`）を知らなくても `print()` を呼べる。

  **インターフェースは多重実装可能**：クラスの継承は1つしかできないが、インターフェースは複数 `implements` できる。

## 問題

`Printable` インターフェースを定義し、`Document`, `Photo`, `Invoice` の3クラスで実装するプログラムを作れ。

**出力**：
```
[文書] Java入門 (200ページ) を印刷中...
[写真] photo01.jpg (4K) を印刷中...
[請求書] 田中商事宛 50000円 を印刷中...
```

**要件**：
- `interface Printable { void print(); }` を定義すること
- 3つのクラスに `implements Printable` すること
- `Printable[]` 型の配列で一括処理すること
