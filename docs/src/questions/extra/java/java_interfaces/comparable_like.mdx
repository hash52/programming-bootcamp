  `interface Comparable` ならぬ、独自の `Rankable` インターフェースで `int getRank()` を定義する。
  `getRank()` の戻り値でオブジェクト同士を比較できる。

sampleAnswer: |
  interface Rankable {
      int getRank();
      String getName();
  }

  class Student implements Rankable {
      String name;
      int score;

      Student(String name, int score) {
          this.name = name;
          this.score = score;
      }

      @Override
      public int getRank() {
          return score;
      }

      @Override
      public String getName() {
          return name;
      }
  }

  class Employee implements Rankable {
      String name;
      int yearsOfService;

      Employee(String name, int yearsOfService) {
          this.name = name;
          this.yearsOfService = yearsOfService;
      }

      @Override
      public int getRank() {
          return yearsOfService;
      }

      @Override
      public String getName() {
          return name;
      }
  }

  public class Main {
      static Rankable findTop(Rankable[] items) {
          Rankable top = items[0];
          for (Rankable r : items) {
              if (r.getRank() > top.getRank()) {
                  top = r;
              }
          }
          return top;
      }

      public static void main(String[] args) {
          Rankable[] students = {
              new Student("田中", 85),
              new Student("佐藤", 92),
              new Student("山田", 78)
          };

          Rankable topStudent = findTop(students);
          System.out.println("最高点学生: " + topStudent.getName() + " (" + topStudent.getRank() + "点)");

          Rankable[] employees = {
              new Employee("Aさん", 5),
              new Employee("Bさん", 12),
              new Employee("Cさん", 8)
          };

          Rankable topEmployee = findTop(employees);
          System.out.println("最長勤続: " + topEmployee.getName() + " (" + topEmployee.getRank() + "年)");
      }
  }
explanation: |
  **インターフェースによる汎用アルゴリズム**：`findTop()` メソッドは `Rankable` 型で書かれているため、`Student` でも `Employee` でも動作する。

  **インターフェースの抽象化力**：比較のロジック（どのフィールドでランク付けするか）は各クラスが `getRank()` で決める。`findTop()` はその詳細を知らなくてよい。

  **Javaの `Comparable` インターフェース**：標準ライブラリには `Comparable<T>` インターフェースがあり、`compareTo()` メソッドで大小比較を定義する。`Collections.sort()` や `Arrays.sort()` はこれを使う。

## 問題

`Rankable` インターフェースを定義し、`Student` と `Employee` で実装。最高ランクを見つける汎用メソッドを実装せよ。

**出力**：
```
最高点学生: 佐藤 (92点)
最長勤続: Bさん (12年)
```

**要件**：
- `interface Rankable { int getRank(); String getName(); }` を定義すること
- `findTop(Rankable[])` メソッドで最高ランクを返すこと
