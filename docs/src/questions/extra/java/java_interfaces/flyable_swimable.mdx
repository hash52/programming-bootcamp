hint: |
  `interface Flyable { void fly(); }` と `interface Swimmable { void swim(); }` を定義する。
  `Duck` はどちらも実装し、`Eagle` は `Flyable` のみ、`Fish` は `Swimmable` のみ実装する。
  `instanceof` で特定のインターフェースを実装しているか確認できる。
sampleAnswer: |
  interface Flyable {
      void fly();
  }

  interface Swimmable {
      void swim();
  }

  class Eagle implements Flyable {
      String name;

      Eagle(String name) {
          this.name = name;
      }

      @Override
      public void fly() {
          System.out.println(name + "が大空を飛んでいる");
      }
  }

  class Fish implements Swimmable {
      String name;

      Fish(String name) {
          this.name = name;
      }

      @Override
      public void swim() {
          System.out.println(name + "が水中を泳いでいる");
      }
  }

  class Duck implements Flyable, Swimmable {
      String name;

      Duck(String name) {
          this.name = name;
      }

      @Override
      public void fly() {
          System.out.println(name + "が飛び立った");
      }

      @Override
      public void swim() {
          System.out.println(name + "が池で泳いでいる");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Object[] animals = {new Eagle("ワシ"), new Fish("サケ"), new Duck("アヒル")};

          for (Object a : animals) {
              if (a instanceof Flyable) {
                  ((Flyable) a).fly();
              }
              if (a instanceof Swimmable) {
                  ((Swimmable) a).swim();
              }
          }
      }
  }
explanation: |
  **インターフェースの組み合わせ**：`Duck implements Flyable, Swimmable` で2つの能力を持たせる。これが多重インターフェースの強力な表現力。

  **`instanceof` でインターフェース確認**：`a instanceof Flyable` で、`a` が `Flyable` を実装しているか確認できる（クラスに限らずインターフェースもチェック可能）。

  **能力ベースの設計**：継承は「種類」（is-a）、インターフェースは「能力」（can-do）を表す。この設計の違いを意識すること。

## 問題

`Flyable` と `Swimmable` インターフェースを定義し、`Eagle`, `Fish`, `Duck` で組み合わせて実装するプログラムを作れ。

**出力**：
```
ワシが大空を飛んでいる
サケが水中を泳いでいる
アヒルが飛び立った
アヒルが池で泳いでいる
```

**要件**：
- `Eagle` は `Flyable` のみ、`Fish` は `Swimmable` のみ、`Duck` は両方を実装すること
- `instanceof` でインターフェースを確認すること
