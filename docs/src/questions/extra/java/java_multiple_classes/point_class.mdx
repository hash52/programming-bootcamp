hint: |
  `class Point` を `Main` クラスとは別に定義する（同じファイル内に `public` なしで定義する）。
  `Point` クラスにはフィールド `x` と `y`（int型）と、`display()` メソッドを持たせる。
  `main` では `new Point()` でインスタンスを生成し、フィールドに値を代入してから `display()` を呼ぶ。
sampleAnswer: |
  class Point {
      int x;
      int y;

      void display() {
          System.out.println("Point(" + x + ", " + y + ")");
      }
  }

  public class Main {
      public static void main(String[] args) {
          Point p1 = new Point();
          p1.x = 3;
          p1.y = 5;
          p1.display();

          Point p2 = new Point();
          p2.x = -1;
          p2.y = 7;
          p2.display();
      }
  }
explanation: |
  **複数クラスの定義**：1つのファイルに複数のクラスを定義できる。ただし `public` クラスはファイル名と同じ名前の1つだけ。
  それ以外のクラスは `public` なしで定義する。

  **インスタンスの生成**：`new Point()` で `Point` クラスのインスタンス（オブジェクト）を作る。
  フィールドにはドット（`.`）でアクセスする（`p1.x = 3`）。

  **複数インスタンス**：同じクラスから複数のオブジェクトを作ることができ、それぞれが独立したフィールドを持つ。

## 問題

`x` と `y` の2つの整数フィールドを持つ `Point` クラスを定義し、`main` から2つのインスタンスを生成して表示せよ。

**出力**：
```
Point(3, 5)
Point(-1, 7)
```

**要件**：
- `Point` クラスには `int x`, `int y` フィールドと `display()` メソッドを定義すること
- `main` で2つのインスタンスを生成してフィールドに値を設定すること
