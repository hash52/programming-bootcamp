sampleAnswer: |
  class Rectangle {
      double width;
      double height;

      double area() {
          return width * height;
      }

      double perimeter() {
          return 2 * (width + height);
      }
  }

  public class Main {
      public static void main(String[] args) {
          Rectangle r = new Rectangle();
          r.width = 4.0;
          r.height = 6.0;

          System.out.println("幅: " + r.width + ", 高さ: " + r.height);
          System.out.println("面積: " + r.area());
          System.out.println("周囲長: " + r.perimeter());
      }
  }
explanation: |
  **インスタンスメソッド**：クラスに定義したメソッドは、インスタンスのフィールドを使って計算できる。
  `area()` の中で `width` と `height` を参照するのは、そのインスタンス自身のフィールドを意味する。

  **戻り値型の選択**：面積や周囲長は小数になり得るため `double` を使う。

  **クラスの役割**：関連するデータ（幅・高さ）と操作（面積・周囲長の計算）をまとめることがクラスの目的。
