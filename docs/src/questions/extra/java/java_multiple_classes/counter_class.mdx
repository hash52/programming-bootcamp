hint: |
  `class Counter` を定義し、`count`（int）フィールドを持たせる。
  `increment()` で `count++`、`decrement()` で `count--`、`reset()` で `count = 0`、
  `getCount()` で `count` を返す。
sampleAnswer: |
  class Counter {
      int count = 0;

      void increment() {
          count++;
      }

      void decrement() {
          count--;
      }

      void reset() {
          count = 0;
      }

      int getCount() {
          return count;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Counter c = new Counter();
          System.out.println("初期値: " + c.getCount());

          c.increment();
          c.increment();
          c.increment();
          System.out.println("3回increment後: " + c.getCount());

          c.decrement();
          System.out.println("1回decrement後: " + c.getCount());

          c.reset();
          System.out.println("reset後: " + c.getCount());
      }
  }
explanation: |
  **状態を持つクラス**：`count` フィールドはオブジェクトの「状態」を表す。
  メソッドを通して状態を変化させるのがオブジェクト指向の基本。

  **メソッドによる操作の抽象化**：`count++` を直接書くより `increment()` を呼ぶ方が、何をしているかが明確になる。
  また、将来ロジックを変えたいときにメソッド内だけ変えればよい。

  **戻り値なしのメソッド**：`void` は戻り値がないことを意味する。状態を変えるだけのメソッドは `void` にする。

## 問題

カウンタの値を管理する `Counter` クラスを実装せよ。

**出力**：
```
初期値: 0
3回increment後: 3
1回decrement後: 2
reset後: 0
```

**要件**：
- `void increment()`, `void decrement()`, `void reset()`, `int getCount()` メソッドを定義すること
