hint: |
  `class Circle` を定義し、`radius`（double）フィールドを持たせる。
  `area()` → `Math.PI * radius * radius`
  `circumference()` → `2 * Math.PI * radius`
  `isLargerThan(Circle other)` → `this.radius > other.radius` で比較する。
sampleAnswer: |
  class Circle {
      double radius;

      double area() {
          return Math.PI * radius * radius;
      }

      double circumference() {
          return 2 * Math.PI * radius;
      }

      boolean isLargerThan(Circle other) {
          return this.radius > other.radius;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Circle c1 = new Circle();
          c1.radius = 5.0;

          Circle c2 = new Circle();
          c2.radius = 3.0;

          System.out.printf("円1（半径%.1f）: 面積=%.2f, 周囲=%.2f%n",
              c1.radius, c1.area(), c1.circumference());
          System.out.printf("円2（半径%.1f）: 面積=%.2f, 周囲=%.2f%n",
              c2.radius, c2.area(), c2.circumference());

          if (c1.isLargerThan(c2)) {
              System.out.println("円1の方が大きい");
          } else {
              System.out.println("円2の方が大きい");
          }
      }
  }
explanation: |
  **`this` キーワード**：`this.radius` は「このオブジェクト自身の `radius`」を意味する。
  `isLargerThan(Circle other)` の中で `this.radius` と `other.radius` を比較することで、自分と相手の半径を区別できる。

  **`Math.PI`**：`3.141592653589793` という定数。円の計算には必須。

  **クラス同士の比較**：`isLargerThan` のように「別のインスタンスと自分を比べる」メソッドを作ることができる。
