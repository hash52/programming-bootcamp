sampleAnswer: |
  class Calculator {
      double add(double a, double b) {
          return a + b;
      }

      double subtract(double a, double b) {
          return a - b;
      }

      double multiply(double a, double b) {
          return a * b;
      }

      double divide(double a, double b) {
          if (b == 0) {
              System.out.println("エラー: 0で割ることはできません");
              return Double.NaN;
          }
          return a / b;
      }
  }

  public class Main {
      public static void main(String[] args) {
          Calculator calc = new Calculator();
          System.out.println("10 + 3 = " + calc.add(10, 3));
          System.out.println("10 - 3 = " + calc.subtract(10, 3));
          System.out.println("10 × 3 = " + calc.multiply(10, 3));
          System.out.println("10 ÷ 3 = " + calc.divide(10, 3));
          System.out.println("10 ÷ 0 = " + calc.divide(10, 0));
      }
  }
explanation: |
  **0除算の対処**：`b == 0` のとき除算を行うと `ArithmeticException` が発生する（intの場合）。
  `double` の場合は `Infinity` が返るが、チェックして明示的にエラーメッセージを出す方が丁寧。

  **`Double.NaN`**：「Not a Number」を表す特別な値。エラー時の戻り値として使える。

  **クラスへのまとめ方**：電卓の4機能をまとめることで、`calc.add(10, 3)` のように直感的に使えるAPIになる。
