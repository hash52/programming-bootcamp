hint: |
  `class ToDoItem` を定義し、`title`（String）と `done`（boolean、初期値false）フィールドを持たせる。
  `complete()` メソッドで `done = true` にする。
  `display()` メソッドで完了状態に応じて `[✓]` または `[ ]` を表示する。
  `main` でToDoItemの配列を作り、一部を完了にしてから全件表示する。
sampleAnswer: |
  class ToDoItem {
      String title;
      boolean done = false;

      void complete() {
          done = true;
      }

      void display() {
          String mark = done ? "[✓]" : "[ ]";
          System.out.println(mark + " " + title);
      }
  }

  public class Main {
      public static void main(String[] args) {
          ToDoItem[] items = new ToDoItem[4];

          items[0] = new ToDoItem();
          items[0].title = "Javaの勉強";

          items[1] = new ToDoItem();
          items[1].title = "買い物";

          items[2] = new ToDoItem();
          items[2].title = "運動";

          items[3] = new ToDoItem();
          items[3].title = "メールの返信";

          items[0].complete();
          items[3].complete();

          System.out.println("=== ToDoリスト ===");
          for (ToDoItem item : items) {
              item.display();
          }
      }
  }
explanation: |
  **オブジェクトの配列**：`new ToDoItem[4]` でオブジェクトを格納する配列を作れる。
  各要素はさらに `new ToDoItem()` で初期化する必要がある。

  **boolean フィールドの初期値**：`boolean done = false;` で宣言時に初期値を設定できる。

  **三項演算子の活用**：`done ? "[✓]" : "[ ]"` で完了状態に応じたマークを簡潔に選べる。
