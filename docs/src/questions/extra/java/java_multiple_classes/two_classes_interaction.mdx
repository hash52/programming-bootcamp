sampleAnswer: |
  class Student {
      String name;
      int score;

      void display() {
          System.out.println("学生: " + name + ", 点数: " + score);
      }
  }

  class Teacher {
      String name;

      void evaluate(Student student) {
          System.out.print(name + "先生 → " + student.name + "の評価: ");
          if (student.score >= 80) {
              System.out.println("優秀");
          } else if (student.score >= 60) {
              System.out.println("良好");
          } else {
              System.out.println("要努力");
          }
      }
  }

  public class Main {
      public static void main(String[] args) {
          Teacher teacher = new Teacher();
          teacher.name = "田中";

          Student s1 = new Student();
          s1.name = "鈴木";
          s1.score = 85;

          Student s2 = new Student();
          s2.name = "佐藤";
          s2.score = 55;

          s1.display();
          s2.display();

          teacher.evaluate(s1);
          teacher.evaluate(s2);
      }
  }
explanation: |
  **クラス間の連携**：`Teacher.evaluate(Student student)` のように、あるクラスのメソッドが別クラスのオブジェクトを引数に受け取ることができる。

  **オブジェクトを引数に渡す**：Javaではオブジェクト（参照型）を引数に渡すと、同じオブジェクトを参照する。

  **責務の分離**：採点の評価ロジックを `Teacher` クラスに集約することで、`Student` はデータ保持に専念できる。

## 問題

`Student` クラスと `Teacher` クラスを定義し、先生が学生を評価するプログラムを実装せよ。

**出力**：
```
学生: 鈴木, 点数: 85
学生: 佐藤, 点数: 55
田中先生 → 鈴木の評価: 優秀
田中先生 → 佐藤の評価: 要努力
```

**要件**：
- `Teacher.evaluate(Student student)` のように、一方のクラスのメソッドが他方のオブジェクトを受け取ること
- 点数が80以上なら「優秀」、60以上なら「良好」、それ未満なら「要努力」と評価すること
