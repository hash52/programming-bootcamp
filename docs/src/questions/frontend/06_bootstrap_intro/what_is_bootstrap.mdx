  Bootstrap は Twitter 社が開発したオープンソースのCSSフレームワークである。
  ボタン・ナビゲーション・カードなどのUIコンポーネントがあらかじめ用意されており、HTML要素に特定のクラスを付けるだけで整ったデザインを実現できる。
  自分でCSSをゼロから書く必要がなく、開発効率を大幅に向上させられる点が特徴である。
