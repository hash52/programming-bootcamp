explanation: |
  Bootstrap のボタンスタイルは `btn` クラスを基本とし、カラークラスを組み合わせて使う。

  ```html
  <button class="btn btn-primary">ボタン</button>
  ```

  - `btn-primary`、`btn-secondary`、`btn-success`、`btn-danger` 等: 塗りつぶしボタン
  - `btn-outline-*`: アウトライン（枠線のみ）ボタン
  - `btn-lg`、`btn-sm`: ボタンのサイズを変更する
  - `ms-2`（margin-start）等のスペーシングユーティリティと組み合わせて余白を調整できる
