explanation: |
  Bootstrap のカードコンポーネントは、`card` クラスを持つ div を外側に配置し、その中に `card-body` クラスの div を入れる構造である。

  ```html
  <div class="card">
    <div class="card-body">
      <h5 class="card-title">タイトル</h5>
      <p class="card-text">テキスト</p>
    </div>
  </div>
  ```

  - `card-title`: カードの見出しに使うクラス
  - `card-text`: カードの本文テキストに使うクラス
  - `card-footer`: カードの下部エリアを定義するクラス
  - カード内には btn クラスのリンクやボタンを配置することもできる
