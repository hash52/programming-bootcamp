explanation: |
  CDN（Content Delivery Network）とは、世界中に分散配置されたサーバー群からコンテンツを配信するネットワークである。
  ユーザーに最も近いサーバーからファイルが配信されるため、読み込みが高速になる。
  Bootstrap を CDN で導入する場合、リンクタグを HTML に追加するだけでよく、ファイルのダウンロードやインストールが不要になる。
