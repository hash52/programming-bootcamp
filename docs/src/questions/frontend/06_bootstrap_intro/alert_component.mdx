    </div>

explanation: |
  Bootstrap のアラートコンポーネントは、`alert` クラスにカラークラスを組み合わせて使う。

  ```html
  <div class="alert alert-success" role="alert">
    登録が完了しました。
  </div>
  ```

  - `alert-success`: 成功メッセージ（緑）
  - `alert-warning`: 警告メッセージ（黄）
  - `alert-danger`: エラーメッセージ（赤）
  - `alert-info`: 情報メッセージ（水色）

  `role="alert"` はスクリーンリーダーなどの支援技術に対してアラートであることを伝えるアクセシビリティ属性であり、付与することが推奨される。
