explanation: |
  Bootstrap のユーティリティクラスを使うと、CSS を書かずにスタイルを指定できる。

  主なクラスの種類：
  - `text-*`（text-primary, text-danger 等）: テキストの色を指定する
  - `bg-*`（bg-warning, bg-success 等）: 背景色を指定する
  - `fw-bold`、`fw-normal`: フォントの太さを指定する
  - `fs-*`（fs-1〜fs-6）: フォントサイズを指定する
  - `text-center`、`text-end`、`text-start`: テキストの配置を指定する
  - `p-*`、`m-*`（p-1〜p-5, m-1〜m-5）: パディングとマージンを指定する
  - `rounded`: 角丸を指定する

  これらを組み合わせることで、様々なスタイルをCSSなしで実現できる。
