sampleHtmlCode: |
  <!DOCTYPE html>
  <html lang="ja">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap CDN</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
  </head>
  <body>
    <div class="container mt-3">
      <h1 class="text-primary">Bootstrap導入テスト</h1>
      <p class="lead">Bootstrapが正しく読み込まれていればこの文字は青色になる。</p>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
  </body>
  </html>
explanation: |
  Bootstrap 5 を CDN で導入するには、`<head>` タグ内に CSS のリンクタグを追加し、`</body>` の直前に JavaScript の script タグを追加する。

  ```html
  <!-- <head> 内に追加 -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">

  <!-- </body> の直前に追加 -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
  ```

  これにより、ファイルをダウンロードせずに Bootstrap のクラスが使えるようになる。
  `viewport` メタタグ（`<meta name="viewport" content="width=device-width, initial-scale=1.0">`）も忘れずに追加すること。
