explanation: |
  モバイルファーストでは基本スタイルをスマホ向けに書き、`@media (min-width: ...)` で大きい画面向けに上書きする。

  ```css
  /* 基本（スマホ向け） */
  .layout { flex-direction: column; }

  /* タブレット以上で上書き */
  @media (min-width: 768px) {
    .layout { flex-direction: row; }
  }
  ```

  デスクトップファーストとの違い：
  - **モバイルファースト**: `min-width` → 基本がスマホ、大きい画面で拡張
  - **デスクトップファースト**: `max-width` → 基本がPC、小さい画面で縮小

  モバイルファーストのほうがシンプルなCSSになりやすく、パフォーマンスも良い。
