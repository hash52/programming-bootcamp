explanation: |
  **レスポンシブデザイン（Responsive Design）** とは、スマートフォン・タブレット・PCなど、異なる画面サイズに応じてレイアウトが自動的に変化するデザイン手法である。

  1つのHTMLファイルで複数のデバイスに対応できるため、デバイスごとに別のページを作る必要がない。

  実現方法：
  - **CSS メディアクエリ**: 画面幅に応じてスタイルを切り替える
  - **Bootstrap グリッド**: ブレークポイントでカラム数を変える
  - **Flexbox / CSS Grid**: 柔軟なレイアウトを実現する
---

レスポンシブデザインの説明として最も適切なものを選べ。
