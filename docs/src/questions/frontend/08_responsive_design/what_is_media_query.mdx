explanation: |
  **メディアクエリ（Media Query）** とは、画面幅・解像度・向きなどの条件に応じてCSSスタイルを切り替える仕組みである。

  ```css
  /* 768px以上のときにスタイルを適用 */
  @media (min-width: 768px) {
    .container {
      display: flex;
    }
  }
  ```

  よく使うブレークポイントの目安：
  - `576px`: スマホ横向き（Bootstrap の sm）
  - `768px`: タブレット（Bootstrap の md）
  - `992px`: 小型PC（Bootstrap の lg）
  - `1200px`: 大型PC（Bootstrap の xl）

  Bootstrap を使う場合はメディアクエリを直接書く機会は少ないが、カスタムCSSと組み合わせる際に知っておくと便利である。
