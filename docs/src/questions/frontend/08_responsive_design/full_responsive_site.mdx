explanation: |
  Bootstrap の主要コンポーネントを組み合わせて完全なレスポンシブサイトを構築する。

  構成要素：
  1. **navbar**: ハンバーガーメニュー付きのナビゲーション
  2. **ヒーローセクション**: `bg-light py-5 text-center` で目立つ導入エリア
  3. **カードグリッド**: `row g-3` + `col-12 col-sm-6 col-lg-4` のレスポンシブ配置
  4. **フッター**: `bg-dark text-white` のシンプルなフッター

  Bootstrap JS（`bootstrap.bundle.min.js`）はナビゲーションの折りたたみ機能に必要である。
  `display-4`・`lead` はタイポグラフィユーティリティで、大きなテキストを表現できる。
