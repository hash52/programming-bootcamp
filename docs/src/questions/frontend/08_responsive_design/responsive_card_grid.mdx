explanation: |
  レスポンシブなカードグリッドは、Bootstrap のグリッドとカードコンポーネントを組み合わせて作る。

  ```html
  <div class="row g-3">
    <div class="col-12 col-sm-6 col-lg-4">
      <div class="card h-100">...</div>
    </div>
  </div>
  ```

  ポイント：
  - `g-3`: カード間のガター（余白）を指定する
  - `h-100`: カードの高さを親要素に合わせる（カードの高さが揃う）
  - `col-12 col-sm-6 col-lg-4`: スマホ1列・タブレット2列・PC3列
