    <h4>お問い合わせ</h4>

      </div>
      <div class="mb-3">
        <label class="form-label">メッセージ</label>

      </div>

    </form>
  </div>
explanation: |
  フォーム内でも Bootstrap のグリッドを使って姓・名フィールドを横並びにできる。

  ```html
  <div class="row mb-3">
    <div class="col-12 col-sm-6 mb-3 mb-sm-0">
      <label class="form-label">姓</label>
      <input type="text" class="form-control">
    </div>
    <div class="col-12 col-sm-6">
      <label class="form-label">名</label>
      <input type="text" class="form-control">
    </div>
  </div>
  ```

  `mb-3 mb-sm-0` はスマホ時のみ下余白を付け、sm以上では余白をなくすことを意味する。
  これによりスマホでは縦並び・タブレット以上では横並びになる。
