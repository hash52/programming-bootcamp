sampleHtmlCode: |
  <form>
    <input type="text" name="username" required placeholder="ユーザー名">
    <input type="email" name="email" required placeholder="メールアドレス">
    <button type="submit">送信</button>
  </form>
explanation: |
  `required` 属性を付けると、そのフィールドが空欄のまま送信しようとしたときにブラウザが自動でエラーを表示し、送信をブロックする。

  ```html
  <input type="text" name="username" required>
  ```

  - `required` は値を持たない **論理属性** であり、属性名を書くだけで有効になる
  - テキスト入力、メール入力、セレクトボックスなど、ほとんどのフォーム要素に使用できる
  - バリデーションはブラウザが行うため、JavaScriptは不要である
