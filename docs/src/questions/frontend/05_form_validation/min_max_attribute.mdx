  </form>
explanation: |
  `min` 属性と `max` 属性を使うと、入力できる値の範囲を制限できる。

  ```html
  <input type="number" min="0" max="120">
  ```

  - `min`: 入力できる最小値。これより小さい値を入力すると送信時にエラーが表示される
  - `max`: 入力できる最大値。これより大きい値を入力すると送信時にエラーが表示される
  - `type="number"` のほか、`type="date"` や `type="range"` でも使用できる
  - テキスト入力の文字数制限には `minlength` / `maxlength` を使う点に注意する
