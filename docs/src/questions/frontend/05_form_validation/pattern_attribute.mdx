explanation: |
  `pattern` 属性に **正規表現** を指定すると、特定の形式の入力のみを受け付けられる。

  ```html
  <input type="text" pattern="[0-9]{3}-[0-9]{4}" title="ハイフン付き7桁の郵便番号を入力してください">
  ```

  よく使う正規表現パターンの例：
  - `[0-9]{3}-[0-9]{4}`: 郵便番号（例: 123-4567）
  - `[0-9]{10,11}`: 電話番号（10〜11桁の数字）
  - `[A-Za-z]+`: アルファベットのみ
  - `[a-zA-Z0-9]+`: 英数字のみ

  `title` 属性を組み合わせると、バリデーション失敗時にブラウザが表示するエラーメッセージに `title` の内容が追加される。
  ユーザーに入力形式を分かりやすく伝えるために積極的に使用するとよい。
