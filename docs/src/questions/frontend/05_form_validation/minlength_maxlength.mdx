    <button type="submit">送信</button>
  </form>
explanation: |
  `minlength` 属性と `maxlength` 属性を使うと、テキスト入力欄の文字数を制限できる。

  ```html
  <input type="text" minlength="3" maxlength="20">
  ```

  - `minlength`: 入力できる最小文字数。これより短いと送信時にエラーが表示される
  - `maxlength`: 入力できる最大文字数。これを超える文字は入力できなくなる
  - `type="text"` や `type="password"` などのテキスト系入力欄で使用できる
  - 数値入力（`type="number"`）には使用できないため、数値の範囲制限には `min` / `max` 属性を使う
