explanation: |
  `placeholder` 属性を使うと、入力欄が空のときにグレーのヒントテキストを表示できる。

  ```html
  <input type="email" placeholder="メールアドレスを入力してください">
  ```

  - 入力欄にフォーカスしても `placeholder` は表示され続け、文字を入力すると消える
  - ラベルの代わりに使うのは避けること。アクセシビリティの観点から `<label>` タグを併用することが推奨される
  - 入力形式の例（例: `example@mail.com`）や補足説明を表示するのに適している
