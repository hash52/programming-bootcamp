explanation: |
  CSSをHTMLに適用する方法は **3種類** ある。

  **1. 外部CSSファイル**（推奨）
  ```html
  <link rel="stylesheet" href="style.css">
  ```

  **2. `<style>` タグ内（内部スタイル）**
  ```html
  <style>
    p { color: red; }
  </style>
  ```

  **3. `style` 属性（インラインスタイル）**
  ```html
  <p style="color: red;">テキスト</p>
  ```

  通常は **外部CSSファイルを使う方法が推奨** される。HTMLとCSSを分離することで保守性が上がり、複数のHTMLファイルに同じスタイルを適用しやすくなる。
