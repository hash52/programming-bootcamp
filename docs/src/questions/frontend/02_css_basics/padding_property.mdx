explanation: |
  `padding` プロパティは要素の **内側の余白** を指定するプロパティである。

  `margin` との違い：
  - **margin**: 要素の **外側** の余白（隣の要素との間隔）
  - **padding**: 要素の **内側** の余白（境界線とコンテンツの間隔）

  ```css
  .card {
    padding: 12px;          /* 内側の余白を12px */
    background-color: white;
  }
  ```

  背景色はpadding領域まで広がるが、margin領域には広がらない。

  個別に指定するプロパティ：
  - `padding-top`, `padding-right`, `padding-bottom`, `padding-left`
