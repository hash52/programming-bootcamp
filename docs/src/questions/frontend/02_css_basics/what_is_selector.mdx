explanation: |
  **セレクタ** は、CSSでスタイルを適用したいHTML要素を指定するための記法である。

  ```css
  /* pタグすべてに適用 */
  p {
    color: blue;
  }

  /* class="highlight"の要素に適用 */
  .highlight {
    background-color: yellow;
  }

  /* id="header"の要素に適用 */
  #header {
    font-size: 24px;
  }
  ```

  セレクタの種類：
  - **要素セレクタ**: タグ名で指定（例: `p`, `h1`, `div`）
  - **クラスセレクタ**: `.(クラス名)` で指定
  - **IDセレクタ**: `#(ID名)` で指定
---

CSSのセレクタの説明として最も適切なものを選べ。
