explanation: |
  `margin` プロパティは要素の **外側の余白** を指定するプロパティである。

  ```css
  p {
    margin: 16px;           /* 上下左右すべて16px */
    margin: 10px 20px;      /* 上下10px、左右20px */
    margin: 5px 10px 15px 20px; /* 上・右・下・左の順 */
  }
  ```

  個別に指定するプロパティ：
  - `margin-top`: 上の余白
  - `margin-right`: 右の余白
  - `margin-bottom`: 下の余白
  - `margin-left`: 左の余白

  `margin: 0 auto;` を使うと要素を中央揃えにできる（要素に幅が指定されている場合）。
