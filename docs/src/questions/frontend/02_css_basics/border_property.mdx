explanation: |
  `border` プロパティは要素の **枠線** を指定するプロパティである。

  ```css
  .box {
    border: 1px solid black; /* 幅・スタイル・色をまとめて指定 */
  }
  ```

  borderの3つの値：
  - **幅**: `1px`, `2px` など
  - **スタイル**: `solid`（実線）, `dashed`（破線）, `dotted`（点線）など
  - **色**: `black`, `#333`, `rgb(0,0,0)` など

  個別に指定するプロパティ：
  - `border-width`: 枠線の幅
  - `border-style`: 枠線のスタイル
  - `border-color`: 枠線の色
  - `border-radius`: 角を丸くする
