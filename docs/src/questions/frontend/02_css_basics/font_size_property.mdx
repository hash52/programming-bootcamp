explanation: |
  `font-size` プロパティは **文字サイズ** を指定するプロパティである。

  ```css
  h1 {
    font-size: 32px;   /* ピクセル単位 */
    font-size: 2rem;   /* ルート要素のフォントサイズの2倍 */
    font-size: 150%;   /* 親要素のフォントサイズの150% */
  }
  ```

  単位の種類：
  - **px（ピクセル）**: 画面上の固定サイズ。最もよく使われる
  - **rem**: ルート要素（`<html>`）のフォントサイズを基準にした相対サイズ
  - **em**: 親要素のフォントサイズを基準にした相対サイズ
  - **%**: 親要素のフォントサイズに対するパーセンテージ
