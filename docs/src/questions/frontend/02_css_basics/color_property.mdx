explanation: |
  `color` プロパティは **文字色** を指定するプロパティである。

  ```css
  p {
    color: red;          /* 色名で指定 */
    color: #ff0000;      /* 16進数カラーコードで指定 */
    color: rgb(255,0,0); /* RGB値で指定 */
  }
  ```

  色の指定方法：
  - **色名**: `red`, `blue`, `green`, `black`, `white` などの英語名
  - **16進数**: `#ff0000`（赤）, `#0000ff`（青）など
  - **RGB値**: `rgb(255, 0, 0)`（赤）など
