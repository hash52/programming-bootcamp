explanation: |
  `background-color` プロパティは **背景色** を指定するプロパティである。

  ```css
  div {
    background-color: lightblue;     /* 色名で指定 */
    background-color: #add8e6;       /* 16進数で指定 */
    background-color: rgba(0,0,255,0.1); /* 半透明の青 */
  }
  ```

  - `background-color` は背景色のみを変える
  - `background` プロパティを使うと背景色・背景画像などをまとめて指定できる
  - `transparent`（透明）を指定することもできる
