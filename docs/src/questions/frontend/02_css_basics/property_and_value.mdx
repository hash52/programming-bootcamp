explanation: |
  CSSのルールは **プロパティ** と **値** の組み合わせで記述する。

  ```css
  セレクタ {
    プロパティ: 値;
  }
  ```

  例：
  ```css
  p {
    color: red;        /* プロパティ: color、値: red */
    font-size: 16px;   /* プロパティ: font-size、値: 16px */
  }
  ```

  - **プロパティ**: 変更したいスタイルの種類（`color`, `font-size`, `margin` など）
  - **値**: そのプロパティをどのように設定するか（`red`, `16px`, `10px` など）
  - プロパティと値はコロン（`:`）で区切り、末尾にセミコロン（`;`）を付ける
