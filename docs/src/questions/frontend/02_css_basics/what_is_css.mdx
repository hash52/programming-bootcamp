explanation: |
  CSSは **Cascading Style Sheets** の略で、Webページの **見た目（スタイル）** を記述するための言語である。

  - **HTML**: ページの構造と内容（「何を表示するか」）
  - **CSS**: ページの見た目（「どのように見せるか」）

  CSSを使うことで、文字の色・サイズ、背景色、余白、レイアウトなどを細かく指定できる。

  ```css
  p {
    color: red;
    font-size: 16px;
  }
  ```
