explanation: |
  Bootstrap の navbar コンポーネントは `<nav>` タグに複数のクラスを組み合わせて使う。

  ```html
  <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container">
      <a class="navbar-brand" href="#">サイト名</a>
      <div class="navbar-nav ms-auto">
        <a class="nav-link" href="#">リンク</a>
      </div>
    </div>
  </nav>
  ```

  主要クラスの役割：
  - `navbar-expand-lg`: lg以上の画面でメニューを展開（それ以下では折りたたむ）
  - `navbar-dark`: 暗い背景向けのテキスト色設定
  - `bg-dark`: 背景色を黒に設定
  - `navbar-brand`: ロゴやサイト名のリンク
  - `ms-auto`: メニューを右端に寄せる
