        <div class="row">
          <div class="col-6 bg-primary text-white p-2">内側: col-6</div>

      </div>

      </div>
    </div>

explanation: |
  グリッドは入れ子にできる。`col` の中にさらに `row > col` を入れると、内側も12列として計算される。

  ```html
  <div class="row">
    <div class="col-8">
      <!-- 内側のグリッド（8列の中の12列） -->
      <div class="row">
        <div class="col-6">左半分</div>
        <div class="col-6">右半分</div>
      </div>
    </div>
    <div class="col-4">サイドバー</div>
  </div>
  ```

  内側の `col-6` は外側の `col-8` の半分（全体の約33%）になる。
  複雑なレイアウトを組む際に活用できる。
