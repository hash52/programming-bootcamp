explanation: |
  Bootstrap のグリッドシステムは、ページを12列に分割してレイアウトを構成する仕組みである。
  基本的な階層構造は container > row > col となる。
  col-{数字} で12列中何列分の幅を占めるかを指定でき、ブレークポイント（sm / md / lg 等）を組み合わせることで、画面サイズに応じてカラム幅を変更するレスポンシブデザインを実現できる。
