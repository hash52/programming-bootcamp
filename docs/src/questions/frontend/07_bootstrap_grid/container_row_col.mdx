explanation: |
  Bootstrap グリッドの3つのクラスにはそれぞれ明確な役割がある。
  container はコンテンツ全体の最大幅を制御し、画面中央に配置する役割を持つ（container-fluid を使うと全幅になる）。
  row はカラムを横並びにするための行であり、内部で Flexbox が設定されている。
  col は実際のコンテンツを配置するカラムで、col-{数字} で12列中の幅を指定する。
  この container > row > col の3層構造を正しく守ることがレイアウト崩れを防ぐ基本となる。
