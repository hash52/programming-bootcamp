    <div class="row">

      </div>
      <div class="col-12 col-md-12 col-lg-4 bg-success text-white p-2 mb-2">
        カラム3（スマホ:全幅、タブレット:全幅、PC:3分の1）
      </div>
    </div>
  </div>
explanation: |
  `col-12 col-md-6 col-lg-4` のようにブレークポイントごとにクラスを重ねて指定する。

  ```html
  <div class="col-12 col-md-6 col-lg-4">
    コンテンツ
  </div>
  ```

  ブレークポイント一覧：
  - `col-*`: すべての画面サイズ（576px未満）
  - `col-sm-*`: 576px以上
  - `col-md-*`: 768px以上
  - `col-lg-*`: 992px以上
  - `col-xl-*`: 1200px以上

  小さい画面のスタイルが基本となり、大きい画面のブレークポイントで上書きされる（モバイルファースト）。
