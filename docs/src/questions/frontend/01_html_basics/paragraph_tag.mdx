explanation: |
  `<p>` タグは **段落（paragraph）** を表すタグである。ブロック要素であり、前後に余白が付く。

  ```html
  <p>最初の段落のテキストをここに書く。</p>
  <p>次の段落のテキストをここに書く。</p>
  ```

  - `<p>` タグはブロック要素のため、自動的に前後で改行される
  - 段落と段落の間には自動的に余白（マージン）が付く
  - `<br>` タグで強制改行することもできるが、段落の区切りには `<p>` タグを使うのが適切である
