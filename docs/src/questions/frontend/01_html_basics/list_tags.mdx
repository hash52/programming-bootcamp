explanation: |
  リストには2種類ある。

  **順序なしリスト（`<ul>`）** - 箇条書き（bullet points）
  ```html
  <ul>
    <li>りんご</li>
    <li>みかん</li>
    <li>ぶどう</li>
  </ul>
  ```

  **順序ありリスト（`<ol>`）** - 番号付きリスト
  ```html
  <ol>
    <li>まず準備する</li>
    <li>次に実行する</li>
    <li>最後に確認する</li>
  </ol>
  ```

  - `<li>` は list item の略で、各リスト項目を表す
  - `<ul>` と `<ol>` の子要素には `<li>` のみを使う
