explanation: |
  `<span>` タグは **汎用的なインラインコンテナ** である。テキストの一部をグループ化するために使う。

  ```html
  <p>今日の気温は<span class="highlight">30度</span>です。</p>
  ```

  - `<div>` がブロック要素なのに対し、`<span>` はインライン要素である
  - テキストの一部にスタイルを当てたい場合によく使う
  - `class` や `id` 属性と組み合わせてCSSを適用する
  - 前後に改行は入らず、テキストと同じ行に並ぶ
