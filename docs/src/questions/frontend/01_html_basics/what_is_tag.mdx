explanation: |
  HTMLの **タグ** は、コンテンツの種類や意味を指定するための記法である。山かっこ（`<` と `>`）で要素名を囲んで書く。

  ```html
  <h1>見出し</h1>
  <p>段落</p>
  <a href="...">リンク</a>
  ```

  - 通常は **開始タグ** と **終了タグ** がペアになる
  - タグには属性（attribute）を持たせることができる
  - タグを使って文書の構造を表現する
