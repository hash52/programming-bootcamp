explanation: |
  `<div>` タグは **汎用的なブロックコンテナ** である。それ自体には意味はないが、複数の要素をグループ化するために使う。

  ```html
  <div class="card">
    <h2>カードのタイトル</h2>
    <p>カードの説明文</p>
  </div>
  ```

  - CSSでスタイルを当てるためのグループ化によく使われる
  - `class` や `id` 属性と組み合わせて使うことが多い
  - `<div>` はブロック要素のため、前後に改行が入る
  - テキストをまとめる場合はインライン要素の `<span>` を使う
