explanation: |
  HTMLは **HyperText Markup Language** の略で、Webページの **構造と内容** を記述するためのマークアップ言語である。

  - **HTML**: ページの構造・内容（見出し、段落、リンクなど）
  - **CSS**: ページのデザイン・レイアウト（色、フォント、配置など）
  - **JavaScript**: ページの動的な動作（クリック処理、アニメーションなど）

  Webページはこの3つの技術が組み合わさって成り立っている。
