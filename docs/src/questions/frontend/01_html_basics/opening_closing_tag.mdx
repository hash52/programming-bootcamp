explanation: |
  開始タグと終了タグはペアで使い、その間にコンテンツを書く。

  ```html
  <p>これが段落のコンテンツです。</p>
  ```

  - **開始タグ**: `<タグ名>` の形式。コンテンツの始まりを示す
  - **終了タグ**: `</タグ名>` の形式。スラッシュ（/）が付く。コンテンツの終わりを示す
  - 開始タグと終了タグで囲まれた部分を **要素（element）** と呼ぶ

  ただし、`<img>` や `<br>` のように終了タグを持たない **空要素** も存在する。
---

HTMLの開始タグと終了タグの役割として最も適切なものを選べ。
