explanation: |
  見出しタグは **h1〜h6** の6段階があり、数字が小さいほど重要度が高い（大きな見出し）である。

  ```html
  <h1>最も重要な見出し（大）</h1>
  <h2>第2レベルの見出し</h2>
  <h3>第3レベルの見出し</h3>
  ```

  - `<h1>` はページ全体のタイトルに使う（1ページに1つが推奨）
  - `<h2>` は章のタイトル、`<h3>` は節のタイトルといった使い方をする
  - 見た目だけでなく、SEO（検索エンジン最適化）でも重要な役割を持つ
