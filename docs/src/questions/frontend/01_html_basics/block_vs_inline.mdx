explanation: |
  ブロック要素とインライン要素は表示の仕方が異なる。

  **ブロック要素**（例：`<div>`, `<p>`, `<h1>`〜`<h6>`, `<ul>` など）
  - 前後に改行が入る
  - 横幅は親要素いっぱいに広がる
  - 高さ・幅の指定ができる

  **インライン要素**（例：`<span>`, `<a>`, `<strong>`, `<img>` など）
  - 前後に改行が入らない（テキストと同じ行に並ぶ）
  - コンテンツの幅だけを占める
  - 通常、高さ・幅の指定はできない
---

ブロック要素とインライン要素の違いとして最も適切なものを選べ。
