explanation: |
  `<a>` タグは **アンカー（anchor）** タグで、リンクを作成するために使う。

  ```html
  <a href="https://example.com">例のサイト</a>
  ```

  - `href` 属性にリンク先のURLを指定する
  - `target="_blank"` を追加すると新しいタブで開く
  - ページ内リンクは `href="#id名"` のように `#` を使う
