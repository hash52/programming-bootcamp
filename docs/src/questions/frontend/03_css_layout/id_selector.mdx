explanation: |
  **IDセレクタ** はHTMLの `id` 属性の値に対してスタイルを適用する。シャープ（`#`）の後にID名を書く。

  ```css
  /* IDセレクタ */
  #header {
    height: 100px;
    background-color: navy;
  }
  ```

  ```html
  <div id="header">ヘッダー</div>
  ```

  クラスセレクタとの違い：
  - **ID**: ページ内で **1つの要素のみ** に使う（一意性が必要）
  - **クラス**: 複数の要素に同じクラスを付けられる

  IDセレクタはCSSの **詳細度（specificity）** が高く、クラスセレクタより優先される。一般的に、スタイルの適用にはクラスセレクタを優先し、IDはアンカーリンクやJavaScriptでの要素取得に使うことが多い。
