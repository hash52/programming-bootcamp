explanation: |
  **Flexbox（フレックスボックス）** は、CSSのレイアウトモデルの1つで、子要素を **横方向・縦方向に柔軟に配置** するための仕組みである。

  ```css
  .container {
    display: flex;          /* Flexboxを有効にする */
    justify-content: center; /* 横方向の配置 */
    align-items: center;     /* 縦方向の配置 */
  }
  ```

  Flexboxを使うと以下のことが簡単にできる：
  - 要素を横並びにする
  - 要素を中央揃えにする
  - 要素を均等に並べる
  - 残りのスペースを均等に配分する
