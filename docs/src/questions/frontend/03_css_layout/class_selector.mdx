explanation: |
  **クラスセレクタ** はHTMLの `class` 属性の値に対してスタイルを適用する。ピリオド（`.`）の後にクラス名を書く。

  ```css
  /* クラスセレクタ */
  .highlight {
    background-color: yellow;
  }
  ```

  ```html
  <p class="highlight">このテキストが黄色くなる</p>
  <p>このテキストは変わらない</p>
  <span class="highlight">これも黄色くなる</span>
  ```

  特徴：
  - 複数の要素に同じクラスを付けられる（1対多）
  - 1つの要素に複数のクラスを付けられる（`class="btn primary"`）
  - 同じCSSファイルで何度でも使い回せる
