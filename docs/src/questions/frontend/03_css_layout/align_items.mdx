explanation: |
  `align-items` プロパティは、Flexbox内の子要素の **交差軸方向（デフォルトは縦方向）の配置** を指定する。

  ```css
  .container {
    display: flex;
    height: 200px;
    align-items: center; /* 縦方向の中央揃え */
  }
  ```

  主な値：
  - `stretch`（デフォルト）: 子要素をコンテナの高さに伸ばす
  - `flex-start`: 上寄せ
  - `flex-end`: 下寄せ
  - `center`: 縦方向の中央揃え
  - `baseline`: テキストのベースラインに揃える

  `justify-content: center` と `align-items: center` を組み合わせると、**縦横の完全中央揃え** ができる。
