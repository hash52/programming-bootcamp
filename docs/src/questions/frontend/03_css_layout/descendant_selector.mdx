explanation: |
  **子孫セレクタ** は、特定の要素の **子孫（内側にあるすべての要素）** に対してスタイルを適用する。セレクタとセレクタの間にスペースを入れて書く。

  ```css
  /* .navの中にあるaタグすべてに適用 */
  .nav a {
    text-decoration: none;
    color: white;
  }
  ```

  ```html
  <nav class="nav">
    <ul>
      <li><a href="/">ホーム</a></li>  <!-- これに適用される -->
      <li><a href="/about">概要</a></li>  <!-- これにも適用される -->
    </ul>
  </nav>
  <a href="/other">別のリンク</a>  <!-- これには適用されない -->
  ```

  - 直下の子要素だけに適用したい場合は `>` を使う（**子セレクタ**: `.nav > a`）
  - 子孫セレクタは階層が深くても適用される
