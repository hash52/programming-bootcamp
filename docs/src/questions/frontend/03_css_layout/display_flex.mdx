explanation: |
  `display: flex` を親要素に指定すると、その要素が **フレックスコンテナ** になり、子要素が **フレックスアイテム** として横並びに配置される。

  ```css
  .container {
    display: flex; /* これだけで子要素が横並びになる */
  }
  ```

  ```html
  <div class="container">
    <div>アイテム1</div>
    <div>アイテム2</div>
    <div>アイテム3</div>
  </div>
  ```

  デフォルトの動作：
  - 子要素が **左から右** に横並びになる
  - 子要素の高さが揃う（stretch）
  - 子要素は内容に応じた幅になる
