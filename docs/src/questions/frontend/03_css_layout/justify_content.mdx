explanation: |
  `justify-content` プロパティは、Flexbox内の子要素の **主軸方向（デフォルトは横方向）の配置** を指定する。

  ```css
  .container {
    display: flex;
    justify-content: center; /* 中央揃え */
  }
  ```

  主な値：
  - `flex-start`（デフォルト）: 左寄せ
  - `flex-end`: 右寄せ
  - `center`: 中央揃え
  - `space-between`: 最初と最後を端に置き、残りを均等配分
  - `space-around`: 各要素の左右に均等なスペースを配置
  - `space-evenly`: 全てのスペースを均等に配分
