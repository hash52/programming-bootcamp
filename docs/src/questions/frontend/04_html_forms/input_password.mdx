explanation: |
  `<input type="password">` はパスワード入力欄を作成する。入力した文字が `●` などに置き換えて表示されるため、画面を見られても内容がわからない。

  ```html
  <input type="password" name="password" placeholder="パスワードを入力">
  ```

  注意点：
  - `type="password"` は画面表示のみを隠す。送信時のデータは通常のテキストと同じ
  - セキュリティのため、HTTPSを使った通信でフォームを送信することが重要である
  - `type="text"` との違いは表示方法のみ
