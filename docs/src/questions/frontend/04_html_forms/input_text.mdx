explanation: |
  `<input type="text">` は1行のテキスト入力欄を作成する。

  ```html
  <input type="text" name="username" placeholder="ユーザー名を入力">
  ```

  重要な属性：
  - `type="text"`: テキスト入力欄の種類を指定
  - `name`: フォーム送信時のキー名（サーバーでの識別に使う）
  - `placeholder`: 入力前に薄く表示されるヒントテキスト
  - `value`: 初期値
  - `maxlength`: 最大入力文字数
  - `required`: 必須入力にする
