explanation: |
  `<label>` タグは入力欄に関連するラベルを付けるためのタグである。

  ```html
  <label for="username">ユーザー名</label>
  <input type="text" id="username" name="username">
  ```

  `<label>` を使うメリット：
  - `for` 属性に入力欄の `id` を指定すると、ラベルをクリックしても入力欄にフォーカスが当たる
  - スクリーンリーダー（視覚障害者向けの読み上げソフト）が入力欄の目的を読み上げられる
  - **アクセシビリティ** の向上に重要

  または `<label>` で入力欄を囲む書き方もある：
  ```html
  <label>
    ユーザー名
    <input type="text" name="username">
  </label>
  ```
