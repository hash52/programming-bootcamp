explanation: |
  `<select>` タグと `<option>` タグの組み合わせでドロップダウンリストを作成する。

  ```html
  <select name="country">
    <option value="jp">日本</option>
    <option value="us">アメリカ</option>
    <option value="uk">イギリス</option>
  </select>
  ```

  属性：
  - `<select>` の `name`: フォーム送信時のキー名
  - `<option>` の `value`: 送信される値
  - `<option>` のテキスト: 画面に表示されるテキスト
  - `selected` 属性: 最初から選択された状態にする（`<option value="jp" selected>`）
  - `multiple` 属性: 複数選択を可能にする（`<select multiple>`）
