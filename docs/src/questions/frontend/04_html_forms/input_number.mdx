explanation: |
  `<input type="number">` は数値入力欄を作成する。数値以外の文字は入力できず、スピナー（上下の矢印ボタン）が表示される。

  ```html
  <input type="number" name="age" min="0" max="120" step="1">
  ```

  特徴と属性：
  - 数値のみ入力可能（ブラウザが制限する）
  - `min`: 最小値
  - `max`: 最大値
  - `step`: 増減のステップ（デフォルトは1）
  - スマートフォンでは数字キーボードが表示される
