explanation: |
  `<form>` タグはフォーム全体を囲むタグである。入力要素（`<input>` など）を内側に配置する。

  ```html
  <form action="/login" method="post">
    <input type="text" name="username" placeholder="ユーザー名">
    <input type="password" name="password" placeholder="パスワード">
    <button type="submit">ログイン</button>
  </form>
  ```

  重要な属性：
  - `action`: フォームデータの送信先URL（省略すると現在のURLに送信）
  - `method`: 送信方法
    - `get`: URLにデータを付けて送信（検索フォームなどに使用）
    - `post`: リクエストのボディにデータを付けて送信（ログインフォームなどに使用）
