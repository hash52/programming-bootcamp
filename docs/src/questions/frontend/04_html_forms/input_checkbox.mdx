explanation: |
  `<input type="checkbox">` はチェックボックスを作成する。ユーザーがオン/オフを切り替えられる。

  ```html
  <input type="checkbox" name="agree" value="yes" id="agree">
  <label for="agree">利用規約に同意する</label>
  ```

  特徴と属性：
  - チェックされた場合のみフォームデータとして送信される
  - `value`: 送信する値（デフォルトは `"on"`）
  - `checked`: 最初からチェックされた状態にする
  - 複数のオプションから複数選択させたい場合に使う（ラジオボタンは1つのみ選択）
