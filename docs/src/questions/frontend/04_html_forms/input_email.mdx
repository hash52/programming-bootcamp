explanation: |
  `<input type="email">` はメールアドレス入力欄を作成する。`type="text"` との違いは、フォーム送信時に **メールアドレスの形式（@を含む）かどうか** を自動でチェックする点である。

  ```html
  <input type="email" name="email" placeholder="example@example.com">
  ```

  特徴：
  - フォーム送信時に `@` を含む形式かどうかを検証する
  - スマートフォンでは @記号 が入力しやすいキーボードが表示される
  - HTMLだけのバリデーションなので、サーバー側でも検証が必要
