explanation: |
  `<input type="radio">` はラジオボタンを作成する。同じ `name` 属性を持つグループの中から **1つだけ** 選択できる。

  ```html
  <input type="radio" name="gender" value="male" id="male">
  <label for="male">男性</label>

  <input type="radio" name="gender" value="female" id="female">
  <label for="female">女性</label>
  ```

  特徴と属性：
  - 同じ `name` を持つラジオボタンは **同じグループ** になり、1つしか選べない
  - `value`: 選択された場合に送信される値
  - `checked`: 最初から選択された状態にする
  - チェックボックスとの違い：ラジオボタンは1つのみ、チェックボックスは複数選択可
