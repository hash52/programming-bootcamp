explanation: |
  `<textarea>` タグは複数行のテキスト入力欄を作成する。`<input type="text">` との違いは、複数行の入力ができる点である。

  ```html
  <textarea name="message" rows="5" cols="40" placeholder="お問い合わせ内容を入力してください"></textarea>
  ```

  重要な点：
  - `<textarea>` は **終了タグが必要**（`<input>` は終了タグ不要）
  - `rows`: 表示する行数
  - `cols`: 表示する列数（文字数）
  - `placeholder`: 入力前に表示されるヒントテキスト
  - 初期値を設定する場合は `value` 属性ではなく、開始タグと終了タグの間にテキストを書く
