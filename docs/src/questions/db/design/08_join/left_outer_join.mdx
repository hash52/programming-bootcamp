explanation: |
  `LEFT JOIN`（または `LEFT OUTER JOIN`）は左テーブルの全行を取得し、
  右テーブルに一致がない場合はNULLを返す。

  基本構文：
  ```sql
  SELECT 列名
  FROM 左テーブル e
  LEFT JOIN 右テーブル d ON 結合条件;
  ```

  **INNER JOIN との違い**：
  - `INNER JOIN`: 両テーブルに一致する行のみ（渡辺あかりは除外される）
  - `LEFT JOIN`: 左テーブル（employees）の全行（渡辺あかりも含まれ、dept_nameはNULL）

  **結果イメージ**：
  ```
  | emp_name   | dept_name |
  |------------|-----------|
  | 田中太郎   | 営業部    |
  | 鈴木花子   | 開発部    |
  | 佐藤次郎   | 開発部    |
  | 渡辺あかり | NULL      |  ← LEFT JOIN なので含まれる
  | 中村健一   | 営業部    |
  ```
hint: "部門未所属の社員も含めるには INNER ではなく何が必要か考えよう"
