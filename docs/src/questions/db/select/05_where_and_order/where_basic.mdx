explanation: |
  `WHERE 条件式` で行の絞り込みができる。

  基本構文：
  ```sql
  SELECT * FROM テーブル名 WHERE 条件式;
  ```

  **主な比較演算子**：
  - `=`: 等しい
  - `!=` または `<>`: 等しくない
  - `>`: より大きい
  - `<`: より小さい
  - `>=`: 以上
  - `<=`: 以下

  `WHERE` 句を使うことで必要なデータのみを効率的に取得できる。
