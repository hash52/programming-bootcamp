explanation: |
  `IN (値1, 値2, ...)` で指定したいずれかの値に一致する行を取得できる。

  基本構文：
  ```sql
  WHERE 列名 IN (値1, 値2, ...)
  ```

  **ポイント**：
  - `OR` の代替として使用でき、値が多い場合はシンプルに記述できる
  - `NOT IN (...)` で指定した値以外の行を取得できる
  - 文字列にも使用可能（例: `WHERE dept_name IN ('営業部', '開発部')`）
  - サブクエリと組み合わせることも可能

  **`OR` との比較**：
  `WHERE dept_id IN (1, 3)` は
  `WHERE dept_id = 1 OR dept_id = 3` と同じ意味である。
