explanation: |
  `LIKE` は部分一致検索（あいまい検索）を行う演算子である。

  **ワイルドカード**：
  - `%`: 任意の0文字以上の文字列にマッチする
  - `_`: 任意の1文字にマッチする

  **使用例**：
  - `LIKE '%田%'`: 「田」を含む文字列（田中、田村、吉田など）
  - `LIKE '田%'`: 「田」で始まる文字列（田中、田村など）
  - `LIKE '%田'`: 「田」で終わる文字列（吉田、上田など）
  - `LIKE '田_'`: 「田」の後に1文字が続く文字列（田中、田村など）
  - `NOT LIKE '%田%'`: 「田」を含まない文字列

  **注意点**：
  - 大文字小文字の区別はDBMSの設定によって異なる
  - 大量データへの `LIKE` 検索はパフォーマンスに影響する場合がある
hint: "部分一致検索に使う演算子を考えよう"
