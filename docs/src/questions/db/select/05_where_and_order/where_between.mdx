sampleAnswer: "SELECT * FROM employees WHERE salary BETWEEN 300000 AND 400000;"
explanation: |
  `BETWEEN 値1 AND 値2` で指定した範囲（両端を含む）のデータを取得できる。

  基本構文：
  ```sql
  WHERE 列名 BETWEEN 最小値 AND 最大値
  ```

  **ポイント**：
  - 両端の値（300000と400000）を含む（`>=` と `<=` と同じ）
  - `NOT BETWEEN` で範囲外のデータを取得できる
  - 文字列や日付にも使用可能（例: `WHERE hire_date BETWEEN '2020-01-01' AND '2023-12-31'`）

  **`AND` との比較**：
  `WHERE salary BETWEEN 300000 AND 400000` は
  `WHERE salary >= 300000 AND salary <= 400000` と同じ意味である。
