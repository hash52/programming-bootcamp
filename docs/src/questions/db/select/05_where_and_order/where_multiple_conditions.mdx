explanation: |
  複数の条件を組み合わせるには `AND`（かつ）または `OR`（または）を使う。

  **論理演算子**：
  - `AND`: 両方の条件を満たす行のみ取得（「かつ」）
  - `OR`: いずれか一方を満たす行を取得（「または」）
  - `NOT`: 条件を否定する（「〜でない」）

  **優先順位**：
  `NOT` > `AND` > `OR` の順に評価される。
  複雑な条件は `()` でグループ化して優先順位を明確にすること。

  例: `WHERE (dept_id = 1 OR dept_id = 2) AND salary >= 300000`
