explanation: |
  `ORDER BY 列名` でデータを並び替えられる。

  基本構文：
  ```sql
  SELECT * FROM テーブル名 ORDER BY 列名 ASC|DESC;
  ```

  **並び替えの方向**：
  - `ASC`（昇順）: 小さい値から大きい値の順（デフォルト）
  - `DESC`（降順）: 大きい値から小さい値の順

  **複数列での並び替え**：
  ```sql
  ORDER BY dept_id ASC, salary DESC
  -- dept_idの昇順で並べ、同じdept_idの場合はsalaryの降順で並べる
  ```

  **NULLの扱い**：
  - デフォルトでは NULL が最後（PostgreSQL）または最初（MySQL）に来る場合がある
  - `NULLS FIRST` / `NULLS LAST` で明示的に指定できる
