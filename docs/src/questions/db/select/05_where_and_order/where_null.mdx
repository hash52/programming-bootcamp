sampleAnswer: "SELECT * FROM employees WHERE dept_id IS NULL;"
explanation: |
  NULLの判定には `IS NULL` を使用する。

  **重要な注意点**：
  - `= NULL` ではなく `IS NULL` と書くことが必須である
  - NULLは「値が存在しない」ことを表す特殊な状態であり、`=` での比較は常にFALSEになる
  - `IS NOT NULL` でNULLでない行を取得できる

  **NULLとは**：
  - 「空文字列（''）」とは異なる（'' は値が存在するが、NULLは値自体が存在しない）
  - NULLを含む計算は基本的にNULLになる（例: `NULL + 1 = NULL`）
  - NULLの比較（`NULL = NULL`）もNULLになる（TRUEにならない）

  サンプルDBでは `渡辺あかり` の `dept_id` がNULLである（部門未所属）。
hint: "NULLの判定には = ではなく特別な演算子を使う"
