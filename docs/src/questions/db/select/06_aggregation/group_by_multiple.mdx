sampleAnswer: "SELECT dept_id, AVG(salary) AS 平均給与 FROM employees GROUP BY dept_id ORDER BY AVG(salary) DESC;"
explanation: |
  `GROUP BY` と `ORDER BY` を組み合わせることで、グループ集計の結果を並び替えることができる。

  **`ORDER BY` での集約関数の使用**：
  - `ORDER BY AVG(salary) DESC`: 平均給与の降順で並べる
  - `ORDER BY 平均給与 DESC`: 別名を使って並べることも可能
  - `ORDER BY 2 DESC`: SELECT句の2番目の列を基準に並べる（列番号参照）

  **複数列での GROUP BY**：
  ```sql
  GROUP BY dept_id, hire_date
  -- dept_id と hire_date の組み合わせごとにグループ化する
  ```

  **実行順序**：
  `FROM` → `WHERE` → `GROUP BY` → `HAVING` → `SELECT` → `ORDER BY` → `LIMIT`
hint: "GROUP BY の後に並び替えをするには ORDER BY を使う"
