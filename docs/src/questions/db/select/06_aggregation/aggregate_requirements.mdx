sampleAnswer: "SELECT dept_id, MAX(salary) AS 最高給与, MIN(salary) AS 最低給与 FROM employees GROUP BY dept_id HAVING MAX(salary) >= 350000;"
explanation: |
  要件を読んで集計SQLに変換するときは、必要な集約関数と条件を整理する。

  **この問題の要件の分解**：
  - 「部門ごとの最高給与と最低給与を取得」
    → `SELECT dept_id, MAX(salary), MIN(salary) ... GROUP BY dept_id`
  - 「最高給与が350000以上の部門のみ」
    → `HAVING MAX(salary) >= 350000`

  **集約関数の選び方**：
  - 「最大」→ `MAX`
  - 「最小」→ `MIN`
  - 「合計」→ `SUM`
  - 「平均」→ `AVG`
  - 「件数」→ `COUNT`

  要件を読んで適切な集約関数を選ぶことが重要である。
hint: "最大・最小それぞれの集約関数を思い出そう"
