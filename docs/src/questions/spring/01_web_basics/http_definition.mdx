explanation: |
  HTTPはHyperText Transfer Protocolの略で、WebブラウザとWebサーバー間の通信規約（プロトコル）である。

  ブラウザがHTTPリクエストを送り、サーバーがHTTPレスポンスを返すことでWebページが表示される。

  - **リクエスト**: ブラウザがサーバーに「このページをください」と送るメッセージ
  - **レスポンス**: サーバーが「はい、これです」と返すメッセージ（HTMLやJSONなど）
