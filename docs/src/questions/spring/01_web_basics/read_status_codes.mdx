explanation: |
  ステータスコード **404** は "Not Found" を意味する。

  クライアントがリクエストしたリソース（ページやファイルなど）がサーバー上に存在しないことを示す。

  よく見るステータスコード：
  - **200 OK**: 成功
  - **301/302**: リダイレクト
  - **404 Not Found**: ページが存在しない
  - **500 Internal Server Error**: サーバー内部エラー
