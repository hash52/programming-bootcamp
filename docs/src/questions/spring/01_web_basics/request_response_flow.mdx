explanation: |
  Webの基本的な通信はクライアント（ブラウザ）とサーバーの間で行われる。

  1. ブラウザがURLにアクセス（HTTPリクエストを送信）
  2. サーバーがリクエストを受け取り処理
  3. サーバーがHTTPレスポンス（HTMLなど）を返す
  4. ブラウザがレスポンスを解析して画面を表示

  この「リクエスト → レスポンス」の繰り返しがWebの基本である。
