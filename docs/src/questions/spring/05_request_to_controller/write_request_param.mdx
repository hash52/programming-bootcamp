explanation: |
  `@RequestParam` をメソッドの引数の前に付けることで、URLのクエリパラメータを受け取ることができる。

  `/search?keyword=spring` というURLにアクセスすると、`keyword` 引数に `"spring"` が渡される。

  複数のパラメータも受け取れる：
  ```java
  public String search(
      @RequestParam String keyword,
      @RequestParam int page) { ... }
  ```
