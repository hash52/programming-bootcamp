explanation: |
  `@Select` アノテーションにSQL文を記述することで、SELECT文を実行するメソッドを定義できる。

  MyBatisはメソッドの戻り値の型に合わせてSQLの結果を自動変換する。

  パラメータを使う場合：
  ```java
  @Select("SELECT * FROM products WHERE id = #{id}")
  Product findById(Long id);
  ```

  `#{変数名}` の形式でメソッドの引数をSQLに渡すことができる。
