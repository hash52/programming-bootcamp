explanation: |
  `@Mapper` アノテーションを付けることでMyBatisがそのインターフェースをマッパーとして認識し、SQLを実行できるようにする。

  MyBatisはインターフェースの実装クラスを自動生成するため、自分で実装クラスを書く必要はない。

  `@Mapper` が付いたインターフェースはSpringのDIコンテナにも登録されるため、`@Autowired` で注入できる。
