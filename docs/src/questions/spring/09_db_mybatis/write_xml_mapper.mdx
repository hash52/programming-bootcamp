explanation: |
  XMLマッパーの基本構造：

  - `namespace`: 対応するMapperインターフェースの完全修飾名
  - `<select id="メソッド名">`: Mapperインターフェースのメソッドと対応するSELECT文
  - `resultType`: 結果をマッピングするJavaクラスの完全修飾名

  XMLマッパーは `src/main/resources` 配下にMapperインターフェースと同じパッケージ構造で配置する。
