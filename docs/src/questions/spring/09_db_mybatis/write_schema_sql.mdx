explanation: |
  `src/main/resources/schema.sql` に記述したSQLは、Spring Boot起動時に自動実行される（H2などの組み込みDBを使う場合）。

  テーブル定義に使う主な制約：
  - `PRIMARY KEY`: 主キー
  - `AUTO_INCREMENT`: 自動採番
  - `NOT NULL`: NULL禁止
  - `VARCHAR(n)`: 可変長文字列

  `application.properties` で `spring.sql.init.mode=always` を設定すると常に実行される。
