explanation: |
  `th:object="${オブジェクト}"` を使うと、その要素の子要素内で `*{フィールド名}` という短縮記法でフィールドにアクセスできる。

  ```html
  <!-- th:objectなし -->
  <p th:text="${user.name}">名前</p>

  <!-- th:objectあり -->
  <div th:object="${user}">
    <p th:text="*{name}">名前</p>
    <p th:text="*{email}">メール</p>
  </div>
  ```

  フォームのバインディングにもよく使われる（`th:field="*{name}"`）。
