explanation: |
  `th:if="${条件}"` は条件が `true` の場合のみ要素を表示する。

  条件が `false` の場合はその要素がHTMLから取り除かれる（コメントアウトではなく完全に削除）。

  反対の条件には `th:unless="${条件}"` を使う（条件が `false` のときに表示）。

  ```html
  <p th:if="${isLoggedIn}">ログイン中です</p>
  <p th:unless="${isLoggedIn}">ログインしていません</p>
  ```
