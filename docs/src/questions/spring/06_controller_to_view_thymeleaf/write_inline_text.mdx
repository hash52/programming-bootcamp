explanation: |
  Thymeleafでは `[[${変数名}]]`（インライン式）を使って、テキストの途中に値を埋め込むことができる。

  ```html
  <p>こんにちは、[[${username}]]さん！</p>
  <!-- "こんにちは、山田太郎さん！" と表示される -->
  ```

  `th:text` との違い：
  - `th:text`: 要素のテキスト全体を置き換える
  - `[[...]]`: テキストの一部に値を埋め込める（インライン）

  どちらもHTMLエスケープが行われるため安全である。
