explanation: |
  Thymeleafには便利なユーティリティオブジェクトが用意されている。

  主なユーティリティ：
  - `#strings`: 文字列操作（`toUpperCase`、`toLowerCase`、`contains`、`isEmpty`など）
  - `#dates`: 日付操作（`format`、`year`、`month`など）
  - `#numbers`: 数値フォーマット（`formatDecimal`など）
  - `#lists`: リスト操作（`size`、`contains`など）

  `${...}` の中で `#ユーティリティ名.メソッド名(引数)` の形式で使用する。
