explanation: |
  `th:each="変数 : ${リスト}"` でリストの各要素を繰り返し表示できる。

  Javaの拡張for文（`for (Item item : items)`）と同様の概念である。

  ```html
  <ul>
    <li th:each="item : ${items}" th:text="${item.name}">商品名</li>
  </ul>
  ```

  ループのインデックスなども取得できる：
  ```html
  <li th:each="item, stat : ${items}">
    [[${stat.index}]]: [[${item.name}]]
  </li>
  ```
