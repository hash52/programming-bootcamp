explanation: |
  `th:text="${変数名}"` で、ModelにaddAttributeした値を要素のテキストコンテンツとして表示できる。

  `th:text` がある場合、HTMLのタグ内のテキストは置き換えられる。

  ```html
  <!-- "こんにちは！" が表示される -->
  <p th:text="${message}">仮テキスト</p>
  ```

  静的なプレースホルダーとして `仮テキスト` と書いても、実行時に動的に置換される。
