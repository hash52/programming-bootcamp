explanation: |
  Thymeleafの主要な属性：

  | 属性 | 用途 |
  |------|------|
  | `th:text` | テキストコンテンツの表示（HTMLエスケープあり） |
  | `th:if` | 条件が真のときのみ要素を表示 |
  | `th:each` | リストや配列を繰り返し表示 |
  | `th:href` | リンクのURL指定 |
  | `th:src` | 画像のパス指定 |
  | `th:value` | フォームのvalue属性 |
  | `th:object` | フォームのバインド対象オブジェクト指定 |
