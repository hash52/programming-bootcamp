explanation: |
  `model.addAttribute("キー", 値)` でModelにデータを追加する。

  - キーは文字列で、ThymeleafのHTML内で `${キー}` の形式で参照できる
  - 値はどのようなオブジェクトでも渡すことができる（String、Integer、List、独自クラスなど）

  複数のデータを渡す場合は、`addAttribute` を複数回呼び出す。
