explanation: |
  `new` とDIの使い分けの判断基準：

  **DIに任せるもの**（Springが管理するもの）:
  - サービスクラス（`UserService`、`EmailSender`など）
  - リポジトリクラス
  - アプリ全体で共有するコンポーネント
  → ライフサイクル管理や依存関係解決をSpringに任せることで保守性が高まる

  **`new` するもの**:
  - 一時的なデータ（`new User("Taro", 20)`、`new ArrayList<>()`）
  - 特定の処理内だけで使うローカルなオブジェクト
  → 毎回新しいインスタンスが必要なものは自分で `new` する
---

以下のコードのうち、SpringのDIに任せるべきものを正しく選んでいるのはどれか。

```java
List<String> names = new ArrayList<>();
User user = new User("Taro", 20);
UserService userService = new UserService();
EmailSender emailSender = new EmailSender();
```
