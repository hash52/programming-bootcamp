explanation: |
  `BindingResult` はバリデーションエラーの情報を保持するオブジェクトである。

  主なメソッド：
  - `result.hasErrors()`: エラーが1つでもある場合に `true` を返す
  - `result.getFieldErrors()`: フィールドごとのエラーリストを取得する
  - `result.getAllErrors()`: すべてのエラーリストを取得する

  `BindingResult` は必ず `@Validated` が付いた引数の直後に配置する。
