explanation: |
  `@Validated`（または `@Valid`）をコントローラのメソッドの引数に付けると、その引数に対するバリデーションが実行される。

  - `@Validated`: Spring独自のアノテーション（グループ検証も可能）
  - `@Valid`: Jakarta標準のアノテーション（シンプルな場合はどちらでも可）

  バリデーション結果は `BindingResult` で受け取る（`@Validated` の引数の直後に記述する）。
