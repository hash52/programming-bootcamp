explanation: |
  `@Email` はメールアドレス形式（`xxx@xxx.xxx`）かどうかを検証するアノテーションである。

  `@Email` だけでは空文字を許可してしまうため、必須チェックと組み合わせる場合は `@NotBlank` も付ける。

  ```java
  @NotBlank
  @Email
  private String email;
  ```
