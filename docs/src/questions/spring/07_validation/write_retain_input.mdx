explanation: |
  Thymeleafの `th:field="*{フィールド名}"` を使うと、バリデーションエラー時に入力値が保持される。

  `th:field` は以下を自動設定する：
  - `id` 属性
  - `name` 属性
  - `value` 属性（フォームオブジェクトの値）

  コントローラ側では、バリデーションエラー時にフォームオブジェクトをモデルに渡す（Springが自動で行う）。
