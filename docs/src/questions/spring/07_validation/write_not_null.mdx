explanation: |
  `@NotBlank` は文字列フィールドに対して、`null`・空文字（`""`）・空白のみ（`"   "`）を禁止するアノテーションである。

  - `@NotNull`: nullのみ禁止（空文字は許可）
  - `@NotEmpty`: nullと空文字を禁止（空白のみは許可）
  - `@NotBlank`: null・空文字・空白のみをすべて禁止（最も厳格）

  フォームの必須入力チェックには `@NotBlank` を使うのが一般的である。
