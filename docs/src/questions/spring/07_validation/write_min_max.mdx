  `@Min(value)` は数値の最小値チェック、`@Max(value)` は最大値チェックのアノテーションである。

  複数のアノテーションを重ねることができる。

  数値の範囲チェックには `@Range(min, max)` を使う方法もある（Hibernate Validatorの拡張）。
