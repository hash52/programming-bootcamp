  `session.setAttribute("key", value)` でセッションにデータを保存する。

  保存できる値はシリアライズ可能なJavaオブジェクトならすべて対応している。

  セッションデータはサーバー側のメモリに保存されるため、大きなオブジェクトを大量に保存するとメモリを圧迫する点に注意する。
