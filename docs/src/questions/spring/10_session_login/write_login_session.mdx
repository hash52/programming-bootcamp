explanation: |
  セッションを使ったログイン機能の実装パターン：

  **ログイン処理**:
  1. フォームの入力値でユーザーを検索する
  2. パスワードを検証する
  3. `session.setAttribute("loginUser", user)` でセッションに保存する

  **ログアウト処理**:
  - `session.invalidate()` でセッション全体を無効化する

  **ログインチェック**:
  - `session.getAttribute("loginUser") == null` の場合は未ログイン
