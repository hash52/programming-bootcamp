explanation: |
  `@Controller` のメソッドの返り値の扱いは `@ResponseBody` の有無で変わる。

  - **`@ResponseBody`なし**: 返り値の文字列がビュー名として扱われ、対応するHTMLテンプレートが描画される
  - **`@ResponseBody`あり**: 返り値がそのままHTTPレスポンスのボディに書き込まれる

  `@RestController` は `@Controller + @ResponseBody` をまとめたアノテーションで、REST APIのコントローラ作成に使う。
---

以下の2つのコントローラメソッドの動作の違いとして、正しいものを選べ。

```java
// メソッドA：@ResponseBodyなし
@GetMapping("/a")
public String methodA() {
    return "hello";
}

// メソッドB：@ResponseBodyあり
@GetMapping("/b")
@ResponseBody
public String methodB() {
    return "hello";
}
```
