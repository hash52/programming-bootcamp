  コントローラからViewへのデータ受け渡しの手順：

  1. コントローラで `model.addAttribute("キー", 値)` を呼び出してデータを登録する
  2. ThymeleafのHTMLで `th:text="${キー}"` を使ってデータを表示する

  `th:text` 属性は要素のテキストコンテンツをモデルの値で置き換える。
  HTMLエスケープが自動的に行われるため、XSS対策にもなる。
