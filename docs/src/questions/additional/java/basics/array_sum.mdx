## 問題

整数の配列を受け取り、その要素の合計値を返すメソッド `sum(int[] arr)` を実装せよ。

**実行例：**

```
sum(new int[]{1, 2, 3, 4, 5})    → 15
sum(new int[]{10, -3, 7})        → 14
sum(new int[]{})                 → 0
```

空の配列のケースも正しく処理できるようにすること。

<details>
<summary>模範解答を見る</summary>

```java
public class ArraySum {

    static int sum(int[] arr) {
        int total = 0;
        for (int num : arr) {
            total += num;
        }
        return total;
    }

    public static void main(String[] args) {
        System.out.println(sum(new int[]{1, 2, 3, 4, 5}));  // 15
        System.out.println(sum(new int[]{10, -3, 7}));       // 14
        System.out.println(sum(new int[]{}));                 // 0
    }
}
```

**ポイント：** 拡張 for 文（for-each）を使うと、配列の境界を意識せずにシンプルに書ける。
`total = 0` で初期化するので空配列でも正しく 0 を返す。
