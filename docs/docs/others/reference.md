# リファレンス集

教材で使用する技術の公式ドキュメントをまとめたものである。
困ったときや詳しく調べたいときに活用すること。

## Java

- [JDK 21 API リファレンス](https://docs.oracle.com/en/java/javase/21/docs/api/) — Java 21 の標準ライブラリAPIの一覧
- [Java チュートリアル（Oracle）](https://docs.oracle.com/javase/tutorial/) — Oracle公式のJavaチュートリアル（英語）

## フロントエンド（HTML / CSS）

- [MDN Web Docs (HTML)](https://developer.mozilla.org/ja/docs/Web/HTML) — HTML要素・属性のリファレンス（日本語）
- [MDN Web Docs (CSS)](https://developer.mozilla.org/ja/docs/Web/CSS) — CSSプロパティ・セレクタのリファレンス（日本語）

## データベース（PostgreSQL）

- [PostgreSQL 公式ドキュメント（日本語）](https://www.postgresql.jp/document/) — PostgreSQLの日本語ドキュメント
- [PostgreSQL 公式ドキュメント（英語最新）](https://www.postgresql.org/docs/) — PostgreSQLの最新英語ドキュメント

## Git

- [Pro Git Book（日本語）](https://git-scm.com/book/ja/v2) — Gitの定番書籍。無料でオンラインで読める
- [Git 公式ドキュメント](https://git-scm.com/doc) — Gitの公式リファレンスとチュートリアル

## Spring / MyBatis / Thymeleaf

- [Spring Boot 公式](https://spring.io/projects/spring-boot) — Spring Bootプロジェクトの公式ページ
- [Spring Boot 公式ドキュメント](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/) — Spring Bootの詳細ドキュメント（英語）
- [MyBatis 公式（日本語）](https://mybatis.org/mybatis-3/ja/index.html) — MyBatisの日本語ドキュメント
- [Thymeleaf 公式ドキュメント](https://www.thymeleaf.org/documentation.html) — Thymeleafテンプレートエンジンの公式ドキュメント

## ツール

- [Eclipse 公式](https://www.eclipse.org/documentation/) — Eclipse IDEの公式ドキュメント
