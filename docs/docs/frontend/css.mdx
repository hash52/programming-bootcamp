# テキストの色を変える（CSS）

Webページでは、文字の色を自由に変えることができる。  
ここでは、HTML と CSS を使って文字色を指定する方法を学ぶ。

## HTML と CSS の基本

HTML で文章を書き、CSS で色を指定する。

例えば、次のように書く：

{/* 
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>文字色のサンプル</title>
</head>
<body>
  <p class="hoge1">これはクラスにhoge1を指定したテキストです。</p>
  <p class="hoge2">これはクラスにhoge2を指定したテキストです。</p>
  <p class="hoge3">これはクラスにhoge3を指定したテキストです。</p>
</body>
</html>
*/}
{/*
.hoge1 { color: red; }
.hoge2 { color: blue; }
.hoge3 { color: green; }
*/}

### 実験してみよう

- `.hoge1` や `.hoge2` の値を別の色名（orange, purple など）に変えてみる。  
- `color: #FF0000;` のように 16 進数で色を指定することもできる。

## まとめ

- CSS の `color` プロパティで文字色を指定できる  
- クラスを使うと複数の要素に同じ色を適用できる  
- 色は名前（red, blue など）や 16 進数（#FF0000 など）で指定できる

### 実際に試すことが大切

色の値を変えて表示を確認することで、文字色の指定方法を理解できる。
