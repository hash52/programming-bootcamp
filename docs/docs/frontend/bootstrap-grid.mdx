# Bootstrap の Grid システム

Bootstrap では、画面をグリッド（行と列）で分割してレイアウトを作ることができる。  
この章では、基本的な Grid システムの使い方を学ぶ。

## Grid の基本

Bootstrap の Grid は、`container` → `row` → `col` の順に要素を配置する。

例えば、次のように書く：

{/*
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Bootstrap Grid サンプル</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <div class="row">
      <div class="col-4" style="background-color: lightcoral;">列1</div>
      <div class="col-4" style="background-color: lightblue;">列2</div>
      <div class="col-4" style="background-color: lightgreen;">列3</div>
    </div>
  </div>
</body>
</html>
*/}

この例では、1 行を 3 つの列に分け、それぞれ幅 4（合計 12）で表示している。

### 実験してみよう

- `col-4` を `col-6` に変えてみるとどうなるか確認する。  
- 列の数を増やしたり減らしたりして、画面幅に応じて自動で調整される様子を試してみる。
