## 1. 商品一覧に検索フォームを追加する

<CodeBlockWithDiff

## 3. `ProductMapper.xml` にLIKE検索SQLを追加する

<CodeBlockWithDiff
title="src/main/resources/mapper/ProductMapper.xml"
code={`   <select id="findById" resultType="com.example.ecsample.entity.Product">
     SELECT id, name, price
     FROM products
     WHERE id = \#{id}
   </select>
+
+  <select id="findByKeyword" resultType="com.example.ecsample.entity.Product">
+    SELECT id, name, price
+    FROM products
+    WHERE name LIKE \#{'%' + keyword + '%'}
+    ORDER BY id
+  </select>

 </mapper>
`}
language="xml"
showLineNumbers={false}
/>

> **ポイント**
> - `LIKE '%キーワード%'` で部分一致検索ができる。
> - MyBatis で `LIKE` を使う場合、`#{...}` の中で文字列連結して `%` を付ける。
> - `${}` を使うとSQLインジェクションの危険があるため、必ず `#{}` を使うこと。

### MVC アーキテクチャの全体像

```
ブラウザ
  ↕ HTTP リクエスト/レスポンス
Controller（@Controller）
  ↕ Model にデータを追加
  ↕ Service に処理を委譲
Service（@Service）
  ↕ Mapper を呼び出す
Mapper（@Mapper）
  ↕ SQL を実行
DB（PostgreSQL）
  ↕ 結果を返す
Mapper → Service → Controller → Model → View（Thymeleaf HTML）
  ↓
ブラウザに HTML を返す
```
